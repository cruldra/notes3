Textual

Textual 是一个用于 Python 的 TUI (文本用户界面) 框架,受现代 Web 开发启发。

📄️ 简介

Textual 是一个为 Python 构建的快速、现代化的 TUI (Terminal User Interface) 框架。它允许开发者使用类似于现代 Web 开发的体验（如 CSS 样式和事件驱动架构）来构建功能丰富的终端应用程序。
