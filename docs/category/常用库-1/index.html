常用库

前端常用库

📄️ DndKit

dnd kit是一个适用于React的轻量级、高性能、可访问且可扩展的拖放工具包.

📄️ Mantine

Mantine是一套功能齐全的用户界面组件库,用于构建现代网络应用程序.

📄️ Pinia

Pinia是Vue的官方状态管理库,作为Vuex的继任者.

📄️ Prisma

Prisma是一个开源的ORM框架,用于与数据库进行交互.它提供了一种简单而强大的方式来管理数据库连接、执行查询和处理数据.

📄️ React Diff Viewer

react-diff-viewer是一个用Diff和React制作的简单而漂亮的文本差异查看器组件.
