Oh My Pi 使用教程

Oh My Pi (OMP) 是一个功能强大的 AI 编码代理，专为终端环境设计。本教程涵盖从基础安装到高级功能的完整使用指南。

📄️ 项目简介

npm version

📄️ 安装指南

系统要求

📄️ 快速开始

本指南将帮助你在 15 分钟内掌握 Oh My Pi 的基本使用方法。

📄️ AI Commit 工具

OMP 的 Commit 工具利用 AI 智能分析代码变更，自动生成符合 Conventional Commits 规范的提交信息。

📄️ Python 执行工具

OMP 集成了完整的 IPython 执行环境，支持持久化内核、富文本输出和强大的辅助函数库。

📄️ LSP 集成

OMP 提供完整的 IDE 级代码智能，支持 40+ 种编程语言，通过 Language Server Protocol (LSP) 实现。

📄️ 子代理系统 (Task Tool)

OMP 的子代理系统允许你并行执行多个任务，每个任务由专门的 AI 代理处理，支持实时流式输出和完整的任务生命周期管理。

📄️ 任务追踪 (Todo Tool)

OMP 提供结构化的任务管理系统，支持分阶段任务列表、实时进度跟踪和自动提醒。

📄️ 浏览器自动化

OMP 集成了 Puppeteer 浏览器自动化，配备 14 种反检测脚本，支持复杂的网页交互和数据提取。

📄️ SSH 远程执行

OMP 的 SSH 工具支持持久连接、自动 OS/Shell 检测和 SSHFS 挂载，让你在远程服务器上像本地一样工作。

📄️ 网络搜索

OMP 提供多提供商搜索和智能内容提取，支持代码仓库、包管理器、安全数据库等多种专业来源。

📄️ MCP 与插件系统

OMP 完整支持 Model Context Protocol (MCP)，允许集成外部工具和服务，同时提供强大的插件系统扩展功能。

📄️ 时间旅行流规则 (TTSR)

TTSR 是 OMP 的创新功能，通过模式触发的规则注入，实现零上下文消耗的按需提示增强。

📄️ 交互式代码审查

OMP 提供结构化的代码审查流程，支持多种审查模式、优先级分级的发现和综合评审结论。

📄️ 模型角色

OMP 的模型角色系统允许为不同任务配置专门的 AI 模型，实现成本与质量的灵活平衡。

📄️ 自定义斜杠命令

OMP 支持使用 TypeScript 编写自定义斜杠命令，实现项目特定的自动化工作流。

📄️ 配置详解

OMP 提供丰富的配置选项，让你根据项目需求和个人偏好定制体验。

📄️ CLI 参考

完整的命令行接口文档，包含所有 OMP 命令和选项。
