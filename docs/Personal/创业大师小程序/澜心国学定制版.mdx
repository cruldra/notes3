## 后端

### 流程

- [ ] 购买2核4g ubuntu24.04服务器,把ip、用户名、密码记录下来
- [ ] 安装docker.io
- [ ] 登录我的阿里云账号，注册两个域名,cn结尾,68一年那种,然后建两个二级域名`api`和`admin`都指向这个新服务器
- [ ] 拿薪炬相关资质去注册两个小程序,其中一个叫`澜心国学`、另一个待定,可以先注册，名字回头可以改
- [ ] 两个小程序认证,300/个/年
- [ ] `api`、`admin`、`uniapp`三个项目都建2个分支对应两个代理商的,在主分支上开发功能,开发好再合并过去
- [ ] 把`api`、`admin`两个项目下的`部署.ps1`中的服务器地址改成新ip
- [ ] `asm-py`项目里面有`docker-compose.yml`,放到`/app`目录下运行`docker-compose up`
