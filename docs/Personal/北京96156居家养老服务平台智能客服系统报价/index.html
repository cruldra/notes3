北京96156居家养老服务平台智能客服系统定制报价

项目名称：北京96156居家养老服务平台智能客服系统

项目周期：6个月（功能开发3个月，微调优化3个月）

总费用：¥600,000.00

项目概述​

本项目旨在为北京96156居家养老服务平台开发一套企业微信RPA智能客服系统，通过自动化技术与AI模型结合，实现与企业微信私域中的用户自动对话。系统将通过RPA技术附加到企业微信窗口，代替人工客服与用户进行自然对话，了解用户需求，提供情绪价值支持，同时收集用户画像，为后续服务转化奠定基础。系统将解决同时在线人数特别多的问题，有效缓解人工客服压力，提高服务效率与用户满意度。

工作流程​

  1. 自动打开企业微信
  2. 通过RPA技术自动寻找并定位企业微信窗口
  3. 智能监听企业微信中用户对话，由AI模型(DeepSeek R1/Claude 3)进行上下文分析并自动回复
  4. 根据对话内容智能分类用户需求，提供针对性服务建议
  5. 复杂问题自动转接人工客服，系统进入辅助模式
  6. 辅助模式下，通过AI分析对话上下文，提取用户关键信息，生成用户画像
  7. 建立用户需求数据库，为后续服务优化提供数据支持
