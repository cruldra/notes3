做过的项目

AI全域​

一个集成多种AI能力的综合服务平台，为企业和个人用户提供一站式AI解决方案，包括智能体、工作流、课程等几大核心模块

智能体

  1. 支持国内外所有主流大语言模型
  2. 具备上下文记忆的多轮对话系统
  3. 语音输入、图像识别等多模态输入支持
  4. 知识库、插件等扩展能力

工作流

  1. 抖音批量文案采集、润色工作流
  2. 数字人视频生成工作流

课程

  1. 课程资源管理
  2. 在线学习功能

ailoveworld.cn​

ailoveworld.cn是一个支持多种大语言模型的AI对话平台,支持ChatGPT、Claude、Gemini、本地模型等多种 AI 模型，提供统一的对话界面和管理功能。

系统主要包含以下功能模块：

  • 多模型支持：集成多个主流大语言模型，支持模型配置和动态切换
  • 对话管理：支持多轮对话、上下文记忆、对话导出导入
  • 提示词系统：内置丰富的提示词模板，支持自定义和分享
  • 用户管理：多用户系统、权限控制、使用额度管理
  • 系统监控：模型调用统计、性能监控、费用统计

AI智剪工作流​

一个专门用于生成数字人视频的工具，实现了从文案采集到ai润色、配音、视频生成、视频剪辑等全流程的自动化，支持多个采集渠道、ai润色模型及数字人视频生成模型。

亳州市固废处理系统​

一套用于管理亳州市固体废弃物全生命周期的智能化管理系统，实现固废从产生、收集、运输到处理的全过程监管。

系统主要包含以下功能模块：

固废管理：固废分类、来源管理、处置跟踪、处理记录 企业监管：处置企业备案、资质审核、评级管理 运输管理：车辆调度、运输轨迹、电子联单 数据分析：处理量统计、环境影响评估、预警分析 移动应用：企业APP、监管APP，支持现场执法和实时监控

中国空军运维系统​

由中国电子科技集团公司第二十八研究所(CETC-28)为中国空军开发的大型硬件设备运维系统.

这套系统的主要目的是对全国各地空军基地的硬件设备进行远程运维管理,包括设备状态监控、故障报警、远程控制、数据采集、数据分析等.

上海市食品安全追溯系统​

一套用于全面监管上海市食品安全的追溯系统,实现从食品原料、生产、流通到消费全过程的追溯管理.

系统主要包含以下功能模块:

  • 食品安全追溯:对食品从原料采购到生产加工、仓储物流、销售等环节进行全程追溯
  • 企业管理:食品生产企业、经营企业的备案管理
  • 监管管理:食品抽检、风险预警、应急处置等监管功能
  • 信息查询:为消费者提供食品安全信息查询服务

安昂商城​

一个支持百万级并发的大型分布式电商交易系统,采用微服务架构,实现商品管理、订单处理、支付结算、库存管理等核心功能.

系统主要包含以下功能模块:

  • 商品中心:商品管理、类目管理、规格管理、价格管理
  • 订单中心:订单处理、购物车、支付管理、售后服务
  • 用户中心:会员管理、积分系统、优惠券管理
  • 库存中心:库存管理、仓储物流、供应商管理
  • 营销中心:秒杀活动、满减优惠、团购拼团、直播带货

货车通​

一站式物流服务微信公众号.主要包括如下功能 1.用户管理 2.物流点入驻 3.周边物流点 4.货物发布 5.寻找货物 6.反馈/关于 7.二维码推广

互粮通​

一款粮油信息化APP,主要包括采购、供应、服务、行情和资讯等几大核心业务板块

潮惠高速公路质量监控子系统​

这是潮惠高速公路施工质量监控系统下的一个子系统,用于采集潮惠高速公路各施工标段上所有沥青试验室的试验结果数据并上传到中央数据库作为道路施工质量的一个标准,通过串口采集试验仪器上的结果数据,试验数据类型包括燃烧炉、马歇尔、针入度、软化点和延度五大类

北京市社会科学院管理系统​

主要包括学者介绍、机构设置、科研立项、成果介绍、科研动态等几大核心模块： （1）学者介绍：展示社科院内部学者的基本信息，学者的角色主要分为研究员、副研究员及青年学者。 （2）机构设置：展示社科院下属各研究所(文化所、历史所等)、职能部门(科研处、机关党委、纪检监察处等)、直属机构及科研基地的详细信息。 （3）科研立项：院内学者通过该链接申请科研立项，具体包括立项申请、立项审批、科研项目论文、学术研讨会等。 （4）科研动态：科研动态详细展示了各科研立项的最新进展。 （5）成果介绍：展示每年各科研立项的最终成果。具体包括立项科研课题、历年科研成果及历年获奖成果

"念想" App​

一个用于家族关系管理的移动端App,支持Android和iOS端.

这套系统意在建立一个全球华人关系网,将中国家庭中的父母妻子姑叔伯舅姨等各种亲属关系建立维护起来形成一个家族树.并基于这个家族关系扩展,整个系统包括家族树、纪念馆、在线祭祀、家族谱、亲友圈和即时通讯等几大模块.
