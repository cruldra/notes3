AI智能销售助手系统服务合同

甲方： 地址： 联系人： 联系电话：

乙方： 北京华坤锦绣科技有限公司 （注：若实际签约主体为技术方案中的“子午线高科智能科技”，请在此处统一修改） 地址： 北京市昌平区国风美唐综合楼5号楼901室 联系人： 柴华 联系电话： 15810632627

根据《中华人民共和国民法典》及其他相关法律法规的规定，乙方向甲方提供智能销售助手技术解决方案，甲乙双方经友好协商一致，就“智能销售助手项目”达成如下采购协议，以兹共同遵守。

一、 项目服务内容​

1. 项目概述 智能销售助手是一款面向线上课程训练营销售团队的 AI 驱动客户关系管理系统。

2. 项目功能需求（本次合同实施范围） 本合同涵盖 P0 — 核心功能（MVP 必须） 的开发与交付：

F1 客户智能列表（批次管理视图）： 本期营/长期池双池视图，按 AI 评分排序，多维筛选。

F2 客户360°视图（记忆辅助）： 基本信息、AI意向分析、训练营进度、聊天记录检索、快捷操作。

F3 AI 意向分析引擎： 四维度评分（价格/需求/共识/信任，各0-100），综合评分，S/A/B 自动分级。

F4 智能提醒系统： 紧急/重要/常规/长期池四级提醒，收单倒计时，未回复预警。

F5 企业微信数据同步： 实时同步客户信息、聊天记录、行为数据；支持企微+星程+抖音+电话录音。

【重要备注】：原需求文档中的 P1 — 重要功能（包含 F6 AI话术推荐、F7 未成交原因分析、F8 长期池智能激活、F9 主管视图与异常预警）为后续迭代内容，不在本次一期合同及预算范围内，将另行立项计费。

二、 项目服务时间​

技术服务周期： 16周（如客户中间有其他服务需求，服务周期经双方同意后另行协商）。

  1. 阶段计划： | 阶段 | 周期 | 核心交付 | | :--- | :--- | :--- | | Phase 1: 基础设施 | 3周 | 数据库设计、企微同步、消息队列、开发环境 | | Phase 2: AI 引擎 | 7周 | 评分Agent、知识库、提示词优化、模型路由 | | Phase 3: 后端 API | 8周（并行） | 全量API、提醒系统、看板接口 | | Phase 4: 前端 | 10周（并行） | React管理后台、Tauri侧边栏、交互开发 | | Phase 5: 测试部署 | 4周 | 集成测试、压力测试、UAT、上线培训 |

项目费用总额： 人民币 620,000 元，（大写：陆拾贰万元整）。 此费用不含云服务器、AI Token 费、企微接口费（如会话存档等）等第三方运营费用。

  • 验收款（30%）： 在项目开发的第16周完成生产部署，且甲方出具 UAT 测试通过记录并签收验收报告后 5 个工作日内，甲方支付人民币 186,000 元（大写：壹拾捌万陆仟元整）。
  • 尾款（10%）： 3个月免费维保服务期满后 5 个工作日内，甲方支付剩余尾款人民币 62,000 元（大写：陆万贰仟元整）。

四、 项目交付及验收​

项目验收将基于 “镜像+文档+实测” 的综合标准进行交付。

软件交付： 提供编译后的 Docker 镜像、数据库初始化包及 Tauri 企微侧边栏客户端安装包。

文档交付： 提交《部署运维手册》《用户操作手册》《系统架构设计文档》及《API 接口文档》。

质量与验收交付： * 提交 UAT 验收报告（包含数据同步、AI 评分等核心业务流程的测试通过记录）。

  • 提交 AI 效果验证报告（验证 AI 意向评分与实际逻辑一致性）。

五、 售后服务与支持​

乙方在本协议签订之日起，提供以下售后服务：

服务周期： 系统上线验收通过后，提供 3 个月 免费维保服务。

技术支持： 通过电话、微信、远程工具等提供日常技术支持服务。

故障修复（SLA）： 产品出现问题时，乙方须在接报后 2 小时内响应；对于导致系统宕机或核心业务中断的严重故障，须在 24 小时内提供解决方案并修复。

六、 双方的权利与义务​

甲方的权利与义务：

  1. 甲方有权要求乙方按照本协议约定提供相关服务，享有定制交付物的完整使用权。

  2. 甲方应及时向乙方提供完成服务所需的业务资料、接口配置及第三方账号（包括但不限于云服务器环境、企业微信会话存档开通及相关双重授权配合、AI 模型 API Key）。甲方未按时提供导致项目延期的，乙方交付时间顺延。

  3. 甲方不得使用乙方提供的服务生产违背法律的内容，由此产生的一切后果由甲方自行承担。

  4. 甲方应按合同约定按时支付服务费用。

乙方的权利与义务：

  1. 乙方应按本协议约定内容及时间提供服务，并提供必要的技术支持确保服务质量。

  2. 乙方有权向甲方收取约定的服务费用。

  3. 知识产权： 本项目定制开发产生的前后端业务代码及相关设计文档，其知识产权在甲方付清全款后归甲方所有；乙方在开发中使用的自有底层通用架构和开源组件，其知识产权归原所有者或乙方所有，但乙方授予甲方永久免费的使用权。

七、 保密条款及数据安全​

  1. 双方均应对在合作过程中知悉的对方商业秘密予以严格保密，未经双方书面同意不得向第三方泄露、公开或不正当地使用。

数据安全： 乙方在提供服务过程中，必须采取 AES-256 加密、脱敏等必要技术手段保护甲方客户的隐私数据（如企微聊天记录、敏感个人信息）。 因乙方系统存在明显安全漏洞或乙方内部管理不善导致甲方客户数据泄露的，乙方应承担相应的赔偿责任及法律后果。

八、 违约责任​

  1. 任何一方违反本协议的约定，违约方应向守约方负责相应的违约赔偿并承担因此遭受的一切损失。

  2. 如因甲方未能按照合同约定期限付款，每逾期一日，应向乙方支付逾期款项万分之五的违约金；逾期超过 15 个工作日的，乙方有权暂停服务。

  3. 因乙方原因导致项目交付延期，每逾期一日，应向甲方支付合同总额万分之五的违约金；逾期超过 15 个工作日仍未交付可用版本的，甲方有权单方解除合同并要求退回已支付款项。

九、 协议的生效及终止​

  1. 本协议自双方签字盖章之日起生效。

  2. 任何一方违约且在合理期限内未改正，守约方有权解除本合同。

  3. 在服务完成及费用结清后，本协议自动终止。

十、 争议解决​

因本协议引起的争议，双方应友好协商解决；协商不成的，可提交北京仲裁委员会仲裁解决。

十一、 其他条款​

  1. 本协议一式两份，甲乙双方各执一份，具有同等法律效力。

  2. 本协议未尽事宜，双方可另行签订补充协议，补充协议与本协议具有同等法律效力。
