AI销售助手 - 产品需求文档 (PRD)

+:-------------------------------------------------------------+ | 产品名称: 智能销售助手 | | | | 产品版本: MVP v1.0 | | | | 创建日期: 2025-10-09 | | | | 企业: 子午线高科智能科技 | +--------------------------------------------------------------+

一、产品概述

1.1 产品定位

智能销售助手是一款面向线上课程训练营销售团队 的AI驱动客户关系管理系统，通过整合多源数据、AI智能分析和自动化提醒，帮助销售人员在4-5天训练营转化窗口 内高效管理客户、精准跟进、提升转化率。

1.2 业务背景

业务模式: 成人线上课程 - AI Agent副业培训

业务流程: 直播引流 → 4天试听训练营 → 销售跟进转化 → 购买正式课程

团队规模: 销售主管管理5-10名销售

工作负载:

每期新增学员: 120人/期

训练营频率: 每周一期（DAY0-DAY5）

长期培育池: 往期未成交学员累积增长，无清理机制

1.3 核心问题

痛点 现状描述 业务影响

120人/期记忆负担极重 每周新增120人，每人都有独特的痛点、顾虑、家庭情况，销售根本记不住 重复询问客户问题，体验差，转化率低

长期池无限累积失控 往期未成交学员持续累积（第5期已有570人），完全靠记忆不可能管理 高价值客户被埋没，激活困难

4-5天转化窗口压力大 DAY0-DAY5必须完成转化，时间紧迫，需精准识别高意向学员 低效跟进导致转化率低

意向识别靠经验判断 凭感觉判断谁是高意向，容易遗漏S/A量学员 高价值客户流失，资源浪费

数据散落多系统 企微聊天、直播数据、试听课数据分散在3-4个系统 查询效率低，准备时间长

1.4 解决方案

核心价值主张: "让销售拥有超强记忆力，精准记住每个客户的所有细节，智能识别高意向学员，高效完成转化"

四大核心能力:

智能记忆中心: 客户360°视图，完整沟通历史，聊天记录智能检索

AI意向分析: 四维度评分（0-100分），自动分级（S/A/B），关键节点识别

批次管理系统: 本期营/往期池分层，长期池智能激活

智能跟进助手: AI话术推荐，优先级提醒，未成交原因分析

Plain Text
【四维度评分】每个维度独立计算0-100分

维度1: 价格维度
├─ 分析因子: 价格反应、消费能力、付款意愿
└─ 评分规则:
├─ 90-100分: 主动询问付款方式，无价格异议
├─ 70-89分: 关注性价比，接受合理价格
├─ 50-69分: 价格敏感，希望优惠
└─ <50分: 强烈要求打折，超出预算

维度2: 需求维度
├─ 分析因子: 痛点强度、紧迫性、主动性
└─ 评分规则:
├─ 90-100分: 痛点明确强烈，多次主动咨询
├─ 70-89分: 有需求但不紧迫
├─ 50-69分: 兴趣了解，痛点模糊
└─ <50分: 被动了解，无明确需求

维度3: 共识维度
├─ 分析因子: 课程认可度、效果信心、竞品对比
└─ 评分规则:
├─ 90-100分: 全勤听课，高度认可
├─ 70-89分: 部分认可，有疑虑
├─ 50-69分: 观望态度，需要案例
└─ <50分: 质疑效果，对比竞品

维度4: 信任维度
├─ 分析因子: 沟通深度、信息分享、互动质量
└─ 评分规则:
├─ 90-100分: 愿意深入沟通，主动分享
├─ 70-89分: 有所保留，需要背书
├─ 50-69分: 戒备心中等，观察态度
└─ <50分: 戒备心强，质疑可靠性

Plain Text
【综合评分计算】
综合评分 = (价格维度 + 需求维度 + 共识维度 + 信任维度) / 4

【自动分级（根据综合评分）】
├─ S量: 综合评分 90-100分
├─ A量: 综合评分 80-89分
└─ B量: 综合评分 70-79分

【时间维度（后台维护，界面不显示）】
系统根据训练营日期自动判断学员所处阶段：
├─ DAY0之前: 训练营开始前阶段
├─ DAY0-DAY5: 训练营进行中阶段
└─ DAY5之后且未成交: 训练营结束后阶段

注意：时间维度仅用于后台逻辑判断和数据分析，
前端界面只显示S/A/B量分级标签，不显示时间阶段标签
