# 钟总AI销售客服系统报价

项目名称：钟总AI销售客服系统

项目周期：2个月

总费用：¥210,000.00

## 项目概述

本项目旨在开发一套企业微信RPA智能辅助系统，通过自动化技术与AI模型结合，实现客户对话的智能监听、分析与回复，以及客户信息的提取与用户画像生成，提高销售效率与客户转化率。

## 工作流程

1. 自动打开微信
2. 通过RPA技术自动寻找并定位微信窗口
3. 智能监听客户对话，由AI模型(DeepSeek R1/Claude 3)进行上下文分析并自动回复
4. 全程自动提取客户关键信息，实时生成和更新用户画像，用于个性化服务优化
