JSAPI

使用场景​

  1. 用户在公众号/小程序中点击支付按钮,通过JSAPI拉起微信支付

前提条件​

  1. 需要先开通商户

  2. 需要有一个通过认证的公众号/小程序

  3. 在商户管理后台中把公众号/小程序关联到商户

  4. 在小程序中关联商户
