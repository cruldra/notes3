枚举

在Python中通过继承Enum类来创建一个自定义枚举类型.

基本示例​

from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# 使用
color = Color.RED
print(color)  # Color.RED
print(color.name)  # 'RED'
print(color.value)  # 1

# 遍历
for color in Color:
    print(color)  # 输出所有枚举成员

自动编号​

from enum import Enum, auto

from enum import Enum, auto

class Status(Enum):
    PENDING = auto()
    RUNNING = auto()
    FINISHED = auto()

print(Status.PENDING.value)  # 1
print(Status.RUNNING.value)  # 2
print(Status.FINISHED.value)  # 3

复杂值的枚举​

class ApiConfig(Enum):
    GITHUB = {
        'url': 'https://api.github.com',
        'token': 'xxx'
    }
    GITLAB = {
        'url': 'https://gitlab.com/api',
        'token': 'yyy'
    }

# 使用
config = ApiConfig.GITHUB.value['url']

带方法的枚举​

class Operation(Enum):
    ADD = '+'
    SUBTRACT = '-'

    def calculate(self, x, y):
        if self is Operation.ADD:
            return x + y
        elif self is Operation.SUBTRACT:
            return x - y

# 使用
result = Operation.ADD.calculate(1, 2)  # 3

唯一性检查​

from enum import Enum, unique

@unique
class Status(Enum):
    PENDING = 1
    RUNNING = 1  # 会报错，因为值重复

枚举成员的查找​

class Animal(Enum):
    DOG = 1
    CAT = 2

# 通过名称查找
animal = Animal['DOG']  # Animal.DOG

# 通过值查找
animal = Animal(1)  # Animal.DOG

标志型枚举(Flag)​

from enum import Flag, auto

class Permissions(Flag):
    READ = auto()
    WRITE = auto()
    EXECUTE = auto()

# 可以组合使用
perm = Permissions.READ | Permissions.WRITE
print(Permissions.READ in perm)  # True
