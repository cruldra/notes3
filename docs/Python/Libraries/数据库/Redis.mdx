[RedisPy](https://redis-py.readthedocs.io/en/stable/index.html#)是最流行的`Redis Python`客户端.

## 安装

```bash
pip install redis
```

## 示例

```python
import redis

redis = redis.Redis(
            host=settings.redis_host,
            port=settings.redis_port,
            db=settings.redis_db,
            password=settings.redis_password,
            decode_responses=True
        )

redis.set("key", "value")
value = redis.get("key")
```
