Pyperclip

Pyperclip用于剪贴板操作.

安装​

pip install pyperclip

使用​

import pyperclip

# 设置剪贴板内容
pyperclip.copy("Hello, world!")

# 获取剪贴板内容
content = pyperclip.paste()
print(content)
