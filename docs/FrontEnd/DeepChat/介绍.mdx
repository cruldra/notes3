# 介绍

## Deep Chat 是一个框架无关的聊天组件，旨在将 AI 服务带入生活

### 愿景[​](#vision "Direct link to Vision")

构建一个可以与远程服务交互的自定义聊天组件并非易事。这是我们全力以赴接受的挑战，
最终发布了一个能够以最少的工作量连接到任何 API 的 Web 组件。

没有两个聊天组件会完全相同。我们理解您的 API 有独特的需求，您的网站需要新颖的用户体验。
这就是为什么 Deep Chat 被构建为完全可定制的；从交互功能到细微的样式细节 - 一切都可以更改以构建
您需要的组件。

开发者体验对这个组件的成功至关重要！这就是为什么 Deep Chat 以跨框架即插即用的包形式发布，让您
无论使用什么平台都可以在几秒钟内开始使用！此外，我们会仔细监控您的所有反馈，以帮助我们优化
您和您的用户的聊天体验，并将其作为帮助我们决定组件未来需要哪些改进的基础。

### 开源[​](#open-source "Direct link to Open source")

开源是我们工作的核心。Deep Chat 由社区构建 - 为社区服务。欢迎对这个[**项目**](https://github.com/OvidijusParsiunas/deep-chat)做出所有贡献！

如果您对增强功能有任何建议、对如何进一步推进项目有想法或发现了错误，请随时创建一个新的[**问题工单**](https://github.com/OvidijusParsiunas/deep-chat/issues/new)，我们会尽快查看！
