# 按钮

与按钮样式相关的属性。

### `customButtons`

```
类型: { `styles?:` {`button?: CustomButtonStyles`, `dropup?: CustomDropupItemStyles`}, `position?: ButtonPosition`, `tooltip?: Tooltip`, `initialState?: CustomButtonState`, `setState?: SetCustomButtonState`, `onClick?: OnClickCustomButton` }[]
```
用于定义自定义按钮对象配置的数组。
`styles` 用于定义按钮和上拉菜单项的样式选项。
`position` 用于设置按钮位置。
`tooltip` 为按钮添加工具提示。
`initialState` 是聊天加载时按钮的初始状态。
`setState` 用于以编程方式更改按钮状态。
`onClick` 用于分配在单击按钮时触发的函数。

*   示例代码
*   完整代码

```html
<deep-chat customButtons="[{}]"></deep-chat>

```

### `submitButtonStyles`

```
类型: { `submit?: ButtonStyles`, `loading?: ButtonStyles`, `stop?: ButtonStyles`, `disabled?: ButtonStyles`, `alwaysEnabled?: boolean`, `position?: ButtonPosition`, `tooltip?: Tooltip` }
```
提交按钮的自定义样式。
`submit` 状态在用户输入有效且可以发送时显示。
`loading` 在等待来自目标服务器的消息响应时使用。
`stop` 在消息响应正在流式传输时使用。
`disabled` 在用户输入无法发送（参见 validateInput）或 websocket 连接未激活时使用。
您还可以通过调用 disableSubmitButton 方法手动设置此状态。
`alwaysEnabled` 可防止使用 `disabled` 按钮状态。
`position` 设置输入区域内按钮的位置。
`tooltip` 为按钮添加工具提示。

info

`loading`、`stop` 和 `disabled` 样式将自动继承您的 `submit` 样式 `default` 状态值。

*   类型: "default" | "active" | "disabled"

### `SetCustomButtonState`

*   类型: (`newState: CustomButtonState`) => `void`

以编程方式更改按钮状态。

#### 示例

调用方法

*   代码

```javascript
chatElementRef.customButtons[0].setState('active');

```

### `OnClickCustomButton`

*   类型: (`currentState: CustomButtonState`) => `CustomButtonState` | `void`

分配一个在单击自定义按钮时触发的函数。
您可以选择返回一个 `CustomButtonState` 字符串来为按钮赋予新状态。

#### 示例

*   函数
*   命名函数

```javascript
chatElementRef.customButtons[0].onClick = (state) => (state === 'default' ? 'active' : 'default');

```

```javascript
chatElementRef.customButtons[0].onClick = function (state) {
  return state === 'default' ? 'active' : 'default';
};

```

### `Tooltip`

```
类型: `true` | {`text?: string`, `timeout?: number`, [`style?: CustomStyle`](/docs/styles#CustomStyle)}

默认: *{timeout: 0}*
```


按钮的工具提示。您可以通过将值设置为 *true* 或定义一个包含以下任意属性的对象来启用它：
`text` 是工具提示文本。
`timeout` 是工具提示出现所需的时间（毫秒）。
style 用于添加自定义样式。

#### 示例

*   示例代码
*   完整代码

```html
<deep-chat images='{"button": {"tooltip": {"text": "Upload an image", "style": {"left": "200px"}}}}'></deep-chat>

```

```html
<!-- 此示例适用于 Vanilla JS，应根据您的框架进行调整（请参阅“示例”） -->

<deep-chat
  demo="true"
  style="border-radius: 8px"

></deep-chat>

```
