# 概述

[仓库](https://github.com/olton/metroui)   [网站](https://metroui.org.ua)

欢迎使用 Metro UI - 用于创建网站的流行框架!

Metro UI 是一个免费、开源、HTML 优先的工具包,用于使用 HTML、CSS 和 JS 开发响应式网站。
使用 Metro UI,您可以轻松快速地将网站从原型制作到生产。

Metro UI 包含通用样式和 JavaScript 例程、响应式网格、布局、排版、150+ 个即用型组件和 800+ 个内置图标。
Metro UI 还包含一组用于处理日期和时间、颜色、字符串、HTML 等的库,包括用于创建响应式页面的模型和用于 SPA 的路由器。

## 特性:

* **声明式**: Metro UI 是声明式的,这意味着您可以使用简单的 HTML 属性创建复杂的界面。无需为基本交互编写 JavaScript。
* **响应式**: Metro UI 完全响应式,可适应不同的屏幕尺寸和方向。它包含响应式网格系统和媒体查询。
* **可定制**: Metro UI 高度可定制,允许您使用 CSS 变量更改颜色、大小和样式。您还可以创建自己的主题。
* **库**: Metro UI 包含九个 JavaScript 库,用于处理 HTML 元素、字符串、颜色、日期时间、路由和数据,包括钩子和响应性。
* **通用样式**: Metro UI 提供一组用于排版、弹性盒、间距、大小和其他元素的通用样式,确保应用程序的外观和感觉一致。
* **通用 JavaScript 例程**: Metro UI 包含通用 JavaScript 例程,使您的工作更轻松。
* **响应式网格**: Metro UI 包含响应式网格系统,允许您轻松创建灵活的布局。
* **150+ 组件**: Metro UI 包含 150+ 个用于任何目的的组件。
* **深色和浅色**: Metro UI 开箱即支持深色-浅色切换,包括相关组件。
* **Metro 图标字体**: 包含 800+ 个精心挑选的图标,适用于任何目的。
* **无依赖**: Metro UI 没有依赖项,因此您可以将其与任何框架或库一起使用。
