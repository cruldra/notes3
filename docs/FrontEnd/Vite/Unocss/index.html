Unocss

安装​

1. 安装 unocss

pnpm i -D unocss

2. 在vite.config.ts中配置

import UnoCSS from 'unocss/vite'
export default defineConfig(configEnv => {
    return {
        plugins: [
            UnoCSS(),
        ],
    }
});

3. 新建uno.config.ts

import { defineConfig} from 'unocss'

export default defineConfig({
  presets: [],
  shortcuts: [
    // 可以在这里添加自定义快捷方式
  ],
  rules: [
    // 可以在这里添加自定义规则
  ],
})

4. 在main.ts中引入

import 'uno.css'

@unocss/preset-attributify预设​

启用属性化模式的预设,允许使用HTML属性的方式来添加样式 例如:

<div bg="blue-400" text="sm white">

而不是

<div class="bg-blue-400 text-sm text-white">

1. 安装

pnpm i -D @unocss/preset-attributify

2. 在uno.config.ts中配置

import { defineConfig, presetAttributify } from 'unocss'
export default defineConfig({
  presets: [
    presetAttributify(),
  ],
})

3. 新建一个uno.d.ts文件

为了防止ts报错,如图:

import type { AttributifyAttributes } from '@unocss/preset-attributify'


declare module 'react' {
  interface HTMLAttributes<T> extends AttributifyAttributes {}
}

@unocss/reset预设​

用于统一不同浏览器的默认样式,确保跨浏览器的一致性

1. 安装

pnpm i -D @unocss/reset

2. 在uno.config.ts中配置

import { defineConfig, presetAttributify } from 'unocss'
export default defineConfig({
  presets: [
    presetReset(),
  ],
})

@unocss/preset-uno预设​

用于提供基础的样式规则,如颜色、字体、间距等

1. 安装

pnpm i -D @unocss/preset-uno

2. 在uno.config.ts中配置

import { defineConfig, presetUno } from 'unocss'
export default defineConfig({
  presets: [
    presetUno(),
  ],
})

3. 在main.tsx中引入

import '@unocss/reset/normalize.css';
