subscribeWithSelector

简介​

subscribeWithSelector中间件用于发布状态片段的变化.它可以让你只关注感兴趣的状态片段的变化,这有助于防止不必要的组件重新渲染.

一个简单的例子​

import {subscribeWithSelector} from "zustand/middleware";
import create from "zustand";
import {StateCreator} from "zustand/vanilla";
//定义状态的类型
type TestStore = {
    count: number;
    avg: number;
    incrementCount: () => void;
    decrementCount: () => void;
};
//创建存储库
const createStore: StateCreator<TestStore> = (
    set, get
) => ({
    count: 0,
    avg: 0,
    incrementCount: () => set((state) => ({count: state.count + 1})),
    decrementCount: () => set((state) => ({count: state.count - 1})),
    incrementAvg: () => set((state) => ({avg: state.avg + 1})),
    decrementAvg: () => set((state) => ({avg: state.avg - 1})),
})

export const useTestStore = create<TestStore>(
    subscribeWithSelector(createStore),
); 
import {useTestStore} from './store';
import {useEffect} from 'react'

export default () => {
    const [count, avg, incrementCount, decrementCount, incrementAvg, decrementAvg] = useTestStore((s) => [
        s.count,
        s.avg,
        s.incrementCount,
        s.decrementCount,
        s.incrementAvg,
        s.decrementAvg
    ]);

    useEffect(() => {
        const unsubscribe = useTestStore.subscribe(
            (s) => s.count,
            (a) => {
                alert(`count:${a}`)
            },
        );
        return () => {
            unsubscribe();
        };
    }, [])
    return <div>
        <h1>count:{count}</h1>
        <button onClick={incrementCount}>+</button>
        <button onClick={decrementCount}>-</button>
        <h1>avg:{avg}</h1>
        <button onClick={incrementAvg}>+</button>
        <button onClick={decrementAvg}>-</button>
    </div>;
}

上面的例子中使用subscribeWithSelector中间件来创建状态库:

export const useTestStore = create<TestStore>(
    subscribeWithSelector(createStore),
);

然后使用subscribe方法来订阅状态片段的变化:

import {useTestStore} from './store';

useEffect(() => {
    const unsubscribe = useTestStore.subscribe(
        (s) => s.count,
        (a) => {
            alert(`count:${a}`)
        },
    );
    return () => {
        unsubscribe();
    };
}, [])

现在点击Count下面的+和-号时,浏览器弹框会显示count的值,而点击Avg下面的+和- 号时不会有任何反应,这是因为我们只订阅了count的变化.

订阅多个状态片段​

subscribe方法的第一个参数可以是一个数组,用于订阅多个状态片段的变化:

import {useTestStore} from './store';

useEffect(() => {
    const unsubscribe = useTestStore.subscribe(
        (s) => [s.count,s.avg],
        ([count,avg]) => {
            alert(`count:${count},avg:${avg}`)
        },
    );
    return () => {
        unsubscribe();
    };
}, [])
