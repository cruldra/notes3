选择器

CSS选择器用来在页面上查找符合条件的元素,或者决定样式规则的生效范围.

标签选择器​

标签选择器是最简单的选择器,它直接给定元素的标签名,然后匹配所有该类型的元素.

示例

<div>
    <p>
        <span>1</span>
        <span>2</span>
    </p>
</div>
  1. 找到所有div元素
div
  1. 找到所有span元素
span

类选择器​

选择包含特定样式的元素

示例

<div class="class1">
    <p class="class1 class2">
        <span>1</span>
        <span>2</span>
    </p>
</div>
  1. 找到所有class属性值包含class1的元素
.class1
  1. 也可以组合标签选择器
div.class1

属性选择器​

选择包含特定属性的元素

示例

<a href="https://www.bilibili.com/video"> B站</a>
<a href="https://www.douyin.com/video"> 抖音</a>
<form action="https://www.baidu.com" method="get">
    <input type="text" name="wd" value="hello">
    <input type="submit" value="提交">
</form>
  1. 属性值等于指定值
input[type="submit"]
  1. 属性值以指定值开头

类似于Java中的startsWith

[href^="//www.bilibili.com/video"]

关系选择器​

:has()​

用于选择包含特定元素的父元素

示例

<!-- 示例HTML结构 -->
<div class="card">
    <svg class="icon">
        <path d="M12..."/>
    </svg>
    <span>文本1</span>
</div>
<div class="card">
    <svg class="icon">
        <path d="M24..."/>
    </svg>
    <span>文本2</span>
</div>
  1. 选择包含icon元素的父元素
.card:has(svg.icon)
  1. 选择包含特定path的svg
svg.icon:has(path[d="M24..."])

兄弟选择器​

示例

<!-- 示例HTML结构 -->
<div class="container">
    <svg class="icon"></svg>
    <p>段落1</p>
    <span>文本1</span>
    <p>段落2</p>
    <span>文本2</span>
</div>

:next-sibling​

:previous-sibling​

~

~选择器用于选择指定元素之后的所有同级元素

  1. 选择icon元素之后的所有同级span元素
svg ~ span  /* 匹配 "文本1" 和 "文本2" */

+​

+选择器用于选择指定元素之后的第一个同级元素

  1. 选择icon元素之后的第一个同级span元素
svg + span  /* 匹配 "文本1" */
