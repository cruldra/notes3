简介

css预处理器是一种程序,它能让你用更强大的方式编写CSS.它在普通CSS的基础上增加编程特性,最终编译成普通的CSS文件.

主要的CSS预处理器有:

  • Sass
  • Less
  • Stylus

以sass为例,可以通过对比来解释预处理器的作用:

  1. 变量使用:
  • CSS
  • Sass
.button {
    background: #1890ff;
    color: #1890ff;
}
.title {
    color: #1890ff;
}
$primary-color: #1890ff;
.button {
    background: $primary-color;
    color: $primary-color;
}
  1. 嵌套规则:
  • CSS
  • Sass
.nav {
    background: #fff;
}
.nav .list {
    margin: 0;
}
.nav .list .item {
    padding: 10px;
}
.nav .list .item:hover {
    background: #f0f0f0;
}
.nav {
    background: #fff;
    .list {
        margin: 0;
        .item {
            padding: 10px;
            &:hover {
                background: #f0f0f0;
            }
        }
    }
}
