# Mantine 容器组件对比

## 容器组件对比表

| 特性 | Box | Container | Flex | Stack | Group | Center |
|---------|-----|-----------|------|-------|-------|--------|
| **主要用途** | 所有 Mantine 组件的基础组件 | 居中内容并限制最大宽度 | 灵活的弹性布局容器，完全可控 | 垂直方向的弹性容器 | 水平方向的弹性容器 | 垂直和水平居中内容 |
| **布局方向** | 不适用（默认不是弹性容器） | 不适用（不是弹性容器） | 水平和垂直都可以（可配置） | 仅垂直方向 | 仅水平方向 | 不适用（居中内容） |
| **默认 HTML 元素** | div | div | div | div | div | div |
| **多态性** | 是 | 否 | 是 | 否 | 否 | 是 |
| **响应式属性** | 是（样式属性） | 否（除非自定义 CSS） | 是 | 否 | 否 | 是（样式属性） |
| **间距控制** | 无内置间距 | 内边距 | gap, rowGap, columnGap | gap | gap | 无内置间距 |
| **对齐控制** | 无内置对齐 | 无内置对齐 | align, justify | align, justify | justify | 始终居中内容 |
| **弹性换行** | 不适用 | 不适用 | wrap 属性 | 否 | 是 | 不适用 |
| **等宽子元素** | 否 | 否 | 否 | 否 | 是（使用 grow 属性） | 否 |
| **尺寸约束** | 无内置约束 | 有 size 属性控制最大宽度 | 无内置约束 | 无内置约束 | 无内置约束 | 无内置约束 |
| **特殊功能** | 样式属性和多态组件的基础 | fluid 属性实现 100% 宽度 | 完全控制弹性盒布局 | 仅垂直堆叠 | preventGrowOverflow 属性 | inline 属性用于内联元素 |
| **最适合用于** | 自定义组件的基础或替代 HTML 元素 | 控制宽度的页面内容 | 有特定弹性需求的复杂布局 | 垂直堆叠元素 | 水平排列元素 | 居中单个元素或元素组 |

## 主要区别解释

1. **Box**：最基本的容器，是所有 Mantine 组件的基础。它本身不是一个布局组件，但提供了样式属性和多态功能的基础。

2. **Container**：专门设计用于水平居中内容，并控制宽度和内边距。适合需要限制特定宽度的页面内容。

3. **Flex**：最灵活的容器，提供对弹性盒属性的完全控制。可以创建水平和垂直布局，并支持响应式属性。

4. **Stack**：专门的垂直弹性容器。比 Flex 简单但灵活性较低 - 只能垂直排列项目。

5. **Group**：专门的水平弹性容器。与 Stack 类似，比 Flex 简单但灵活性较低 - 只能水平排列项目。

6. **Center**：专门用于垂直和水平居中内容的容器。主要用于对齐而非布局。

## 何时使用每种组件

- **Box**：当你需要一个带有样式属性或多态功能的基本容器时
- **Container**：当你需要水平居中内容并控制最大宽度时
- **Flex**：当你需要完全控制弹性盒属性或响应式布局时
- **Stack**：当你需要垂直堆叠元素并保持一致间距时
- **Group**：当你需要水平排列元素并保持一致间距时
- **Center**：当你需要垂直和水平居中内容时

## Flex vs Group vs Stack 对比

| 特性 | Group | Stack | Flex |
|------|-------|-------|------|
| 方向 | 水平 | 垂直 | 两者都可以 |
| 等宽子元素 | ✅ | ❌ | ❌ |
| flex-wrap 支持 | ✅ | ❌ | ✅ |
| 响应式弹性盒属性 | ❌ | ❌ | ✅ |
| 多态组件 | ❌ | ❌ | ✅ |
