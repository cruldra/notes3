开发流程​

资源管理列表页面​

  1. 确定要管理的资源,通常是数据库表,管理后台中,数据库表定义位于$projectRoot/prisma/schema.prisma
  2. 在$projectRoot/src/app/下创建资源文件夹,文件夹名称是表模型名称的复数形式,如模型名称SystemParameter,对应文件夹名称是systemParameters
  3. 在资源文件夹下创建layout.tsx文件,这是资源的布局文件,用于定义资源的布局,如导航栏,侧边栏等
  4. 在资源文件夹下创建page.tsx文件,这是资源的列表页面
  5. page.tsx的内容可以参考Refine列表页模板

资源创建页面​

  1. 在资源文件夹下创建create/page.tsx文件,这是资源的创建页面
  2. create/page.tsx的内容可以参考Refine创建页模板

资源编辑页面​

  1. 在资源文件夹下创建edit/page.tsx文件,这是资源的编辑页面
  2. edit/page.tsx的内容可以参考Refine编辑页模板
