入门

if __name__ == '__main__':
    from transformers import AutoTokenizer, AutoModelForCausalLM

    tokenizer = AutoTokenizer.from_pretrained("distilgpt2")
    model = AutoModelForCausalLM.from_pretrained("distilgpt2", output_hidden_states=True)
    print(tokenizer)
    print(model)

要运行上面的例子:

  1. 使用nvidia-smi查看CUDA版本
  2. 安装pytorch - pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126
  3. 安装transformers - pip install transformers
