等差数列

**等差数列（Arithmetic Progression, AP）**是数学中一种非常基础且重要的数列。

简单来说，如果一个数列从第二项起，每一项与它的前一项的差等于同一个常数，这个数列就叫做等差数列。这个常数被称为公差（Common Difference），通常用字母 ddd 表示。

1. 核心要素​

要确定一个等差数列，你只需要知道两个关键信息：

  • 首项 (a1a_1a1​)：数列的第一个数字。
  • 公差 (ddd)：相邻两项之间的固定差值（可以是正数、负数或零）。

直观理解：你可以把等差数列想象成爬楼梯。如果你每一步都跨越相同高度的台阶，那么你所处的高度就是一个等差数列。

2. 基本公式​

等差数列有两个最核心的数学公式，分别用于求“某一项的值”和“前 nnn 项的和”：

通项公式​

如果你想知道数列中第 nnn 个数字是多少，可以使用这个公式： an=a1+(n−1)da_n = a_1 + (n - 1)dan​=a1​+(n−1)d

  • ana_nan​：第 nnn 项的值
  • a1a_1a1​：首项
  • ddd：公差
  • nnn：项数

前 nnn 项和公式（高斯求和）​

如果你想把前 nnn 个数字全部加起来，公式如下： Sn=n(a1+an)2S_n = \frac{n(a_1 + a_n)}{2}Sn​=2n(a1​+an​)​ 或者代入通项公式展开： Sn=na1+n(n−1)d2S_n = na_1 + \frac{n(n-1)d}{2}Sn​=na1​+2n(n−1)d​

3. 举个例子​

让我们通过几个具体的例子来看看公差 ddd 是如何影响数列的：

数列示例首项 (a1a_1a1​)公差 (ddd)类型
1, 3, 5, 7, 912递增（d>0d > 0d>0）
10, 7, 4, 1, -210-3递减（d<0d < 0d<0）
5, 5, 5, 5, 550常数列（d=0d = 0d=0）

4. 等差中项​

如果在 aaa 和 bbb 中间插入一个数 AAA，使得 a,A,ba, A, ba,A,b 成等差数列，那么 AAA 称为 aaa 和 bbb 的等差中项。 计算公式非常简单，其实就是两者的平均数： A=a+b2A = \frac{a + b}{2}A=2a+b​

5. 生活中的应用​

  • 电影院座位：如果第一排有 20 个座位，每一排比前一排多 2 个，座位的排布就是一个等差数列。
  • 分期付款：如果你每个月还固定的本金，本金余额的递减（在不计复利的情况下）也是一种等差关系。
