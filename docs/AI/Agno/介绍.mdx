**Agno是一个轻量级的多模态Agent构建库。它将LLM暴露为统一API并赋予它们记忆、知识、工具和推理等超能力。**

*   **构建能够生成文本、图像、音频和视频的极速Agent**
*   **根据需要添加记忆、知识、工具和推理能力**
*   **随处运行，Agno是开源的**

这是一个能够搜索网络的Agent示例：

websearch_agent.py

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    markdown=True
)
agent.print_response("纽约有什么新鲜事？", stream=True)
```

## 主要特性

Agno简单、快速且模型无关。以下是一些主要特性：

*   **极速**: Agent创建速度比LangGraph快10,000倍(参见[性能](https://github.com/agno-agi/agno#performance))
*   **模型无关**: 使用任何模型、任何供应商，无锁定
*   **多模态**: 原生支持文本、图像、音频和视频
*   **多Agent**: 构建专业Agent团队
*   **记忆管理**: 在数据库中存储Agent会话和状态
*   **知识存储**: 使用向量数据库进行RAG或动态few-shot学习
*   **结构化输出**: 让Agent以结构化格式响应
*   **监控**: 在[agno.com](https://app.agno.com/)上实时跟踪Agent会话和性能
