OpenRouter

OpenRouter是一个提供大型语言模型端点的平台。

认证​

设置您的OPENROUTER_API_KEY环境变量。从这里获取您的密钥。

Mac:

export OPENROUTER_API_KEY=***

Windows:

# Windows命令行设置方式

示例​

将OpenRouter与您的Agent一起使用：

from agno.agent import Agent, RunResponse
from agno.models.openrouter import OpenRouter

agent = Agent(
    model=OpenRouter(id="gpt-4o"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

参数​

参数类型默认值描述
idstr"gpt-4o"用于生成响应的特定模型ID。
namestr"OpenRouter"OpenRouter代理的名称标识符。
providerstr"OpenRouter:"+id模型的提供者，将"OpenRouter"与模型ID组合。
api_keyOptional[str]-用于向OpenRouter服务验证请求的API密钥。从环境变量OPENROUTER_API_KEY中获取。
base_urlstr"https://openrouter.ai/api/v1"向OpenRouter服务发出API请求的基本URL。
max_tokensint1024在响应中生成的最大令牌数。

OpenRouter还支持OpenAI的参数。
