Agent.run()

Agent.run()函数运行代理并生成响应，可以是一个RunResponse对象或一个RunResponse对象流。

我们的许多示例使用agent.print_response()，这是一个在终端中打印响应的辅助工具。它在底层使用agent.run()。

以下是如何运行您的代理。响应被捕获在response和response_stream变量中。

from typing import Iterator
from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat
from agno.utils.pprint import pprint_run_response

agent = Agent(model=OpenAIChat(id="gpt-4o-mini"))

# 运行代理并将响应作为变量返回
response: RunResponse = agent.run("Tell me a 5 second short story about a robot")
# 运行代理并将响应作为流返回
response_stream: Iterator[RunResponse] = agent.run("Tell me a 5 second short story about a lion", stream=True)

# 以markdown格式打印响应
pprint_run_response(response, markdown=True)
# 以markdown格式打印响应流
pprint_run_response(response_stream, markdown=True)

设置stream=True以返回RunResponse对象流。

RunResponse​

Agent.run()函数返回一个RunResponse对象，或者当stream=True时返回一个Iterator[RunResponse]。它具有以下属性：
