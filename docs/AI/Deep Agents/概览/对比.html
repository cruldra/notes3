对比

比较 LangChain Deep Agents 与 Claude Agent SDK 和 Codex SDK，为你的用例选择合适的工具

本页面帮助你了解 LangChain Deep Agents 如何与 Claude Agent SDK 和 Codex SDK 进行比较。

主要差异​

LangChain Deep Agents：

  • 模型灵活性：随时切换模型提供商（Anthropic、OpenAI 或 100+ 其他），并运行评估。
  • 长期记忆：使用 Memory Store 跨会话和线程持久化上下文
  • 沙盒即工具模式：在代理外部运行时，从不同提供商的隔离沙盒中运行单个操作，或在沙盒内运行完整代理
  • 虚拟文件系统：使用可插拔后端（内存、磁盘、持久存储、沙盒）进行上下文和检查点管理
  • 生产部署：通过 LangSmith 部署或自托管使用 Agent Server
  • 可观测性：使用 LangSmith 进行原生追踪和调试

Claude Agent SDK：

  • 标准化使用 Claude：为 Claude 模型提供 Anthropic、Azure、Vertex AI 和 AWS Bedrock 的一流支持
  • 自定义托管：构建你自己的 HTTP/WebSocket 层，并在容器中运行 SDK
  • 钩子：轻松拦截和控制代理行为

Codex SDK：

  • 标准化使用 OpenAI：GPT-5.3-Codex 和 OpenAI 专用工具
  • 操作系统级沙盒模式：使用内置的 read-only、workspace-write 或 danger-full-access 模式
  • MCP 服务器模式：使用 codex mcp-server 将你的代理作为 MCP 服务器暴露
  • 可观测性：使用 OpenAI Traces
