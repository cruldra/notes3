## 成本计算工具

### 批量计算示例

```javascript
// DALL-E 成本计算器
function calculateDALLECost(model, quality, resolution, quantity) {
    const exchangeRate = 7.17; // 当前汇率
    
    const pricing = {
        'dall-e-2': {
            '1024x1024': 0.02,
            '512x512': 0.018,
            '256x256': 0.016
        },
        'dall-e-3': {
            'standard': {
                '1024x1024': 0.04,
                '1024x1792': 0.08,
                '1792x1024': 0.08
            },
            'hd': {
                '1024x1024': 0.08,
                '1024x1792': 0.12,
                '1792x1024': 0.12
            }
        }
    };
    
    let pricePerImage;
    
    if (model === 'dall-e-2') {
        pricePerImage = pricing['dall-e-2'][resolution];
    } else if (model === 'dall-e-3') {
        pricePerImage = pricing['dall-e-3'][quality][resolution];
    }
    
    const totalUSD = pricePerImage * quantity;
    const totalCNY = totalUSD * exchangeRate;
    
    return {
        pricePerImage: pricePerImage,
        totalUSD: totalUSD,
        totalCNY: totalCNY.toFixed(2),
        quantity: quantity
    };
}

// 使用示例
console.log(calculateDALLECost('dall-e-3', 'hd', '1024x1024', 100));
// 输出: 100张 HD 1024x1024 图片成本约 57元
```

## 成本优化建议

### 1. 选择合适的模型
- **DALL-E 2**: 适合批量生成、成本敏感的场景
- **DALL-E 3**: 适合需要高质量、精细控制的场景

### 2. 分辨率选择策略
- **小图标、简单图形**: 256×256 或 512×512
- **社交媒体**: 1024×1024
- **打印/高清展示**: 1024×1792 或 1792×1024

### 3. 质量设置建议
- **Standard**: 日常使用、快速原型
- **HD**: 商业用途、最终作品

### 4. 提示词优化
- 精确的提示词可以减少重复生成
- 使用英文提示词通常效果更好
- 避免模糊或矛盾的描述
