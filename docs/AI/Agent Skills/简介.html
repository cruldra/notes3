简介

原文：Agent Skills - Overview

Agent Skills 是包含指令、脚本和资源的文件夹，智能体可以发现并使用它们来更准确、更高效地完成任务。

为什么需要 Agent Skills？​

智能体的能力越来越强，但往往缺乏可靠完成实际工作所需的上下文。Skills 通过让智能体能够按需加载程序性知识以及公司、团队和用户特定的上下文来解决这个问题。拥有一组 skills 的智能体可以根据正在处理的任务扩展其能力。

对于 skill 作者：构建一次能力，跨多个智能体产品部署。

对于兼容的智能体：支持 skills 让最终用户能够开箱即用地为智能体提供新能力。

对于团队和企业：以可移植、版本控制的包形式捕获组织知识。

Agent Skills 能实现什么？​

  • 领域专业知识：将专业知识打包成可重用的指令，从法律审查流程到数据分析管道
  • 新能力：为智能体提供新能力（例如创建演示文稿、构建 MCP 服务器、分析数据集）
  • 可重复的工作流程：将多步骤任务转变为一致且可审计的工作流程
  • 互操作性：在不同的 skills 兼容智能体产品之间重用相同的 skill

采用情况​

Agent Skills 已被领先的 AI 开发工具支持，包括：

  • Agentman
  • Spring AI
  • Command Code
  • Amp
  • Databricks
  • Cursor
  • Claude
  • Claude Code
  • OpenCode
  • OpenAI Codex
  • VS Code
  • GitHub
  • 以及更多...

开放开发​

Agent Skills 格式最初由 Anthropic 开发，作为开放标准发布，并已被越来越多的智能体产品采用。该标准对更广泛生态系统的贡献开放。
