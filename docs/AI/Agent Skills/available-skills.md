# 可用技能列表

本文件列出了当前环境中所有可用的技能，分为 **用户技能** 和 **OpenCode 技能** 两大类。

---

## 用户技能 (User Skills)

这些技能是用户安装的技能，具有最高优先级。

| 名称 | 描述 | 适用场景 |
|------|------|----------|

## 技能使用优先级

当多个技能可能适用于同一任务时，请按以下顺序使用：

1. **流程技能优先** (brainstorming, debugging) - 这些决定了如何处理任务
2. **实现技能其次** (frontend-design, mcp-builder) - 这些指导执行

### 技能类型说明

- **严格型** (TDD, debugging)：严格按照要求执行，不要偏离
- **灵活型** (patterns)：根据上下文调整原则
