
```
$PSVersionTable.PSVersion
```

输出结果如下:

```
Major  Minor  Build  Revision
-----  -----  -----  --------
7      5      1      0
```


**Major** - 主要版本,表示重大版本更新,当有不向后兼容的API更改时会增加

**Minor** - 次要版本,表示功能性更新,但保持向后兼容,当增加功能但不破坏现有功能时使用

**Patch** - 补丁版本,表示bug修复和小的改进,不会添加新功能,只修复问题

**PreReleaseLabel** - 预发布标签,用于预发布版本的标识,如 alpha、beta、rc (release candidate)

**BuildLabel** - 构建标签,用于标识特定的构建版本,通常用于内部版本跟踪
