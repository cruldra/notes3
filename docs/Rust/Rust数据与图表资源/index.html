Rust 数据与图表资源

如果你需要向他人展示 Rust 的优势，以下是最权威的数据来源和可视化图表资源。

📊 权威调查报告​

1. Stack Overflow Developer Survey​

最受欢迎的编程语言调查

  • 官方网站：https://survey.stackoverflow.co/2024/
  • Rust 相关数据页面：https://survey.stackoverflow.co/2024/technology

核心数据：

  • ⭐ Rust 连续 9 年（2016-2024）成为"最受喜爱的编程语言"
  • 📈 2024 年数据：83% 的 Rust 开发者表示想继续使用（Admired rate）
  • 📊 2025 年数据：72% 的 Rust 开发者表示想继续使用

可用图表：

  • Most Admired Programming Languages（最受喜爱的编程语言）
  • Most Desired Programming Languages（最想学习的编程语言）
  • Programming Language Trends（编程语言趋势对比）
  • Top Paying Technologies（高薪技术排行）

如何使用：

  1. 访问官方页面
  2. 滚动到 "Technology" 部分
  3. 查看 "Admired and Desired" 图表
  4. 可以下载原始数据或截图

历年数据对比：

  • 2024: https://survey.stackoverflow.co/2024/technology
  • 2023: https://survey.stackoverflow.co/2023/technology
  • 2022: https://survey.stackoverflow.co/2022/technology

2. JetBrains Developer Ecosystem Survey​

开发者生态系统调查

  • 官方网站：https://www.jetbrains.com/lp/devecosystem-2024/
  • 完整报告：https://www.jetbrains.com/lp/devecosystem-2024/

核心数据：

  • 📈 JetBrains Language Promise Index：Rust 位列前三（与 TypeScript、Python 并列）
  • 🚀 增长趋势：Rust 是唯一一个在最受欢迎语言中创下新使用记录的语言
  • 💼 采用意向：11% 的开发者计划采用 Rust

可用图表：

  • Language Promise Index（语言前景指数）
  • Programming Language Adoption Trends（编程语言采用趋势）
  • Language Growth Over 5 Years（5 年增长趋势）
  • Primary Language Distribution（主要语言分布）

特色内容：

  • 详细的语言使用统计
  • 开发者薪资数据
  • 技术栈组合分析
  • AI 工具使用情况

历年报告：

  • 2024: https://www.jetbrains.com/lp/devecosystem-2024/
  • 2023: https://www.jetbrains.com/lp/devecosystem-2023/
  • 2022: https://www.jetbrains.com/lp/devecosystem-2022/

3. GitHub Octoverse​

GitHub 年度报告

  • 官方网站：https://octoverse.github.com/
  • 2024 报告：https://octoverse.github.com/2024/

核心数据：

  • 📊 Rust 项目增长率：年增长超过 50%
  • 🌍 全球开发者数量：Rust 社区规模持续扩大
  • 🔥 热门项目：Rust 在系统编程、Web3、云原生领域的应用

可用图表：

  • Top Programming Languages（顶级编程语言）
  • Fastest Growing Languages（增长最快的语言）
  • Language Trends Over Time（语言趋势）
  • Open Source Activity（开源活动统计）

特色内容：

  • 开源项目统计
  • 贡献者增长趋势
  • 语言生态系统分析
  • 行业应用案例

历年报告：

  • 2024: https://octoverse.github.com/2024/
  • 2023: https://octoverse.github.com/2023/
  • 2022: https://octoverse.github.com/2022/

4. TIOBE Index​

编程语言流行度指数

  • 官方网站：https://www.tiobe.com/tiobe-index/
  • Rust 专页：https://www.tiobe.com/tiobe-index/rust/

核心数据：

  • 📈 2024 年排名：Rust 达到历史最高排名 #13
  • 📊 增长趋势：持续上升的排名曲线
  • 🔥 市场份额：1.19%（2024 年数据）

可用图表：

  • TIOBE Index Ranking（排名趋势图）
  • Long-term History（长期历史趋势）
  • Programming Language Ratings（语言评分）

更新频率：

  • 每月更新一次
  • 提供历史数据对比

5. PYPL PopularitY of Programming Language​

编程语言流行度指数（基于 Google 搜索）

  • 官方网站：https://pypl.github.io/PYPL.html

核心数据：

  • 📊 Rust 排名：#9（2024 年）
  • 📈 增长趋势：+0.2% 年增长率
  • 🔍 搜索热度：基于 Google Trends 数据

特点：

  • 基于教程搜索量
  • 反映学习兴趣
  • 月度更新

📈 关键数据点总结​

Rust 最受欢迎​

Stack Overflow 2024：

  • 83% 的开发者想继续使用 Rust（Admired）
  • 连续 9 年排名第一
  • 比第二名高出 10+ 个百分点

可视化建议：

  • 使用柱状图展示历年 "Most Admired" 排名
  • 对比 Rust 与其他语言的 Admired 百分比
  • 展示 2016-2024 年的连续冠军记录

Rust 增长趋势​

GitHub Octoverse：

  • 年增长率超过 50%
  • 项目数量持续增加
  • 贡献者社区快速扩大

JetBrains Survey：

  • Language Promise Index 前三
  • 11% 的开发者计划采用
  • 使用率逐年上升

可视化建议：

  • 折线图展示 Rust 使用率增长
  • 对比 Rust 与其他语言的增长速度
  • 展示 Rust 项目数量变化

Rust 薪资优势​

Stack Overflow 2024：

  • Rust 开发者中位数薪资：$76,292
  • 位列高薪语言前 15 名
  • 比 JavaScript、Python 等主流语言更高

可视化建议：

  • 横向柱状图对比各语言薪资
  • 突出 Rust 的薪资优势
  • 展示薪资与经验的关系

Rust 应用领域​

主要应用：

  • 系统编程：操作系统、驱动
  • Web 后端：高性能服务
  • 区块链：Solana、Polkadot
  • 云原生：容器、边缘计算
  • 嵌入式：物联网设备

可视化建议：

  • 饼图展示 Rust 应用领域分布
  • 列举知名项目和公司
  • 展示行业采用趋势
