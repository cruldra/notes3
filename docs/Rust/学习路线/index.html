Rust 学习路线

📚 学习路线概览​

本学习路线将帮助你从零开始系统地学习 Rust 编程语言，分为基础、进阶和实战三个阶段。

🎯 第一阶段：基础入门（1-2个月）​

1. 环境搭建与工具链​

  • 安装 Rust（rustup、cargo、rustc）
  • 配置开发环境（VS Code + rust-analyzer）
  • 了解 Cargo 包管理器
  • 学习基本的 cargo 命令（new、build、run、test）

学习资源：

  • Rust 官方安装指南
  • Cargo Book

2. 基础语法​

  • 变量与可变性（let、mut）
  • 数据类型（标量类型、复合类型）
  • 函数定义与调用
  • 注释与文档注释
  • 控制流（if、loop、while、for）

实践项目：

  • 编写简单的计算器程序
  • 实现猜数字游戏

3. 所有权系统 ⭐ 核心概念​

  • 所有权规则
  • 移动（Move）语义
  • 克隆（Clone）与复制（Copy）
  • 引用与借用
  • 可变引用与不可变引用
  • 切片（Slice）类型

学习资源：

  • 所有权与借用

实践项目：

  • 实现字符串处理工具
  • 编写内存安全的数据结构

4. 结构体与枚举​

  • 结构体定义与实例化
  • 方法与关联函数
  • 元组结构体
  • 枚举类型
  • Option 和 Result 枚举
  • 模式匹配（match、if let）

学习资源：

  • 结构体

实践项目：

  • 实现一个简单的用户管理系统
  • 编写配置文件解析器

5. 集合类型​

  • Vector（动态数组）
  • String 和 &str
  • HashMap 和 BTreeMap
  • HashSet 和 BTreeSet

实践项目：

  • 实现单词计数器
  • 编写简单的数据库（内存版）

🚀 第二阶段：进阶特性（2-3个月）​

6. 错误处理​

  • panic! 宏
  • Result<T, E> 类型
  • ? 操作符
  • 自定义错误类型
  • thiserror 和 anyhow 库

实践项目：

  • 实现健壮的文件处理程序
  • 编写带错误处理的 HTTP 客户端

7. 泛型与 Trait​

  • 泛型函数与结构体
  • Trait 定义与实现
  • Trait 约束
  • Trait 对象（动态分发）
  • 关联类型
  • 默认泛型参数

实践项目：

  • 实现通用的数据容器
  • 编写插件系统

8. 生命周期​

  • 生命周期注解语法
  • 函数中的生命周期
  • 结构体中的生命周期
  • 生命周期省略规则
  • 静态生命周期

实践项目：

  • 实现引用计数的数据结构
  • 编写零拷贝的解析器

9. 智能指针与包装类型​

  • Box<T>（堆分配）
  • Rc<T> 和 Arc<T>（引用计数）
  • RefCell<T> 和 Mutex<T>（内部可变性）
  • Cow<T>（写时克隆）
  • 自定义智能指针

学习资源：

  • 包装类型

实践项目：

  • 实现链表数据结构
  • 编写线程安全的缓存系统

10. 闭包与迭代器​

  • 闭包语法与类型推断
  • 闭包捕获环境
  • Fn、FnMut、FnOnce trait
  • 迭代器 trait
  • 迭代器适配器（map、filter、fold）
  • 自定义迭代器

实践项目：

  • 实现函数式编程工具库
  • 编写数据处理管道

11. 模块系统与包管理​

  • 模块定义（mod）
  • 路径引用（use）
  • 可见性控制（pub）
  • 工作空间（workspace）
  • 发布 crate 到 crates.io

实践项目：

  • 组织大型项目结构
  • 发布自己的开源库

12. 并发编程​

  • 线程创建与管理
  • 消息传递（channel）
  • 共享状态（Mutex、RwLock）
  • Send 和 Sync trait
  • 原子类型（Atomic）
  • 线程池

实践项目：

  • 实现多线程下载器
  • 编写并发任务调度器

💡 第三阶段：异步编程与实战（2-3个月）​

13. 异步编程​

  • async/await 语法
  • Future trait
  • tokio 运行时
  • async-std 库
  • 异步 I/O
  • Stream trait

实践项目：

  • 实现异步 HTTP 服务器
  • 编写异步爬虫

14. 宏编程​

  • 声明宏（macro_rules!）
  • 过程宏基础
  • 派生宏（Derive macro）
  • 属性宏（Attribute macro）
  • 函数宏（Function-like macro）

学习资源：

  • 条件编译

实践项目：

  • 实现 ORM 框架
  • 编写代码生成工具

15. Unsafe Rust​

  • unsafe 关键字
  • 裸指针
  • 调用 unsafe 函数
  • FFI（外部函数接口）
  • 内联汇编

实践项目：

  • 封装 C 库
  • 实现高性能数据结构

16. Web 开发​

  • actix-web 框架
  • Rocket 框架
  • axum 框架
  • 数据库集成（diesel、sqlx）
  • ORM 使用
  • RESTful API 设计

学习资源：

  • actix-web 快速开始

实践项目：

  • 实现完整的 RESTful API
  • 编写博客系统后端

17. 测试与文档​

  • 单元测试
  • 集成测试
  • 文档测试
  • 基准测试（criterion）
  • 代码覆盖率
  • 文档编写（rustdoc）

实践项目：

  • 为现有项目添加完整测试
  • 编写高质量的 API 文档

18. 性能优化​

  • 性能分析工具（perf、flamegraph）
  • 内存优化
  • 编译优化选项
  • SIMD 编程
  • 零成本抽象

实践项目：

  • 优化现有项目性能
  • 实现高性能算法库

🎓 进阶方向​

系统编程​

  • 操作系统开发
  • 嵌入式开发
  • 驱动程序开发

Web3 与区块链​

  • Substrate 框架
  • 智能合约开发
  • 去中心化应用

游戏开发​

  • Bevy 引擎
  • 图形编程
  • 物理引擎

命令行工具​

  • clap 参数解析
  • 终端 UI（tui-rs）
  • 系统工具开发

📖 推荐学习资源​

官方文档​

  • The Rust Programming Language（Rust 圣经）
  • Rust by Example
  • The Rustonomicon（Unsafe Rust）

在线课程​

  • Rust Course（中文）
  • Rustlings（练习题）

社区资源​

  • Rust 中文社区
  • This Week in Rust
  • Awesome Rust

✅ 学习建议​

  1. 每天编码：保持每天至少 1-2 小时的编码练习
  2. 阅读源码：学习优秀开源项目的代码
  3. 参与社区：在论坛提问、回答问题、贡献代码
  4. 做项目：通过实际项目巩固知识
  5. 写博客：记录学习过程，加深理解
  6. 循序渐进：不要急于求成，扎实掌握每个概念

🎯 学习检查点​

基础阶段检查​

  • 能够独立编写基本的 Rust 程序
  • 理解所有权和借用规则
  • 熟练使用结构体和枚举
  • 掌握错误处理机制

进阶阶段检查​

  • 能够使用泛型和 trait 设计抽象
  • 理解生命周期注解
  • 掌握并发编程基础
  • 能够阅读和理解标准库源码

实战阶段检查​

  • 能够独立完成 Web 项目
  • 掌握异步编程
  • 能够编写高质量的库
  • 具备性能优化能力

祝你学习愉快！🦀
