## Leader 键

OpenCode 大多数快捷键使用 `leader` 键。这样可以避免与您的终端冲突。

默认情况下，`ctrl+x` 是 leader 键，大多数操作需要您先按 leader 键，然后再按快捷键。例如，要开始新会话，您需要先按 `ctrl+x`，然后按 `n`。

您不一定要为快捷键使用 leader 键，但我们建议这样做。

---

## 禁用快捷键

您可以通过在配置中添加键并将其值设置为 "none" 来禁用快捷键。

```json
{
  "$schema": "https://opencode.ai/config.json",
  "keybinds": {
    "session_compact": "none"
  }
}
```

## 桌面应用提示符快捷键

OpenCode 桌面应用的提示输入支持常见的 Readline/Emacs 风格的文本编辑快捷键。这些是内置的，目前无法通过 `opencode.json` 配置。

| 快捷键 | 操作 |
|--------|------|
| `ctrl+a` | 移动到当前行的开头 |
| `ctrl+e` | 移动到当前行的末尾 |
| `ctrl+b` | 光标向后移动一个字符 |
| `ctrl+f` | 光标向前移动一个字符 |
| `alt+b` | 光标向后移动一个单词 |
| `alt+f` | 光标向前移动一个单词 |
| `ctrl+d` | 删除光标下的字符 |
| `ctrl+k` | 删除到行尾 |
| `ctrl+u` | 删除到行首 |
| `ctrl+w` | 删除前一个单词 |
| `alt+d` | 删除下一个单词 |
| `ctrl+t` | 转置字符 |
| `ctrl+g` | 取消弹出窗口 / 中止正在运行的响应 |

## Shift+Enter

某些终端默认不会发送带有 Enter 键的修饰键。您可能需要配置终端以将 `Shift+Enter` 作为转义序列发送。

### Windows Terminal

打开您的 `settings.json`，位于：

```
%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json
```

在根级别的 `actions` 数组中添加以下内容：

```json
"actions": [
  {
    "command": {
      "action": "sendInput",
      "input": "\\u001b[13;2u"
    },
    "id": "User.sendInput.ShiftEnterCustom"
  }
]
```

在根级别的 `keybindings` 数组中添加以下内容：

```json
"keybindings": [
  {
    "keys": "shift+enter",
    "id": "User.sendInput.ShiftEnterCustom"
  }
]
```

保存文件并重新启动 Windows Terminal 或打开新标签页。

## 快捷键速查表

### 应用控制

| 快捷键 | 操作 | 默认绑定 |
|--------|------|----------|
| `app_exit` | 退出应用 | `ctrl+c`, `ctrl+d`, `<leader>q` |
| `editor_open` | 打开编辑器 | `<leader>e` |
| `theme_list` | 主题列表 | `<leader>t` |
| `sidebar_toggle` | 切换侧边栏 | `<leader>b` |
| `status_view` | 状态视图 | `<leader>s` |

### 会话管理

| 快捷键 | 操作 | 默认绑定 |
|--------|------|----------|
| `session_new` | 新建会话 | `<leader>n` |
| `session_list` | 会话列表 | `<leader>l` |
| `session_timeline` | 会话时间线 | `<leader>g` |
| `session_export` | 导出会话 | `<leader>x` |
| `session_interrupt` | 中断会话 | `escape` |
| `session_compact` | 紧凑模式 | `<leader>c` |
| `session_child_cycle` | 切换子会话 | `<leader>right` |
| `session_child_cycle_reverse` | 反向切换子会话 | `<leader>left` |
| `session_parent` | 父会话 | `<leader>up` |

| 快捷键 | 操作 | 默认绑定 |
|--------|------|----------|
| `messages_page_up` | 上翻页 | `pageup`, `ctrl+alt+b` |

| `messages_line_down` | 下一行 | `ctrl+alt+e` |
| `messages_half_page_up` | 上翻半页 | `ctrl+alt+u` |
| `messages_half_page_down` | 下翻半页 | `ctrl+alt+d` |

| `messages_last` | 最后一条消息 | `ctrl+alt+g`, `end` |
