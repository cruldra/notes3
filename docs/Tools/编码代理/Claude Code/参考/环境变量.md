# 环境变量

> 控制 Claude Code 行为的环境变量完整参考。

Claude Code 支持以下环境变量来控制其行为。在启动 `claude` 之前在 shell 中设置它们，或在 [`settings.json`](/en/settings#available-settings) 的 `env` 键下配置它们以应用于每个会话或推广到你的团队。

| 变量                                                    | 用途                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| :------------------------------------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ANTHROPIC_API_KEY`                                     | 作为 `X-Api-Key` 头部发送的 API 密钥。设置时，即使你已登录，此密钥也会代替你的 Claude Pro、Max、Team 或 Enterprise 订阅使用。在非交互模式（`-p`）中，密钥在存在时始终使用。在交互模式中，你会被提示批准一次密钥才能覆盖你的订阅。要改用订阅，运行 `unset ANTHROPIC_API_KEY`                                                                                                                                                                                                                                                                                                                                                     |
| `ANTHROPIC_AUTH_TOKEN`                                  | `Authorization` 头部的自定义值（你在这里设置的值将以 `Bearer ` 为前缀）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `ANTHROPIC_BASE_URL`                                    | 覆盖 API 端点以通过代理或网关路由请求。当设置为非第一方主机时，[MCP 工具搜索](/en/mcp#scale-with-mcp-tool-search)默认禁用。如果你的代理转发 `tool_reference` 块，设置 `ENABLE_TOOL_SEARCH=true`                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `ANTHROPIC_BEDROCK_BASE_URL`                            | 覆盖 Bedrock 端点 URL。用于自定义 Bedrock 端点或通过 [LLM 网关](/en/llm-gateway)路由时。见 [Amazon Bedrock](/en/amazon-bedrock)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `ANTHROPIC_BEDROCK_MANTLE_BASE_URL`                     | 覆盖 Bedrock Mantle 端点 URL。见 [Mantle 端点](/en/amazon-bedrock#use-the-mantle-endpoint)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| `ANTHROPIC_BETAS`                                       | 要包含在 API 请求中的额外 `anthropic-beta` 头部值的逗号分隔列表。Claude Code 已经发送它需要的 beta 头部；用这个在 Claude Code 添加原生支持之前选择加入 [Anthropic API beta](https://platform.claude.com/docs/en/api/beta-headers)。与需要 API 密钥认证的 [`--betas` 标志](/en/cli-reference#cli-flags)不同，这个变量适用于所有认证方法包括 Claude.ai 订阅                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_CUSTOM_HEADERS`                              | 添加到请求的自定义头部（`Name: Value` 格式，多个头部用换行分隔）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `ANTHROPIC_CUSTOM_MODEL_OPTION`                         | 作为自定义条目添加到 `/model` 选择器的模型 ID。用这个使非标准或网关特定的模型可选而不替换内置别名。见[模型配置](/en/model-config#add-a-custom-model-option)                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| `ANTHROPIC_CUSTOM_MODEL_OPTION_DESCRIPTION`             | `/model` 选择器中自定义模型条目的显示描述。未设置时默认为 `Custom model (<model-id>)`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `ANTHROPIC_CUSTOM_MODEL_OPTION_NAME`                    | `/model` 选择器中自定义模型条目的显示名称。未设置时默认为模型 ID                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL`                         | 见[模型配置](/en/model-config#environment-variables)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL_DESCRIPTION`             | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL_NAME`                    | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL_SUPPORTED_CAPABILITIES`  | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_OPUS_MODEL`                          | 见[模型配置](/en/model-config#environment-variables)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `ANTHROPIC_DEFAULT_OPUS_MODEL_DESCRIPTION`              | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_OPUS_MODEL_NAME`                     | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_OPUS_MODEL_SUPPORTED_CAPABILITIES`   | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_SONNET_MODEL`                        | 见[模型配置](/en/model-config#environment-variables)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `ANTHROPIC_DEFAULT_SONNET_MODEL_DESCRIPTION`            | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_SONNET_MODEL_NAME`                   | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_DEFAULT_SONNET_MODEL_SUPPORTED_CAPABILITIES` | 见[模型配置](/en/model-config#customize-pinned-model-display-and-capabilities)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `ANTHROPIC_FOUNDRY_API_KEY`                             | 用于 Microsoft Foundry 认证的 API 密钥（见 [Microsoft Foundry](/en/microsoft-foundry)）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `ANTHROPIC_FOUNDRY_BASE_URL`                            | Foundry 资源的完整基础 URL（例如 `https://my-resource.services.ai.azure.com/anthropic`）。`ANTHROPIC_FOUNDRY_RESOURCE` 的替代方案（见 [Microsoft Foundry](/en/microsoft-foundry)）                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `ANTHROPIC_FOUNDRY_RESOURCE`                            | Foundry 资源名称（例如 `my-resource`）。如果未设置 `ANTHROPIC_FOUNDRY_BASE_URL` 则需要（见 [Microsoft Foundry](/en/microsoft-foundry)）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `ANTHROPIC_MODEL`                                       | 要使用的模型设置名称（见[模型配置](/en/model-config#environment-variables)）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| `ANTHROPIC_SMALL_FAST_MODEL`                            | [已弃用] 用于后台任务的 [Haiku 类模型](/en/costs)名称                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `ANTHROPIC_SMALL_FAST_MODEL_AWS_REGION`                 | 使用 Bedrock 或 Bedrock Mantle 时覆盖 Haiku 类模型的 AWS 区域                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `ANTHROPIC_VERTEX_BASE_URL`                             | 覆盖 Vertex AI 端点 URL。用于自定义 Vertex 端点或通过 [LLM 网关](/en/llm-gateway)路由时。见 [Google Vertex AI](/en/google-vertex-ai)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `ANTHROPIC_VERTEX_PROJECT_ID`                           | Vertex AI 的 GCP 项目 ID。使用 [Google Vertex AI](/en/google-vertex-ai)时需要                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `API_TIMEOUT_MS`                                        | API 请求的超时时间（毫秒）（默认：600000，即 10 分钟；最大：2147483647）。在网络慢或通过代理路由时增加此值。超过最大值的值会溢出底层定时器并导致请求立即失败                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `AWS_BEARER_TOKEN_BEDROCK`                              | Bedrock API 密钥用于认证（见 [Bedrock API 密钥](https://aws.amazon.com/blogs/machine-learning/accelerate-ai-development-with-amazon-bedrock-api-keys/)）                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `BASH_DEFAULT_TIMEOUT_MS`                               | 长时间运行的 bash 命令的默认超时时间（默认：120000，即 2 分钟）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `BASH_MAX_OUTPUT_LENGTH`                                | bash 输出中的最大字符数，超过时从中间截断                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `BASH_MAX_TIMEOUT_MS`                                   | 模型可以为长时间运行的 bash 命令设置的最大超时时间（默认：600000，即 10 分钟）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `CCR_FORCE_BUNDLE`                                      | 设置为 `1` 强制 [`claude --remote`](/en/claude-code-on-the-web#send-local-repositories-without-github) 打包并上传你的本地仓库，即使 GitHub 访问可用                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `CLAUDECODE`                                            | 在 Claude Code 生成的 shell 环境（Bash 工具、tmux 会话）中设置为 `1`。不在 [hooks](/en/hooks) 或[状态行](/en/statusline)命令中设置。用于检测脚本是否在 Claude Code 生成的 shell 中运行                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_AGENT_SDK_DISABLE_BUILTIN_AGENTS`               | 设置为 `1` 禁用所有内置[子代理](/en/sub-agents)类型如 Explore 和 Plan。仅适用于非交互模式（`-p` 标志）。对想要空白画布的 SDK 用户有用                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `CLAUDE_AGENT_SDK_MCP_NO_PREFIX`                        | 设置为 `1` 在 SDK 创建的 MCP 服务器的工具名称上跳过 `mcp__<server>__` 前缀。工具使用它们的原始名称。仅限 SDK 使用                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `CLAUDE_AUTOCOMPACT_PCT_OVERRIDE`                       | 设置自动压缩触发的上下文容量百分比（1-100）。默认情况下，自动压缩在约 95% 容量时触发。使用较低的值如 `50` 以更早压缩。高于默认阈值的值无效。适用于主对话和子代理。此百分比与[状态行](/en/statusline)中可用的 `context_window.used_percentage` 字段对齐                                                                                                                                                                                                                                                                                                                                                                   |
| `CLAUDE_AUTO_BACKGROUND_TASKS`                          | 设置为 `1` 强制启用长时间运行的代理任务的自动后台化。启用时，子代理在运行约两分钟后被移到后台                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `CLAUDE_BASH_MAINTAIN_PROJECT_WORKING_DIR`              | 在主会话中每个 Bash 或 PowerShell 命令后返回到原始工作目录                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `CLAUDE_CODE_ACCESSIBILITY`                             | 设置为 `1` 保持原生终端光标可见并禁用反文本光标指示器。允许屏幕放大器如 macOS Zoom 跟踪光标位置                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `CLAUDE_CODE_ADDITIONAL_DIRECTORIES_CLAUDE_MD`          | 设置为 `1` 从 `--add-dir` 指定的目录加载 CLAUDE.md 文件。默认情况下，附加目录不加载记忆文件                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `CLAUDE_CODE_API_KEY_HELPER_TTL_MS`                     | 凭证应刷新的间隔毫秒数（使用 [`apiKeyHelper`](/en/settings#available-settings)时）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_CODE_AUTO_COMPACT_WINDOW`                       | 设置用于自动压缩计算的 token 上下文容量。默认为模型的上下文窗口：标准模型为 200K 或[扩展上下文](/en/model-config#extended-context)模型为 1M。在 1M 模型上使用较低的值如 `500000` 以将窗口视为 500K 用于压缩目的。该值以模型的实际上下文窗口为上限。`CLAUDE_AUTOCOMPACT_PCT_OVERRIDE` 作为此值的百分比应用。设置此变量将压缩阈值与状态行的 `used_percentage` 解耦，后者始终使用模型的完整上下文窗口                                                                                                   |
| `CLAUDE_CODE_AUTO_CONNECT_IDE`                          | 覆盖自动 [IDE 连接](/en/vs-code)。默认情况下，Claude Code 在受支持的 IDE 集成终端内启动时自动连接。设置为 `false` 以防止此行为。设置为 `true` 在自动检测失败时强制连接尝试，如 tmux 遮蔽父终端时                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `CLAUDE_CODE_CERT_STORE`                                | TLS 连接的 CA 证书源的逗号分隔列表。`bundled` 是随 Claude Code 提供的 Mozilla CA 集。`system` 是操作系统信任库。默认是 `bundled,system`。需要原生二进制发行版才能与系统集成。在 Node.js 运行时上，无论此值如何，仅使用捆绑集                                                                                                                                                                                                                                                                                                                                                                                               |
| `CLAUDE_CODE_CLIENT_CERT`                               | 用于 mTLS 认证的客户端证书文件路径                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_CODE_CLIENT_KEY`                                | 用于 mTLS 认证的客户端私钥文件路径                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_CODE_CLIENT_KEY_PASSPHRASE`                     | 加密的 CLAUDE_CODE_CLIENT_KEY 的密码短语（可选）                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `CLAUDE_CODE_DEBUG_LOGS_DIR`                            | 覆盖调试日志文件路径。尽管名称如此，这是文件路径而不是目录。需要通过 `--debug` 或 `/debug` 单独启用调试模式：仅设置此变量不会启用日志记录。[`--debug-file`](/en/cli-reference#cli-flags)标志同时完成这两项。默认为 `~/.claude/debug/<session-id>.txt`                                                                                                                                                                                                                                                                                                                                                                          |
| `CLAUDE_CODE_DEBUG_LOG_LEVEL`                           | 写入调试日志文件的最低日志级别。值：`verbose`、`debug`（默认）、`info`、`warn`、`error`。设置为 `verbose` 以包含高容量诊断如完整状态行命令输出，或提高到 `error` 以减少噪音                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `CLAUDE_CODE_DISABLE_1M_CONTEXT`                        | 设置为 `1` 禁用 [100 万上下文窗口](/en/model-config#extended-context)支持。设置时，100 万模型变体在模型选择器中不可用。适用于有合规要求的企业环境                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `CLAUDE_CODE_DISABLE_ADAPTIVE_THINKING`                 | 设置为 `1` 为 Opus 4.6 和 Sonnet 4.6 禁用[自适应推理](/en/model-config#adjust-effort-level)。禁用时，这些模型回退到由 `MAX_THINKING_TOKENS` 控制的固定思考预算                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `CLAUDE_CODE_DISABLE_ATTACHMENTS`                       | 设置为 `1` 禁用附件处理。带 `@` 语法的文件提及作为纯文本发送而不是扩展为文件内容                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `CLAUDE_CODE_DISABLE_AUTO_MEMORY`                       | 设置为 `1` 禁用[自动记忆](/en/memory#auto-memory)。设置为 `0` 在逐渐推出期间强制开启自动记忆。禁用时，Claude 不创建或加载自动记忆文件                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `CLAUDE_CODE_DISABLE_BACKGROUND_TASKS`                  | 设置为 `1` 禁用所有后台任务功能，包括 Bash 和子代理工具上的 `run_in_background` 参数、自动后台化和 Ctrl+B 快捷键                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_CODE_DISABLE_CLAUDE_MDS`                        | 设置为 `1` 防止将任何 CLAUDE.md 记忆文件加载到上下文中，包括用户、项目和自动记忆文件                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `CLAUDE_CODE_DISABLE_CRON`                              | 设置为 `1` 禁用[定时任务](/en/scheduled-tasks)。`/loop` 技能和 cron 工具变得不可用，任何已调度的任务停止触发，包括会话中途运行的任务                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS`                | 设置为 `1` 从 API 请求中剥离 Anthropic 特定的 `anthropic-beta` 请求头部和 beta 工具 schema 字段（如 `defer_loading` 和 `eager_input_streaming`）。当代理网关拒绝带有"Unexpected value(s) for the `anthropic-beta` header"或"Extra inputs are not permitted"等错误的请求时使用。标准字段（`name`、`description`、`input_schema`、`cache_control`）被保留                                                                                                                                                                                                                                                                   |
| `CLAUDE_CODE_DISABLE_FAST_MODE`                         | 设置为 `1` 禁用[快速模式](/en/fast-mode)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `CLAUDE_CODE_DISABLE_FEEDBACK_SURVEY`                   | 设置为 `1` 禁用"Claude 做得怎么样？"会话质量调查。当设置 `DISABLE_TELEMETRY` 或 `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC` 时也会禁用调查。见[会话质量调查](/en/data-usage#session-quality-surveys)                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `CLAUDE_CODE_DISABLE_FILE_CHECKPOINTING`                | 设置为 `1` 禁用文件[检查点](/en/checkpointing)。`/rewind` 命令将无法恢复代码更改                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `CLAUDE_CODE_DISABLE_GIT_INSTRUCTIONS`                  | 设置为 `1` 从 Claude 的系统提示中移除内置的 commit 和 PR 工作流指令以及 git 状态快照。使用你自己的 git 工作流技能时有用。设置时优先于 [`includeGitInstructions`](/en/settings#available-settings)设置                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `CLAUDE_CODE_DISABLE_LEGACY_MODEL_REMAP`                | 设置为 `1` 防止在 Anthropic API 上将 Opus 4.0 和 4.1 自动重新映射到当前 Opus 版本。当你有意想要固定到旧模型时使用。重新映射不在 Bedrock、Vertex 或 Foundry 上运行                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `CLAUDE_CODE_DISABLE_MOUSE`                             | 设置为 `1` 禁用[全屏渲染](/en/fullscreen)中的鼠标跟踪。使用 `PgUp` 和 `PgDn` 的键盘滚动仍然有效。用这个保持终端的原生复制选择行为                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC`              | 等同于同时设置 `DISABLE_AUTOUPDATER`、`DISABLE_FEEDBACK_COMMAND`、`DISABLE_ERROR_REPORTING` 和 `DISABLE_TELEMETRY`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `CLAUDE_CODE_DISABLE_NONSTREAMING_FALLBACK`             | 设置为 `1` 在流式请求中途失败时禁用非流式回退。流式错误传播到重试层。当代理或网关导致回退产生重复工具执行时有用                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `CLAUDE_CODE_DISABLE_OFFICIAL_MARKETPLACE_AUTOINSTALL`  | 设置为 `1` 跳过首次运行时自动添加官方插件市场                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `CLAUDE_CODE_DISABLE_TERMINAL_TITLE`                    | 设置为 `1` 禁用基于对话上下文的自动终端标题更新                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
