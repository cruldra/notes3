# 命令

> Claude Code 中可用命令的完整参考，包括内置命令和捆绑技能。

命令在会话内控制 Claude Code。它们提供了一种快速方式来切换模型、管理权限、清除上下文、运行工作流等。

输入 `/` 查看所有可用命令，或输入 `/` 后跟字母进行过滤。

下表列出了 Claude Code 包含的所有命令。标记为 **[Skill](/en/skills#bundled-skills)** 的条目是捆绑技能。它们使用与你自己编写的技能相同的机制：交给 Claude 的提示，Claude 也可以在相关时自动调用。其他都是内置命令，其行为编码在 CLI 中。要添加你自己的命令，见[技能](/en/skills)。

并非每个命令都对每个用户可见。可用性取决于你的平台、计划和环境。例如，`/desktop` 仅在 macOS 和 Windows 上显示，`/upgrade` 仅在 Pro 和 Max 计划上显示。

在下表中，`<arg>` 表示必需参数，`[arg]` 表示可选参数。

## MCP 提示

MCP 服务器可以暴露显示为命令的提示。这些使用 `/mcp__<server>__<prompt>` 格式，并从连接的服务器动态发现。详情见 [MCP 提示](/en/mcp#use-mcp-prompts-as-commands)。
