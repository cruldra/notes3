## 前置条件

**包管理器：uv**

Serena 由 `uv` 管理。如果你尚未安装，请按照[官方指南](https://docs.astral.sh/uv/getting-started/installation/)进行安装。

**语言相关依赖**

使用语言服务器后端时，可能需要安装额外的依赖以支持特定语言。
查看[语言支持](language-servers)页面了解支持的语言列表。
许多依赖由 Serena 即时安装，但如果某种语言需要手动提供依赖，会在对应语言下方的备注中说明。

## 安装与初始化 Serena

确保 `uv` 已安装并在 PATH 中后，使用以下命令安装 Serena：

    uv tool install -p 3.13 serena-agent@latest --prerelease=allow

安装完成后，`serena` 命令应该已可在终端中使用。

要测试安装并初始化 Serena，运行以下命令之一：

- `serena init` — 如果你打算使用默认的语言智能后端（语言服务器）
- `serena init -b JetBrains` — 如果你打算使用 JetBrains 后端（需配合 [JetBrains 插件](025_jetbrains_plugin)）

> **提示**：你可以随时通过 Serena 的[配置](050_configuration)切换后端。

## 更新 Serena

要更新到最新版本，运行：

    uv tool upgrade serena-agent --prerelease=allow

> **提示**：想及时了解更新？定期打开 [Serena 仪表盘](060_dashboard)，我们会在那里发布新版本及其功能和改进。

## 卸载 Serena

使用以下命令卸载 Serena：

    uv tool uninstall serena-agent
