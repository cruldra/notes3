# 网络搜索工具

OMP 提供多提供商搜索和智能内容提取，支持代码仓库、包管理器、安全数据库等多种专业来源。

## 搜索提供商

### 支持的提供商

| 提供商 | 特点 | 配置 |
|--------|------|------|
| `auto` | 自动选择最佳提供商 | 默认 |
| `exa` | 高质量内容提取 | `EXA_API_KEY` |
| `brave` | 隐私保护搜索 | `BRAVE_API_KEY` |
| `jina` | AI 摘要 | 免费 |
| `kimi` | 中文优化 | `KIMI_API_KEY` |
| `zai` | 快速搜索 | `ZAI_API_KEY` |
| `anthropic` | Claude 集成 | 内置 |
| `perplexity` | 引用来源 | `PERPLEXITY_API_KEY` |
| `gemini` | Google 搜索 | `GOOGLE_API_KEY` |
| `codex` | 代码专用 | OpenAI |
| `synthetic` | 模拟搜索（离线） | 无需配置 |

### 切换提供商

```
> /search provider brave
已切换到 Brave Search

> 搜索 "React 18 新特性"
```

## 搜索配置

### 默认设置

```json
// ~/.omp/settings.json
{
  "search": {
    "defaultProvider": "auto",
    "maxResults": 8,
    "timeout": 30000,
    "safeSearch": true,
    "includeImages": false
  }
}
```

### 提供商配置

```json
{
  "search": {
    "providers": {
      "exa": {
        "apiKey": "${EXA_API_KEY}",
        "highlights": true,
        "includeDomains": ["github.com", "stackoverflow.com"]
      },
      "brave": {
        "apiKey": "${BRAVE_API_KEY}",
        "extraSnippets": true
      },
      "perplexity": {
        "apiKey": "${PERPLEXITY_API_KEY}",
        "model": "sonar-pro"
      }
    }
  }
}
```

### 环境变量

```bash
export EXA_API_KEY="your-exa-key"
export BRAVE_API_KEY="your-brave-key"
export PERPLEXITY_API_KEY="your-perplexity-key"
export KIMI_API_KEY="your-kimi-key"
```

## 故障排除

### 问题：搜索结果质量差

**切换提供商：**

```
> /search provider perplexity
> 重新搜索
```

**优化查询：**

```
❌ "python help"
✅ "Python asyncio context manager best practices 2024"
```

### 问题：API 限制

**检查配额：**

```
> /search status

提供商状态：
- exa: 98/1000 剩余
- brave: 无限
- jina: 免费
```

**使用免费提供商：**

```json
{
  "search": {
    "defaultProvider": "jina"  // 免费
  }
}
```

### 问题：内容提取失败

**使用备用方法：**

```
> 使用 browser 工具打开页面
> 提取页面内容
```

## 最佳实践

1. **明确查询**：具体的问题得到更好的答案
2. **多提供商对比**：不同提供商有不同优势
3. **验证来源**：重要信息交叉验证
4. **缓存结果**：常用搜索启用缓存

```json
{
  "search": {
    "cache": {
      "enabled": true,
      "ttl": 3600  // 1小时
    }
  }
}
```
