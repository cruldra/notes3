## 搜索配置

### 默认设置

```json
// ~/.omp/settings.json
{
  "search": {
    "defaultProvider": "auto",
    "maxResults": 8,
    "timeout": 30000,
    "safeSearch": true,
    "includeImages": false
  }
}

提供商配置​

{
  "search": {
    "providers": {
      "exa": {
        "apiKey": "${EXA_API_KEY}",
        "highlights": true,
        "includeDomains": ["github.com", "stackoverflow.com"]
      },
      "brave": {
        "apiKey": "${BRAVE_API_KEY}",
        "extraSnippets": true
      },
      "perplexity": {
        "apiKey": "${PERPLEXITY_API_KEY}",
        "model": "sonar-pro"
      }
    }
  }
}

环境变量​

export EXA_API_KEY="your-exa-key"
export BRAVE_API_KEY="your-brave-key"
export PERPLEXITY_API_KEY="your-perplexity-key"
export KIMI_API_KEY="your-kimi-key"

最佳实践​

  1. 明确查询：具体的问题得到更好的答案
  2. 多提供商对比：不同提供商有不同优势
  3. 验证来源：重要信息交叉验证

  "search": {
    "cache": {
      "enabled": true,

    }
