# 浏览器自动化工具

OMP 集成了 Puppeteer 浏览器自动化，配备 14 种反检测脚本，支持复杂的网页交互和数据提取。

## 核心特性

### 14 种 Stealth 插件

| 插件 | 功能 |
|------|------|
| `chrome-runtime` | 修复 Chrome Runtime 指纹 |
| `iframe-content-window` | 隐藏 iframe 自动化标记 |
| `media-codec` | 模拟真实编解码器支持 |
| `navigator-languages` | 设置真实语言列表 |
| `navigator-permissions` | 模拟权限 API |
| `navigator-plugins` | 添加真实插件列表 |
| `navigator-vendor` | 设置正确的 vendor |
| `navigator-webdriver` | 移除 webdriver 标记 |
| `sourceurl` | 隐藏源代码 URL |
| `user-agent-override` | 自定义 User Agent |
| `webgl` | 修改 WebGL 指纹 |
| `window-outerdimensions` | 设置窗口尺寸 |
| `chrome-csi` | 模拟 Chrome CSI API |
| `chrome-load-times` | 模拟加载时间 API |

### User Agent 模拟

自动移除 `HeadlessChrome` 标识：

```
原始: Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 
      (KHTML, like Gecko) HeadlessChrome/120.0.0.0 Safari/537.36

模拟: Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 
      (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36
```

## 模式切换

### 无头模式 / 可见模式

```
> /browser headless false

切换到可见模式，浏览器窗口将显示。

> /browser headless true

切换回无头模式。
```

### 配置默认模式

```json
// ~/.omp/settings.json
{
  "browser": {
    "headless": true,
    "viewport": {
      "width": 1920,
      "height": 1080
    },
    "userAgent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)...",
    "stealth": true
  }
}
```

## 故障排除

### 问题：被检测为机器人

**加强 stealth：**

```json
{
  "browser": {
    "stealth": true,
    "extraPlugins": ["user-agent-override", "webgl"],
    "viewport": {
      "width": 1920,
      "height": 1080
    }
  }
}
```

**添加延迟：**

```
> 随机延迟 1-3 秒
> 模拟人类鼠标移动
```

### 问题：元素找不到

**使用无障碍树：**

```
> 获取无障碍快照
> 使用显示的 ID 交互
```

**等待元素：**

```
> 等待 ".dynamic-content" 出现
> 超时 10 秒
```

### 问题：页面加载超时

```json
{
  "browser": {
    "navigationTimeout": 60000,
    "waitUntil": "networkidle2"
  }
}
```

## 安全注意事项

⚠️ **重要提醒：**

1. **不要在公共网站滥用**：遵守网站的 robots.txt 和服务条款
2. **保护敏感信息**：不要记录密码或 API 密钥
3. **遵守法律**：仅用于合法的数据抓取和测试目的
4. **注意频率**：避免对同一网站频繁请求
