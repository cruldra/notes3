Certbot

Certbot是一个免费的开源软件工具,用于自动获取和更新Let's Encrypt证书,以便为网站启用HTTPS

安装​

Ubuntu​

apt update && \
apt install certbot python3-certbot-nginx

如果出现

解决办法是运行以下命令:

pip install --upgrade kfp

申请证书​

证书目录​

生成的证书位于/etc/letsencrypt/live/域名目录下,如图:

其中包含4个文件分别是:

  1. cert.pem - 服务器证书文件,包含域名的公钥证书
  2. chain.pem - 中间证书文件,包含中间证书颁发机构的证书
  3. fullchain.pem - 完整的证书链文件,是cert.pem和chain.pem的组合.大多数服务器配置(如Nginx)会使用这个文件
  4. privkey.pem - 私钥文件,是最重要且需要保密的文件,用于证明证书的所有权
