专业术语

和证书相关的专业术语涉及:

SSL​

SSL(Secure Sockets Layer)最早的安全协议,目前已经被淘汰，最后版本是SSL 3.0

TLS​

TLS(Transport Layer Security)是SSL的继任者,目前最新版本是TLS 1.3

HTTPS​

HTTPS(Hypertext Transfer Protocol Secure)是HTTP+SSL/TLS的组合,即HTTP协议的安全版本
