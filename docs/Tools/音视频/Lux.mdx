[lux](https://github.com/iawia002/lux)是一个用`Go`构建的快速、简单的视频下载器命令行程序.

## 使用说明

```text
名称：
   lux - 一个快速简单的视频下载器

用法：
   lux [全局选项] 命令 [命令选项] [参数...]

版本：
   0.24.1

命令：
   help, h  显示命令列表或某个命令的帮助

全局选项：
   --aria2                          使用 Aria2 RPC 下载 (默认: false)
   --aria2-addr value              Aria2 地址 (默认: "localhost:6800")
   --aria2-method value           Aria2 方法 (默认: "http")
   --aria2-token value            Aria2 RPC 令牌
   --audio-only, --ao              仅下载最佳质量的音频 (默认: false)
   --caption, -C                    下载字幕 (默认: false)
   --chunk-size value, --cs value   下载时的 HTTP 分片大小(MB) (默认: 1)
   --cookie value, -c value         Cookie
   --debug, -d                      调试模式 (默认: false)
   --end value                      指定播放列表或文件输入的结束项 (默认: 0)
   --episode-title-only, --eto      哔哩哔哩每集文件名不包含播放列表标题 (默认: false)
   --file value, -F value           URL 文件路径
   --file-name-length value         文件名最大长度, 0 表示无限制 (默认: 255)
   --info, -i                       仅显示信息 (默认: false)
   --items value                    指定要下载的文件或播放列表项。用逗号分隔，如: 1,5,6,8-10
   --json, -j                       打印提取的 JSON 数据 (默认: false)
   --multi-thread, -m               使用多线程下载单个视频 (默认: false)
   --output-name value, -O value    指定输出文件名
   --output-path value, -o value    指定输出路径
   --playlist, -p                   下载播放列表 (默认: false)
   --refer value, -r value          使用指定的 Referrer
   --retry value                    下载失败时重试次数 (默认: 10)
   --silent, -s                     最小化输出 (默认: false)
   --start value                    指定播放列表或文件输入的起始项 (默认: 1)
   --stream-format value, -f value  选择特定流格式下载
   --thread value, -n value         下载线程数(仅适用于多部分视频) (默认: 10)
   --user-agent value, -u value     使用指定的 User-Agent
   --youku-ccode value             优酷 ccode (默认: "0502")
   --youku-ckey value              优酷 ckey (默认: "7B19C0AB...")
   --youku-password value          优酷密码
   --help, -h                       显示帮助 (默认: false)
   --version, -v                    打印版本 (默认: false)
```

## 示例

```bash
# 快手
lux https://www.kuaishou.com/short-video/3x2masgrsjrr9ia

# b站
lux https://www.bilibili.com/video/BV14VyUYeEk1
```
