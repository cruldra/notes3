在 Docusaurus 中隐藏底部的页脚（Footer）有多种方式，最直接的方法是在配置文件中禁用它。

## 方法一：在配置文件中移除（推荐）

这是最标准的方法。打开 `docusaurus.config.ts`（或 `docusaurus.config.js`），找到 `themeConfig` 下的 `footer` 属性，将其注释掉或删除。

```typescript title="docusaurus.config.ts"
export default {
  // ...
  themeConfig: {
    // 注释掉 footer 配置即可隐藏页脚
    /* footer: {
      style: 'dark',
      links: [...],
      copyright: `...`,
    }, */
  },
};
```

## 方法二：使用 CSS 隐藏

如果你只想在特定情况下通过样式隐藏页脚，或者不想修改配置文件，可以在 `src/css/custom.css` 中添加以下代码：

```css title="src/css/custom.css"
.footer {
  display: none;
}
```

## 方法三：条件隐藏

如果你需要根据页面内容动态隐藏页脚，通常需要使用 Docusaurus 的 `swizzle` 功能来包装 `Footer` 组件，并根据当前的路由或页面元数据决定是否渲染。

```bash
pnpm docusaurus swizzle @docusaurus/theme-classic Footer --wrap
```

然后在生成的组件中添加判断逻辑。

---

**注意**：在大多数个人笔记或文档项目中，如果不需要展示版权信息和外链，直接使用方法一是最简洁的。
