# Figma基础入门

Figma是一款基于云端的界面设计和协作工具，它提供了与传统设计软件如Sketch和Adobe XD类似的功能，但最大的特点是完全在线，可以通过浏览器随时随地访问，支持多人实时协作。

## 什么是Figma？

Figma是一款全平台可使用的设计软件，不再受系统的限制，只要你可以打开网页，理论上基本可以使用这款工具：

- 它不需要使用苹果笔记本电脑也能使用
- 它储存文件不需要占用本地内存
- 它可以将你的设计以一个链接的形式分享给所有人

## Figma的主要功能亮点

### 1. 全平台使用

Figma最大的特点是可以在Mac、Windows PC、Linux计算机甚至Chromebook上使用，不再受系统的限制。只要你可以打开网页，理论上基本可以使用这款工具。

### 2. 云端文件

在Figma创建的文件全部都是保存在你的账号上的，不用占用本地内存。这意味着你可以在任何设备上访问你的设计文件，而不必担心文件同步或存储问题。

### 3. 历史版本

Figma自动保存功能可以让你随时回到之前的设计版本。免费版最多保存30天的历史版本，专业或团队版则无限制。这使得你可以轻松恢复到任何之前的设计状态。

### 4. 共享协作

Figma可以将任何文件、页面或画板基于不同权限（查看、编辑权限）进行共享。文件就是一个链接，非常轻便。团队成员想查看或修改你的作品，你直接分享一个链接即可。

### 5. 实时协作

基于共享功能，Figma允许多人同时在同一个设计文件上工作：

- 可以多人在一个页面共同完成一个项目
- 可以实时查看他人的设计进度
- 可以评论他人的设计
- 可以进行线上实时设计评审

## Figma界面介绍

Figma的界面直观简洁，主要由以下几部分组成：

1. **顶部菜单栏**：包含文件、编辑、视图等常用功能
2. **左侧面板**：包含页面、图层管理
3. **右侧面板**：包含设计属性、原型设置等
4. **中央画布**：主要设计工作区域
5. **工具栏**：包含各种设计工具，如选择、框架、形状等

## 开始使用Figma

### 创建账号

1. 访问[Figma官网](https://www.figma.com/)
2. 点击"Sign up"注册账号
3. 可以使用Google账号、Apple账号或邮箱注册

### 创建新文件

1. 登录Figma后，点击"New File"创建新文件
2. 选择适合的画布尺寸（如手机、平板、网页等）
3. 开始设计

### 基本操作

- **缩放**：使用鼠标滚轮或按住Ctrl/Cmd + 滚轮
- **平移**：按住空格键 + 鼠标拖动
- **选择**：点击对象或拖动选择框
- **复制**：按住Alt/Option + 拖动对象
- **调整大小**：选中对象后拖动控制点
