动画的产生是元素的视觉属性随时间变换的结果。让我们来看看变换元素的五种常见方式。

已经熟悉如何在 React 中应用 CSS 变换?[跳过本页](https://www.remotion.dev/docs/animating-properties)。

## 5 种基本变换

从左到右:不透明度、缩放、倾斜、平移、旋转

### 不透明度 (Opacity)

不透明度决定了元素的可见程度。`0` 表示完全不可见,`1` 表示完全可见。中间的值会使元素半透明,下方的元素将可见。

你可以使用 `opacity` 属性设置元素的不透明度。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    opacity: 0.5,
  }}
/>
```

opacity: `1`

小于 `0` 和大于 `1` 的值也会被接受,但不会产生进一步的效果。

### 缩放 (Scale)

缩放决定了元素的大小。`1` 是自然大小,`2` 会使元素的高度和宽度都变为两倍。
小于 `1` 的值会使元素变小。`0` 使元素不可见。小于 `0` 的值也会被接受,会使元素再次变大,但会被镜像翻转。

你可以使用 `scale` 属性设置元素的缩放。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    scale: 0.5,
  }}
/>
```

scale: `1`

与使用 `height` 和 `width` 改变元素大小不同,使用 `scale()` 不会改变其他元素的布局。

### 倾斜 (Skew)

倾斜元素会导致扭曲的外观,就像元素的两个角被拉伸了一样。倾斜接受一个角度,可以使用 `rad`(弧度)和 `deg`(度)来指定。

你可以使用 `transform` 属性设置元素的倾斜。

请查看下面的示例以了解倾斜如何影响元素。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    transform: `skew(20deg)`,
  }}
/>
```

skew: `0`

### 平移 (Translate)

平移元素意味着移动它。平移可以在 X、Y 甚至 Z 轴上进行。变换可以用 `px` 指定。

你可以使用 `transform` 属性设置元素的平移。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    transform: `translateX(100px)`,
  }}
/>
```

translateX: `0`
translateY: `0`

与使用 `margin-top` 和 `margin-left` 改变元素位置不同,使用 `translate()` 不会改变其他元素的位置。

### 旋转 (Rotate)

通过旋转元素,你可以使其看起来像围绕其中心旋转了一样。旋转可以用 `rad`(弧度)或 `deg`(度)指定,你可以围绕 Z 轴(默认)旋转元素,也可以围绕 X 和 Y 轴旋转。

你可以使用 `transform` 属性设置元素的旋转。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    transform: `rotate(45deg)`, // 与 rotateZ(45deg) 相同
  }}
/>
```

rotateZ: `0`

如果你想围绕 X 或 Y 轴旋转元素,应该对父元素应用 [`perspective`](https://developer.mozilla.org/en-US/docs/Web/CSS/perspective) 属性。

如果你不想围绕中心旋转,可以使用 [`transform-origin`](https://developer.mozilla.org/en-US/docs/Web/CSS/transform-origin) 属性来改变旋转的原点。

注意,当旋转 SVG 元素时,默认情况下变换原点是左上角。你可以通过添加 `style={{transformBox: 'fill-box', transformOrigin: 'center center'}}` 来获得与其他元素相同的行为。

## 多重变换

通常,你希望组合多个变换。如果它们使用不同的 CSS 属性,如 `transform` 和 `opacity`,只需在 `style` 对象中指定两个属性即可。

如果两个变换都使用 `transform` 属性,请指定多个用空格分隔的变换。

**MyComponent.tsx**

```tsx
<div
  style={{
    height: 100,
    width: 100,
    backgroundColor: 'red',
    transform: `translateX(100px) scale(2)`,
  }}
/>
```

注意顺序很重要。变换按照它们指定的顺序应用。

## 使用 `makeTransform()` 辅助函数

安装 [`@remotion/animation-utils`](https://www.remotion.dev/docs/animation-utils) 以[获取类型安全的辅助函数](https://www.remotion.dev/docs/animation-utils/make-transform)来生成 `transform` 字符串。

```tsx
import {makeTransform, rotate, translate} from '@remotion/animation-utils';

const transform = translate(50, 50);
// => "translate(50px, 50px)"

const multiTransform = makeTransform([rotate(45), translate(50, 50)]);
// => "rotate(45deg) translate(50px, 50px)"
```

## 更多变换对象的方式

这些只是一些基本的变换。以下是一些更多可能的变换:

- `<div>` 的高度和宽度
- 使用 `border-radius` 的元素圆角边缘
- 使用 `box-shadow` 的元素阴影
- 使用 `color` 和 [`interpolateColors()`](https://www.remotion.dev/docs/interpolate-colors) 的颜色
- 使用 [`evolvePath()`](https://www.remotion.dev/docs/paths/evolve-path) 的 SVG 路径演变
- [动态字体](https://twitter.com/JNYBGR/status/1598983409367683072)的粗细和倾斜
- `linear-gradient` 的停止点
- CSS `filter()` 的值
