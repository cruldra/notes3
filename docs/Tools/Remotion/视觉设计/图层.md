## 图层

与普通网站不同,视频具有固定的尺寸。这意味着,使用 `position: "absolute"` 是完全可以的!

在 Remotion 中,你经常需要将元素"分层"叠加在一起。  
这是视频编辑器和 Photoshop 中的常见模式。

一个简单的方法是使用 [`<AbsoluteFill>`](https://www.remotion.dev/docs/absolute-fill) 组件:

**MyComp.tsx**

```tsx
import React from 'react';
import {AbsoluteFill, Img, staticFile} from 'remotion';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <AbsoluteFill>
        <Img src={staticFile('bg.png')} />
      </AbsoluteFill>
      <AbsoluteFill>
        <h1>This text appears on top of the video!</h1>
      </AbsoluteFill>
    </AbsoluteFill>
  );
};
```

这将在图片上方渲染文本。

如果你想只在特定时长内显示某个元素,可以使用 [`<Sequence>`](https://www.remotion.dev/docs/sequence) 组件,它默认也是绝对定位的。

**MyComp.tsx**

```tsx
import React from 'react';
import {AbsoluteFill, Img, staticFile, Sequence} from 'remotion';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Sequence>
        <Img src={staticFile('bg.png')} />
      </Sequence>
      <Sequence from={60} durationInFrames={40}>
        <h1>This text appears after 60 frames!</h1>
      </Sequence>
    </AbsoluteFill>
  );
};
```

在树中位置越靠下的绝对定位元素,在图层堆栈中的层级就越高。  
如果你了解这一行为,就可以利用它,并在大多数情况下避免使用 `z-index`。
