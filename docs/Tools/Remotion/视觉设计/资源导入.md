## 导入资源

要在 Remotion 中导入资源,请在项目中创建一个 `public/` 文件夹,并使用 [`staticFile()`](https://www.remotion.dev/docs/staticfile) 来导入它。

```txt
my-video/
├─ node_modules/
├─ public/
│  ├─ logo.png
├─ src/
│  ├─ MyComp.tsx
│  ├─ Root.tsx
│  ├─ index.ts
├─ package.json
```

**src/MyComp.tsx**

```tsx
import {Img, staticFile} from 'remotion';

export const MyComp: React.FC = () => {
  return <Img src={staticFile('logo.png')} />;
};
```

## 使用图片

使用 Remotion 的 [`<Img/>`](https://www.remotion.dev/docs/img) 标签。

**MyComp.tsx**

```tsx
import {Img, staticFile} from 'remotion';

export const MyComp: React.FC = () => {
  return <Img src={staticFile('logo.png')} />;
};
```

你也可以传入一个 URL:

**MyComp.tsx**

```tsx
import {Img} from 'remotion';

export const MyComp: React.FC = () => {
  return <Img src="https://picsum.photos/id/237/200/300" />;
};
```

## 使用图片序列

如果你有一系列图片,例如从 After Effects 或 Rotato 等程序导出的图片,你可以通过插值路径来创建动态导入。

```txt
my-video/
├─ public/
│  ├─ frame1.png
│  ├─ frame2.png
│  ├─ frame3.png
├─ package.json
```

```tsx
import {Img, staticFile, useCurrentFrame} from 'remotion';

const MyComp: React.FC = () => {
  const frame = useCurrentFrame();
  return <Img src={staticFile(`/frame${frame}.png`)} />;
};
```

## 使用视频

使用 [`<OffthreadVideo />`](https://www.remotion.dev/docs/offthreadvideo)、[`<Video />`](https://www.remotion.dev/docs/media/video) 或 [`<Html5Video />`](https://www.remotion.dev/docs/html5-video) 组件来保持时间轴与视频同步。

```tsx
import {OffthreadVideo, staticFile} from 'remotion';

export const MyComp: React.FC = () => {
  return <OffthreadVideo src={staticFile('vid.webm')} />;
};
```

通过 URL 加载视频也是可行的:

```tsx
import {OffthreadVideo} from 'remotion';

export const MyComp: React.FC = () => {
  return <OffthreadVideo src="https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4" />;
};
```

另见: [Remotion 支持哪些视频格式?](https://www.remotion.dev/docs/miscellaneous/video-formats)

## 使用音频

使用 [`<Html5Audio>`](https://www.remotion.dev/docs/html5-audio) 组件。

```tsx
import {Html5Audio, staticFile} from 'remotion';

export const MyComp: React.FC = () => {
  return <Html5Audio src={staticFile('tune.mp3')} />;
};
```

从 URL 加载音频也是可行的:

```tsx
import {Html5Audio} from 'remotion';

export const MyComp: React.FC = () => {
  return <Html5Audio src="https://file-examples.com/storage/fe48a63c5264cbd519788b3/2017/11/file_example_MP3_700KB.mp3" />;
};
```

有关包含音频的指导,请参阅[音频指南](https://www.remotion.dev/docs/using-audio)。

## 使用 CSS

将 .css 文件与 JavaScript 源文件放在一起,并使用 `import` 语句。

```txt
my-video/
├─ node_modules/
├─ src/
│  ├─ style.css
│  ├─ MyComp.tsx
│  ├─ Root.tsx
│  ├─ index.ts
├─ package.json
```

**MyComp.tsx**

```tsx
import './style.css';
```

**注意**

想使用 SASS、Tailwind 或类似工具?[查看如何覆盖 Webpack 配置的示例](https://www.remotion.dev/docs/webpack)。

## `import` 语句

作为导入文件的另一种方式,Remotion 允许你在项目中 `import` 或 `require()` 以下几种类型的文件:

- 图片 (`.png`, `.svg`, `.jpg`, `.jpeg`, `.webp`, `.gif`, `.bmp`)
- 视频 (`.webm`, `.mov`, `.mp4`)
- 音频 (`.mp3`, `.wav`, `.aac`, `.m4a`)
- 字体 (`.woff`, `.woff2`, `.otf`, `.ttf`, `.eot`)

例如:

**MyComp.tsx**

```tsx
import {Img} from 'remotion';
import logo from './logo.png';

export const MyComp: React.FC = () => {
  return <Img src={logo} />;
};
```

### 注意事项

虽然这以前是导入文件的主要方式,但我们现在不推荐使用它,原因如下:

- 仅支持上述列出的文件扩展名。
- 最大文件大小为 2GB。
- 动态导入(如 `require('img' + frame + '.png')`)比较[棘手](https://www.remotion.dev/docs/webpack-dynamic-imports)。

如果可能,请优先使用 [`staticFile()`](https://www.remotion.dev/docs/staticfile) 导入。

## 项目外的文件

Remotion 在浏览器中运行,因此它无法访问计算机上的任意文件。  
也不能在浏览器中使用 Node.js 的 `fs` 模块。  
相反,将资源放在 `public/` 文件夹中,并使用 [`getStaticFiles()`](https://www.remotion.dev/docs/getstaticfiles) 来枚举它们。

参见[为什么 Remotion 不支持绝对路径](https://www.remotion.dev/docs/miscellaneous/absolute-paths)。

## 打包后添加资源

在渲染之前,代码会使用 Webpack 打包,之后只能访问打包的资源。  
因此,在调用 [`bundle()`](https://www.remotion.dev/docs/bundle) 后添加到 public 文件夹的资源在渲染期间将无法访问。  
但是,如果你使用[服务端渲染 API](https://www.remotion.dev/docs/ssr-node),你可以在事后向打包内的 `public` 文件夹添加资源。

## 使用 `<Img>`、`<Video>` 和 `<Audio>`

**使用 [`<Img />`](https://www.remotion.dev/docs/img) 或 [`<Gif />`](https://www.remotion.dev/docs/gif)** 而不是原生 `<img>` 标签、Next.js 的 `<Image>` 和 CSS `background-image`。  
**使用 [`<OffthreadVideo />`](https://www.remotion.dev/docs/offthreadvideo)、[`<Video />`](https://www.remotion.dev/docs/media/video) 或 [`<Html5Video />`](https://www.remotion.dev/docs/html5-video)** 而不是原生 `<video>` 标签。  
**使用 [`<Audio />`](https://www.remotion.dev/docs/media/audio) 或 [`<Html5Audio />`](https://www.remotion.dev/docs/html5-audio)** 而不是原生 `<audio>` 标签。  
**使用 [`<IFrame />`](https://www.remotion.dev/docs/iframe)** 而不是原生 `<iframe>` 标签。

通过使用 Remotion 的组件,你可以确保:

**1.** 资源在渲染帧之前完全加载

**2.** 图片和视频与 Remotion 的时间轴同步
