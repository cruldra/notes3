## 使用字体

这里有一些在 Remotion 中使用自定义字体的方法。

### 使用 `@remotion/google-fonts` 加载 Google 字体

*从 v3.2.40 版本开始可用*

[`@remotion/google-fonts`](https://www.remotion.dev/docs/google-fonts) 是一种类型安全的方式，可以直接加载 Google 字体，而无需手动创建 CSS 文件。

**MyComp.tsx**

```tsx
import { loadFont } from "@remotion/google-fonts/TitanOne";

const { fontFamily } = loadFont();

const GoogleFontsComp: React.FC = () => {
  return <div style={{ fontFamily }}>Hello, Google Fonts</div>;
};
```

### 通过 CSS 使用 Google 字体

导入 Google 字体提供的 CSS。

:::note
从 2.2 版本开始，Remotion 会自动等待字体加载完成。
:::

**font.css**

```css
@import url("https://fonts.googleapis.com/css2?family=Bangers");
```

**MyComp.tsx**

```tsx
import "./font.css";

const MyComp: React.FC = () => {
  return <div style={{ fontFamily: "Bangers" }}>Hello</div>;
};
```

### 使用 `@remotion/fonts` 加载本地字体

*从 v4.0.164 版本开始可用*

将字体文件放入你的 `public/` 文件夹中。  
在应用中执行 [`loadFont()`](https://www.remotion.dev/docs/fonts-api/load-font) 调用：

**load-fonts.ts**

```tsx
import { loadFont } from "@remotion/fonts";
import { staticFile } from "remotion";

const fontFamily = "Inter";

loadFont({
  family: fontFamily,
  url: staticFile("Inter-Regular.woff2"),
  weight: "500",
}).then(() => {
  console.log("字体已加载！");
});
```

现在可以使用该字体了：

**MyComp.tsx**

```tsx
<div style={{ fontFamily: fontFamily }}>一些文本</div>;
```

### 手动加载本地字体

你可以使用 Web 原生的 [`FontFace`](https://developer.mozilla.org/en-US/docs/Web/API/FontFace) API 来加载字体。

**load-fonts.ts**

```tsx
import { continueRender, delayRender, staticFile } from "remotion";

const waitForFont = delayRender();
const font = new FontFace(
  `Bangers`,
  `url('${staticFile("bangers.woff2")}') format('woff2')`,
);

font
  .load()
  .then(() => {
    document.fonts.add(font);
    continueRender(waitForFont);
  })
  .catch((err) => console.log("加载字体时出错", err));
```

:::note
如果你的 TypeScript 类型报错，请安装最新版本的 `@types/web` 包。
:::
