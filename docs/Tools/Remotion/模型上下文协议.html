模型上下文协议

你可以使用 Remotion 的模型上下文协议 (MCP) 客户端，让你编辑器的 AI 聊天更好地理解 Remotion。

我们的 MCP 服务器使用 CrawlChat 的技术将 Remotion 的文档索引到向量数据库中。

警告

测试阶段: 目前，此工具无需身份验证即可开放使用。如果运行此工具的成本过高，我们可能会限制使用。

Cursor​

一键安装​

手动安装​

  1. 转到你的 Cursor 设置：Cmd + Ctrl + P，然后输入 > Cursor Settings 并按 Enter 确认。

  2. 添加一个 MCP：

    {
      "mcpServers": {
        "remotion-documentation": {
          "command": "npx",
          "args": ["@remotion/mcp@latest"]
        }
      }
    }

    如果操作成功，应该会出现一个绿点，并且工具 remotion-documentation 应该会显示出来。

    如果你现在在 composer 中要求 Cursor 使用 Remotion 文档，MCP 应该会被调用。

VS Code​

一键安装​

安装 MCP 服务器 →

手动安装​

  1. 打开命令面板：Cmd + Shift + P (macOS) 或 Ctrl + Shift + P (Windows/Linux)。

  2. 输入 > MCP: Add Server 并按 Enter 确认。

  3. 选择 Add from JSON 并粘贴以下配置：

    {
      "name": "remotion-documentation",
      "command": "npx",
      "args": ["@remotion/mcp@latest"]
    }

  4. 服务器现在应该出现在 MCP 服务器列表中。你可以通过打开扩展视图 (Cmd + Shift + X / Ctrl + Shift + X) 并检查 MCP 服务器部分来验证它是否正在运行。

    如果你现在要求 GitHub Copilot Chat 使用 Remotion 文档，MCP 应该会被调用。
