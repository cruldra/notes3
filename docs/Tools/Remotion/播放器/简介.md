使用 Remotion 播放器，你可以将 Remotion 视频嵌入到任何 React 应用程序中，并在运行时自定义视频内容。

## 演示

播放视频，然后调整视频下方的参数。

> 注意：本文档为静态翻译，无法直接运行交互式演示。
>
> 查看此示例的 [源代码](https://github.com/remotion-dev/remotion/blob/main/packages/docs/components/PlayerExampleWithControls.tsx)。

## API

*   [**`<Player>`**](https://www.remotion.dev/docs/player/player)

*   [**`<Thumbnail>`**](https://www.remotion.dev/docs/player/thumbnail)
