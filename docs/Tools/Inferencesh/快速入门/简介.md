**一个让智能体真正执行任务的 AI 工作空间**

Inference.sh 让智能体处理那些繁琐的工作——它们规划步骤、挑选合适的工具或工作流，甚至在需要更多输入时自动生成小型 UI 组件。你描述目标，智能体负责执行。

## 核心特性

* **100+ 预置应用**：从 Grid 中直接访问图像生成、音频转录、文本分析等能力
* **可视化工作流构建器**：无需编码即可将多个应用串联成多步骤工作流
* **带工具的 AI 智能体**：创建能真正执行任务、而不只是聊天的助手
* **统一 API 访问**：所有应用通过一套 API 即可调用，提供 Python 和 JavaScript SDK
* **实时执行**：进度、日志和结果实时流式推送
* **随处运行**：Cloud Worker 开箱即用，Private Worker 掌控数据
* **模型原生运行时**：状态管理、重试、监控——一切在一条时间线上可观测

Inference.sh 让开发者无需管理基础设施，即可构建 AI 驱动的工作流。创建能使用工具的智能体，构建处理数据的流水线，一条命令即可部署。

工作区功能详见[工作区导览](./workspace-tour.md)。API 访问参见 [API & SDK](../api/authentication)。

## 快速上手

### 用 Python 运行一个应用

```python
from inferencesh import inference

client = inference(api_key="inf_your_key")

result = client.run({
    "app": "infsh/stable-diffusion",
    "input": {"prompt": "A serene Japanese garden"}
})

print(result["output"]["images"][0]["uri"])
```

### 用 JavaScript 运行一个应用

```typescript
import { inference } from '@inferencesh/sdk';

const client = inference({ apiKey: 'inf_your_key' });

const result = await client.run({
  app: 'infsh/stable-diffusion',
  input: { prompt: 'A serene Japanese garden' }
});

console.log(result.output.images[0].uri);
```

### 创建一个智能体（在工作区中）

1. 进入 **Agents** → **New Agent**
2. 选择 LLM（GPT-4、Claude）
3. 添加工具（stable-diffusion、whisper 等）
4. 开始对话——智能体会自动调用工具
