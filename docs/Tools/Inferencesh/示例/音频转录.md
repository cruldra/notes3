**使用 Whisper 将语音转为文字**

---

## 通过 Grid 使用

1. 进入 [Apps](https://app.inference.sh/apps)
2. 找到 `whisper`
3. 上传音频文件
4. 点击 **Run**

支持的格式：MP3、WAV、M4A、FLAC、OGG

---

## 使用 Python SDK

```python
from inferencesh import inference

client = inference(api_key="inf_your_key")

result = client.run({
    "app": "infsh/whisper",
    "input": {
        "audio": "/path/to/audio.mp3"  # 自动上传
    }
})

print(result["output"]["text"])
```

---

## 带时间戳

获取单词级或片段级时间戳：

```python
result = client.run({
    "app": "infsh/whisper",
    "input": {
        "audio": "/path/to/audio.mp3",
        "timestamps": "word"  # 或 "segment"
    }
})

for segment in result["output"]["segments"]:
    print(f"[{segment['start']:.1f}s] {segment['text']}")
```

输出：
```
[0.0s] Welcome to today's episode.
[2.1s] We're going to discuss AI agents.
[4.5s] Let's get started.
```

---

## 附带翻译

将音频翻译为英文：

```python
result = client.run({
    "app": "infsh/whisper",
    "input": {
        "audio": "/path/to/french_audio.mp3",
        "task": "translate"  # 翻译为英文
    }
})
```

## 搭配 Agent 使用

创建一个将 `whisper` 作为工具的智能体：

```
你： 转录这期播客

Agent：我来为你转录这段音频。
       [调用 whisper...]

       转录结果如下：

       "欢迎收听 AI Weekly 播客。今天我们将
       讨论生成式 AI 的最新进展……"

       全文共 2,450 字。需要我帮你总结一下吗？
```

## 模型选择

使用 **setup 参数**指定 Whisper 模型大小：

```python
client.run({
    "app": "infsh/whisper",
    "setup": {
        "model_size": "large-v3"  # tiny, base, small, medium, large, large-v3
    },
    "input": { ... }
})
```

---

## 选项

| 选项 | 可选值 | 说明 |
|------|--------|------|
| `task` | `transcribe`、`translate` | 转录或翻译为英文 |
| `language` | `en`、`es`、`fr` 等 | 强制指定语言 |
| `timestamps` | `none`、`segment`、`word` | 时间戳粒度 |

---

## 技巧

**提高准确率：**
- 使用高质量音频
- 尽可能去除背景噪音
- 已知语言时手动指定

**处理长文件：**
- Whisper 支持最长约 2 小时的文件
- 超长文件会自动分段处理
