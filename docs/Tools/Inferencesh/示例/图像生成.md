**使用 AI 模型生成图像**

---

## 通过 Grid 使用

最快的图像生成方式：

1. 进入 [Apps](https://app.inference.sh/apps)
2. 找到 `stable-diffusion` 或 `flux`
3. 输入提示词
4. 点击 **Run**

---

## 提示词示例

| 风格 | 提示词 |
|------|--------|
| 照片级写实 | "Professional headshot of a woman, studio lighting, 4k" |
| 插画 | "Cute cartoon fox in a forest, children's book style" |
| 抽象 | "Flowing waves of blue and purple, digital art" |
| 产品 | "White sneakers on marble surface, product photography" |

---

## 使用 Python SDK

```python
from inferencesh import inference

client = inference(api_key="inf_your_key")

result = client.run({
    "app": "infsh/stable-diffusion",
    "input": {
        "prompt": "A serene Japanese garden with cherry blossoms",
        "negative_prompt": "blurry, low quality",
        "num_images": 1,
        "size": "1024x1024"
    }
})

image_url = result["output"]["images"][0]["uri"]
print(f"Generated: {image_url}")
```

---

## 使用 JavaScript SDK

```typescript
import { inference } from '@inferencesh/sdk';

const client = inference({ apiKey: 'inf_your_key' });

const result = await client.run({
  app: 'infsh/stable-diffusion',
  input: {
    prompt: 'A serene Japanese garden with cherry blossoms',
    negative_prompt: 'blurry, low quality',
    num_images: 1,
    size: '1024x1024'
  }
});

console.log('Generated:', result.output.images[0].uri);
```

## 搭配 Agent 使用

创建一个将 `stable-diffusion` 作为工具的智能体：

```
你： 帮我为一家叫"Morning Brew"的咖啡店设计 logo

Agent：我来为 Morning Brew 生成一个 logo。
       [调用 stable-diffusion...]

       这是你的 logo！我生成了一个温暖、有亲和力的设计，
       包含咖啡杯和日出图案。

       需要变体或者其他风格吗？
```

## 热门图像应用

| 应用 | 最适合 |
|------|--------|
| `stable-diffusion` | 通用、灵活 |
| `flux` | 高质量、细节丰富 |
| `dalle` | 创意、艺术感 |
| `sdxl-turbo` | 快速生成 |
| `controlnet` | 引导式生成（姿态、边缘） |

## 模型配置

使用 **setup 参数**加载特定 checkpoint 或更改精度：

```python
client.run({
    "app": "infsh/stable-diffusion",
    "setup": {
        "model_id": "stabilityai/stable-diffusion-xl-base-1.0",
        "precision": "fp16"
    },
    "input": { ... }
})
```

## 技巧

**提示词要具体：**
```
❌ "a dog"
✓ "Golden retriever puppy playing in autumn leaves, warm sunlight, shallow depth of field"
```

**使用负面提示词：**
```
negative_prompt: "blurry, low quality, distorted, ugly, duplicate"
```

**尝试不同模型：**
- SDXL 追求质量
- Turbo 追求速度
- Flux 追求细节
