ScreenshotToCode

screenshot-to-code是一个开源项目,用于屏幕截图转换为代码.

支持的语言​

  • HTML + Tailwind
  • HTML + CSS
  • React + Tailwind
  • Vue + Tailwind
  • Bootstrap
  • Ionic + Tailwind
  • SVG

支持的AI模型​

  • Claude Sonnet 3.5
  • GPT-4o

安装​

使用Docker​

  1. 下载源码
git clone https://github.com/abi/screenshot-to-code.git
  1. 配置gpt4o或者claude密钥
echo "OPENAI_API_KEY=sk-your-key" > .env
echo "ANTHROPIC_API_KEY=your-key" > .env
  1. 启动
docker compose up -d --build
