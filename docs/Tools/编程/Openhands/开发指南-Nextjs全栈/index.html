开发指南-Nextjs全栈

  • Nextjs用于提供全栈能力
  • Nextjs路由在src/app/api中定义
  • 对于Nextjs路由响应,统一采用如下格式(你需要根据使用的具体语言做相应转换) 

    T = TypeVar('T')


    class Error(BaseModel):
        """表示一个错误

        Attributes:
            type: 错误类型
            message: 错误信息
            code: 错误码

        Author: dongjak
        Created: 2024/11/08
        Version: 1.0
        Since: 1.0
        """
        type: Optional[str] = Field(default=None, description="错误类型")
        message: Optional[str] = Field(default=None, description="错误信息")
        code: Optional[int] = Field(default=None, description="错误码")


    class ResponsePayloads(BaseModel, Generic[T]):
        """响应载荷"""
        data: Optional[T] = Field(default=None, description="数据")
        error: Optional[Error] = Field(default=None, description="错误信息")
  • Prisma是用于数据库操作的ORM框架,用于提供数据库操作能力
  • Prisma模型定义文件位于prisma/schema.prisma
  • Refine用于提供CRUD能力
  • Refine路由定义位于src/app/layout.tsx
  • Refine资源定义位于src/app/[resource]
  • 每个Refine资源目录中最少包含列表(page.tsx)和布局(layout.tsx)两个文件,新增(create.tsx)和编辑(edit.tsx)两个文件视具体的需求而定
  • Antd用于提供UI组件
  • 默认不做任何异常处理
