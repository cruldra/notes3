VSCode

自动保存​

{
  "files.autoSave": "afterDelay",
  "files.autoSaveDelay": 1000
}

调整界面缩放​

  1. 打开设置

    Ctrl+Shift+P (Windows/Linux)
    Cmd+Shift+P (Mac)

  2. 配置缩放

    {
      "window.zoomLevel": 1.2  // 大于1放大，小于1缩小
    }

安装Fira Code字体​

  1. 下载安装 Fira Code

    • 下载 .zip 文件，解压后进入 ttf 文件夹
    • 全选所有 .ttf 文件 → 右键 → "安装"

  2. 配置 VS Code

    {
      "editor.fontFamily": "Fira Code",
      "editor.fontLigatures": true, // 启用连字
      "editor.fontSize": 16  // 可选
    }

powershell7作为默认终端​

"terminal.integrated.profiles.windows": {
    "PowerShell": {
        "source": "PowerShell",
        "icon": "terminal-powershell",
        "path": "D:\\Applications\\PowerShell7\\pwsh.exe"
    }
}

无法连接到wsl主机​

  1. 完全退出 VSCode。
  2. 在 WSL2 终端中，输入 rm -rf ~/.vscode-server/ 并回车。
  3. 再次打开 VSCode。
