## 修改配置及缓存目录

`Jetbrains`系列`IDE`的默认目录:

| 目录类型 | 目录位置 |
| --- | --- |
| 配置目录 | `C:\Users\xxx\AppData\Roaming\JetBrains` |
| 缓存目录 | `C:\Users\xxx\AppData\Local\JetBrains` |

可以通过修改`$IDE/bin`目录下的的配置文件`idea.properties`来修改配置及缓存目录:

```properties
#配置目录
idea.config.path=D:/ProgramData/WebStorm/config

#缓存目录
idea.system.path=D:/ProgramData/WebStorm/system

#下面这两个配置取消注释即可
idea.plugins.path=${idea.config.path}/plugins
idea.log.path=${idea.system.path}/log
```

## 破解

在安装目录下找到`$product.exe.vmoptions`,然后写入:

```properties
--add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED
-javaagent:D:\Sources\ja-netfilter-V2023\ja-netfilter.jar=jetbrains
```
