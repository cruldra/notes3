在 Godot 中，你可以使用 GDScript 和 C# 编程语言编写代码。

如果你是编程新手，我们建议从 GDScript 开始，因为我们将其设计得比 C# 等通用语言更简单。它学起来会更快、更容易。

虽然 GDScript 是 Godot 特有的语言，但你通过它学到的技术也适用于其他编程语言。

请注意，程序员学习和使用多种语言是完全正常的。编程语言之间的相似之处多于不同之处，因此一旦你掌握了一种语言，你就可以更快地学习另一种语言。
