# 第一课：代码是什么样的 (What Code is Like)

学习编程可能会让人望而生畏。

然而，既然你想制作电子游戏，那么**学习编程是必经之路**。*每一个*电子游戏本质上都是一个计算机程序。

## 告诉计算机做什么 (Telling the computer what to do)

编程是编写精确指令的过程，这些指令告诉计算机如何执行一项任务。

游戏的指令例如：移动角色、绘制生命条或播放声音。

要做到这些，你需要学习一种**编程语言**：一种专门用于告诉计算机做什么的语言。

*(在此处显示：CodeProgrammingLanguage.tscn)*

编程语言与英语或西班牙语等自然语言不同。计算机不会思考。与我们不同，它无法*领会*你告诉它的内容。

你不能告诉它像“画一个圆”这样模糊的指令。

哪一个圆？在哪里？什么颜色？多大？

## 计算机需要精确的指令 (The computer needs exact instructions)

要绘制一个实心圆，计算机需要知道精确的绘制坐标、半径、厚度和颜色。

实现这一点的代码*可能*看起来像这样。

*点击按钮运行代码示例并查看结果。*

*(在此处显示：RunDrawCircle.tscn)*

在接下来的课程中，你将学习这段代码是如何工作的。

现在，我们想让你感受一下计算机代码是什么样子的。在这个例子中，一切都很重要：每个括号、大写字母、句点和逗号。

计算机总是**精确地**执行你告诉它要做的事情。不多也不少。它*盲目地*遵循每一条指令。

**当你编程时，你就是掌控者，你可以自由地做*任何*你想做的事情。**

### 测验：你如何向计算机发出指令？

- [ ] 使用简单的英语散文

> **解释**：计算机不懂像英语这样的自然语言。要让它们做任何事情，你需要使用编程语言给出它们能理解的精确指令。

## 你将学习使用 GDScript 编程 (You'll learn to code with GDScript)

在本课程中，你将学习 GDScript 编程语言（名称代表“Godot 脚本”）。

这是由游戏开发者为游戏开发者设计的语言。你可以在 Godot 游戏引擎中使用它来创建游戏和应用程序。

它是像 [Dome Keeper](https://store.steampowered.com/app/1637320/Dome_Keeper/) 和 [Brotato](https://store.steampowered.com/app/1942280/Brotato/) 这样的游戏所使用的语言。

特斯拉的工程师也用它来开发汽车仪表盘。

*(在此处显示：ImageBrotato.tscn)*

GDScript 是开始学习编程的绝佳语言，因为它是专门化的。与其他一些语言不同，它没有*压倒性*数量的功能需要你去学习。

## 大多数编程语言都是相似的 (Most programming languages are similar)

不要害怕被锁定。你在第一门编程语言中学到的概念将适用于所有其他语言。

大多数语言的相似之处多于不同之处。一旦你学会了一种，学习下一种语言并变得高效所需的时间就会少得多。

这是一个在三种语言中实现相同代码的例子：GDScript、JavaScript 和 Python。

*(在此处显示：ThreeCodeExamples.tscn)*

看起来并没有*那么*不同，是吗？

### 测验：编程语言完全不同吗？

- [x] 不，它们有很多相似之处
- [ ] 是的，它们完全不同

> **解释**：大多数编程语言都建立在相同的编程思想之上。因此，它们大多是相似的。这并不是说所有语言都是一样的。与 GDScript 相比，有些语言提供了非常独特的语法，需要完全不同的思维方式。然而，像 GDScript、Python、JavaScript、C++、C# 和许多其他语言都建立在相似的编程哲学之上。
