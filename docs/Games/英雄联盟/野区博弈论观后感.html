野区博弈论观后感

  • 打野这个位置，比的就是谁能收集更多信息，然后做出更正确的决策，最后就是谁执行的更好
  • 开局分析阵容,搞清楚阵容优劣势，是前期阵容还是后期阵容，是进攻还是防守
  • 捕捉对面开野路线(中单1分20秒对面f6放眼)
  • 如果自己是速4打野英雄(劫、人马、莉莉亚、死歌、婕拉、艾翁)要做好防止入侵的眼位(河道草丛点眼)
  • 如果对方是速4英雄，一定要想办法打乱敌方速4节奏，不然他刷的比你快，会比你更早做事情
  • 一级是否入侵取决于=>前期阵容需要进攻、一级强势、队友有线权
  • 一级入侵前有一波心理博弈，就是你决定入侵他哪个半区，就在另外一个半区漏一下视野，这样可以给对方制造错觉，减少防御心理
  • 如果是劣势，那就尽量换资源，如果是优势，那就不要让对面换到资源
  • 如何判断是否是劣势也很重要，并不是说死了一两次就是劣势，英雄技能、召唤师技能、人数、装备差都能短暂弥补劣势
  • 打野入侵几种常见方式=>1级卡1分20秒时间进f6(凯影),三狼开进红(非常规变奏),f6开进蓝(常规变奏)
  • 抓人之前如果另一个半区有野怪，那就要评估一下风险，有70%概率抓到再去，不然抓不到还留另外一个半区会很亏
  • lol是一个回合制游戏，打好进攻回合之后的防守回合，是保持优势的关键
  • 多组野怪同时存在时，先刷最有可能被对面入侵的那个半区
  • 越塔前要按沾波浪键避免失误a到塔或者小兵
  • 面对强gank类型英雄(ps、赵信、皇子、挖掘机)，宁愿自己牺牲一点，多帮队友反蹲，不要让对面进入抓人节奏,不然队友都被抓崩了，反两个野没用
  • 进别人野区前一定要多观察线上线权情况，在有线权时才能进别人野区，野区打起来时也要通过观察小地图双方动向来决定是打到底还是撤退
  • 还是要多观察线上线权情况这一点，可以捕捉到很多信息，比如兵线被卡需要帮推或者反蹲，兵线回推对面又没闪等情况可以gank,又或者是兵线进敌方塔下，敌方残血且没技能可以越塔
  • 多利用计分版收集敌方信息，比如打野刷野数量、辅助召唤师技能是否变成海克斯闪等等
  • 去强势路打架、入侵、越塔,去弱势路保护反蹲
  • 双方都是从上往下刷的情况下，刷完上半区3组野时是红色方打野强势期(因为有红buff)，此时在线权允许情况下可以去蓝色方红buff蹲对面打野
  • 玩打野千万不能哪里有野就刷哪里，一定要多观察线上打架情况，往更有危险也更有机会的那一路刷，即使对方打野在换你另外一个半区，你也能在抓死对方以后利用线权进对方另外半区换东西
  • 在下半区打输了架以后记得看下时间点和小龙，打输了人家肯定要开龙，这时候去对面半区换资源
  • 假设性原则，想到某个事，比如草里有眼、对方在打龙、对面打野一级要入侵，那就假设这件事真的发生了
  • 想好围绕哪个半区来打以后，想尽一切办法将对方打野赶到另一个半区去
  • 开局想以下几件事情:1. 往哪个半区刷? 2. 有没有入侵的机会(根据线权)? 3. 有没有被入侵的可能(防守眼) 4. 比如说围绕下半区来打，那往下半区刷边刷边等机会，此时中上如果有兵线回推的机会，就立马过去gank或者反蹲，然后再从上往下,这样能尽量照顾到三线打架都很频繁且对面打野位置不明的情况
  • 打架前先从小地图看对面人在哪，侵入前先看线权
  • 在一个半区做事的时候，如果另一个半区的队友有危险，记得给它发信号
  • 不要盯着队友的失误，去想怎么赢游戏,不要给队友压力
  • 状态好的时候打这个游戏，状态差输个一两把心态炸了就连跪了,连跪三把就可以休息会了
