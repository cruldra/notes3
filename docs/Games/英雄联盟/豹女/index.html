豹女

技能​

被动​

  • 在草丛中有移速加成
  • 面向敌人时巨额提升
  • 人形态的qw命中敌人时也有这个效果

Q​

  • q到人时条件允许别急着扑上去，利用好被动的移速加成平a拉扯，最后再w上去eq收割
  • 豹形态q可以重置普攻,比如aq
  • 豹形态q是基于已损失生命值的百分比伤害
  • 豹形态q的伤害只有升级大招时才加

W​

  • 人形态的夹子用来封走位，敌人踩中也有被动移速加成
  • 夹子造成少量ap持续伤害
  • 夹子持续2分钟，最多同时存在10个
  • 豹形态w是向前跳跃的aoe,如果人形态qw命中敌人，切到豹形态使用w会减w cd,如果豹形态又击杀敌方英雄,再减,一级就6秒cd,前期如果q能中相当于全减cd
  • 豹形态w可以取消qe施法后的僵直
  • 届时时利用w取消僵直这一机制aqw来快速清野

E​

  • 这个没什么好讲的，对拼时alt+e给自己奶一口即可
  • 为目标提供奶量及攻速加成，血量越低奶的越多
  • e提供的巨额攻速加成在后期打团豹女乏力时期给已方ad

R​

  • 从豹子切回人形态时如果qw命中敌人会刷新r

思路​

  • 野区对拼爆发连招是人形态q+w(提前放在敌人逃跑路线上)+e(提供攻速,a的更快)+a+a(利用q或w命中提供的高移速拉扯)+豹形态weq
  • 如果对拼很激烈，就要放弃一开始e的攻速把奶留到最后,因为奶是血量越低奶的越多，只有在确保自己没有危险时才会在qw命中时给自己e加攻速来提高平a的速度
  • 如果贴脸的话是豹形态eqw(向后)拉开,再切到人形态qw最后利用被动拉扯
