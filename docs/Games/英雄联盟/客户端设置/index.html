客户端设置

依据鼠标指针攻击移动​

  • 默认情况下是攻击离自己距离最近的敌人，开启这个设置变成攻击离鼠标指针最近的敌人
  • 比如下路2v2对拼,对方泰坦顶在前面，此时你距离对方ad距离比泰坦远，a键或者右键可能会打到泰坦，开启这个设置后，只要将鼠标移动到对方ad身边按a键即可打到ad

将只以英雄为目标视一个可选项​

  • 依据鼠标指针攻击移动有个问题是如果对方在兵堆里，a键有可能误a到小兵
  • 此时按~键打开这个开关，指针变成红色,此时鼠标即使在兵堆里或者防御塔内，只要鼠标放在想要打的人旁边，按a键就能打到他，在兵堆里或者越塔时可以很好避免失误
