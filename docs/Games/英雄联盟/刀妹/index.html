刀妹

技能​

被动​

  • a英雄、小兵和防御塔可以续杯
  • 面对致盲、闪避、格挡和法术被动等，都可以续被动

Q​

  • 掘墓的小鬼、蚂蚱的虫子和婕拉普通状态的花都能用q秒杀
  • q出来以后如果对方进入不可选取状态(小鱼、吸血鬼)也躲不了q的伤害
  • q能触发攻击特效和吸血、强攻、海妖
  • q死敌人或者敌人在q过程中死亡或者q有“重心不稳"buff敌人时，都会刷新q的cd
  • 不同的是q死敌人或者敌人在q过程中死亡q是秒刷，q有“重心不稳"buff敌人时会有0.2秒的cd

W​

  • 同时打到4个英雄就可以瞬间满被动
  • w也可以续被动
  • 在w1 80%左右可以w闪，但仍有机率失败,感觉刀妹w闪容易出错
  • w一旦施放不会被打断，可以用来做控制抵消,但仍然会吃到控制，只是二段w必定会放出来
  • w减伤对防御塔有效果
  • w最多在防御状态下蓄力1.5秒，但二段w伤害在0.75秒时最大化

E​

  • q死小兵时在放一段e,再q敌方英雄放二段e,最基础的qee连招
