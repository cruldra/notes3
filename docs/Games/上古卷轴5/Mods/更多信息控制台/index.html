更多信息控制台 (More Informative Console)

模组简介​

"更多信息控制台"(More Informative Console)是一个为《上古卷轴5：天际》特别版开发的用户界面增强模组，它大幅扩展了游戏控制台的功能，使玩家能够查看游戏中各种对象的详细信息。这个模组是老滚5传奇版中流行的MFG控制台模组的特别版替代品。

主要功能​

该模组可以帮助玩家：

  1. 识别模组来源：查明是哪个模组添加或移动了游戏世界中的对象
  2. 查看NPC详细信息：包括种族、等级、技能、法术、派系、AI包等
  3. 检查物品属性：武器伤害、护甲防御值、附魔效果等
  4. 分析容器内容：查看箱子、尸体等容器中的物品
  5. 查看模型和纹理：显示对象使用的3D模型和纹理文件路径
  6. 显示编辑器ID：查看对象的编辑器ID（需要配合其他模组）
  7. 查看脚本信息：显示附加在对象上的脚本

使用方法​

  1. 安装模组：将下载文件解压到游戏数据文件夹，或使用模组管理器安装
  2. 打开控制台：在游戏中按下 ~ 键打开控制台
  3. 选择对象：点击想要查看的对象（如NPC、物品、容器等）
  4. 查看信息：
    • 屏幕右下方会显示基本信息
    • 屏幕左上方会打开详细信息窗口
  5. 切换模式：按下 Tab 键可以在四种不同的信息显示模式之间切换：
    • 对象信息模式：显示选中对象的详细信息
    • 世界数据模式：显示当前单元格、世界空间、天气和音乐信息
    • MFG模式：显示选中NPC的表情信息
    • 全屏控制台模式：隐藏信息窗口，让控制台文本占据整个屏幕

信息窗口交互​

要与左上角的详细信息窗口交互：

  1. 将鼠标悬停在想要查看的条目上
  2. 按下 Shift 或 Ctrl 键（如果Shift键触发Windows粘滞键功能，可以使用Ctrl键）
  3. 带有括号中数字的条目可以展开，显示更多详细信息
  4. 如果信息太多，可以使用上下箭头按钮滚动
  5. 如果打开了太多窗口，可以使用左右箭头按钮移动窗口组

显示的信息类型​

基本信息（右下角）​

右下角显示的基本信息包括：

  • 对象名称
  • 基础表单ID和引用表单ID
  • 基础表单类型
  • 定义和最后修改对象的模组

详细信息（左上角窗口）​

根据对象类型的不同，详细信息窗口会显示各种信息，例如：

NPC信息​

  • 种族和外观数据
  • 生命值、魔法值和耐力值
  • 技能和属性值
  • 已知法术列表
  • 当前效果
  • 装备和物品栏
  • 派系关系
  • 保护状态（必要/受保护/无保护）
  • 当前AI包
  • 等级倍率信息

物品信息​

  • 武器伤害和速度
  • 护甲防御值
  • 附魔效果
  • 装备槽位
  • 关键词
  • 模型路径

容器信息​

  • 内容物列表
  • 是否为安全容器
  • 锁级别（如果有锁）
  • 所有权信息

自定义设置​

从版本0.2开始，模组提供了一个INI文件用于自定义UI：

  • 文件位置：Data/SKSE/Plugins/MoreInformativeConsole.ini
  • 可调整信息窗口的大小和位置
  • 可调整字体大小
  • 支持非1080p显示器的设置选项

控制台命令增强​

该模组集成了Better Console模组的功能，添加了以下新命令和功能：

新命令​

  • clear - 清除可见的控制台历史记录
  • ?TERM - 向上搜索历史记录中的TERM
  • /TERM - 从顶部开始向下搜索历史记录中的TERM
  • ? - 重复上一次向上搜索
  • / - 重复上一次向下搜索
  • GetQuestData id - 打开包含任务信息的窗口

新快捷键​

  • PgUp/PgDn - 上下滚动一屏
  • Shift + PgUp/PgDn - 上下滚动5屏
  • Home/End - 滚动到历史记录的顶部/底部
  • Shift + Home/End - 滚动到匹配当前搜索的下一行

模组要求​

  • SKSE (Skyrim Script Extender) - 任何与游戏版本匹配的版本
  • Address Library for SKSE Plugins - 0.40版本以上必需
  • powerofthree's Tweaks - 可选，用于显示编辑器ID

兼容性​

  • 支持所有Skyrim特别版和周年纪念版
  • 支持Skyrim VR（需要VR版本的地址库）
  • 与大多数UI模组兼容

常见问题​

问：为什么我看不到某些对象的编辑器ID？
答：默认情况下，游戏只加载少数表单类型的编辑器ID到内存中。安装powerofthree's Tweaks模组可以让游戏加载所有编辑器ID。

问：如何查看哪个模组添加了某个对象？
答：选择对象后，查看右下角的"Base defined in"（基础定义于）和"Ref defined in"（引用定义于）行。

问：如何查看NPC的派系关系？
答：选择NPC后，在左上角的详细信息窗口中展开"Factions"（派系）条目。

问：如何查看物品的附魔效果？
答：选择物品后，在左上角的详细信息窗口中展开该物品条目，然后查找并展开"Enchantment"（附魔）条目。

总结​

"更多信息控制台"是一个强大的工具，对于模组玩家和开发者都非常有用。它可以帮助你识别游戏中的对象来源，查看详细属性，以及解决模组冲突问题。对于想要深入了解游戏机制或进行模组调试的玩家来说，这是一个必备的工具。
