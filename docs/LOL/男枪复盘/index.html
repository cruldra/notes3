男枪复盘

  1. 队友全在上半区
  2. 在对面下半区没有布置视野
  3. 地图上只看一个对面的盖伦

这波为了反个红跑去送掉1千块大人头,直接导致丢掉大龙,天坑.

  1. 对面ad带的疾跑
  2. 自家辅助是个软辅
  3. 对于男枪这种特别吃前期节奏的打野来说,像这种耽误自己刷野的事是一点都不能干

这里犯了以下几个错误:

  1. 对面打野是瞎子,带的还是电刑这种爆发型符文,开局就要想到他很可能过来搞事,应该跟他镜像开野避免前期碰撞
  2. 本来交完e技能他已经成了红buff首选攻击目标,自己走位又换回去了

这里犯了以下几个错误:

  1. 红buff草里有一个眼还剩下1分钟左右,此时应该先去刷石头人,盲仔不可能1分钟不去刷野
  2. 打石头人的时候可以通过红buff那里的眼观察盲仔是否回来打我们的红
  3. 如果他敢趁我们打石头的时间来打红，那首先他开打时间要比我慢(因为他要绕妖姬视野)
  4. 其次就是我们一直有他视野,妖姬支援也更快,还有就是他已经交过闪现
  5. 所以如果他敢来打红,那他必死,就算红被他cj反掉，我也省了一个打石头人的时间,而不是像现在这样打了半天被人家反掉
  6. 为什么视野都看到了还不停手????
  1. 不打1级团的情况下一定要去草丛站位丢眼,防止对面进野区反野
  1. 在1分钟时打完1级团,1分08秒时取消回城,此时完全有时间去蓝buff草丛做个眼,这个狗比泥鳅跑来刷我蓝了

这里又犯一个很低级的精心大意的毛病:

  1. 已知已方下路劣势而且还回城了
  2. 也就是说清完这个眼也没法入侵
  3. 兵线还没到线上，没有对方下路的视野,并且是两个有控制能留人的英雄
  4. 此时上半区还有两组野且泥鳅有偷野的前科
  5. 所以打完这个河蟹应该赶紧去上半区发育而不是在这里浪费时间
  1. 要抓住稍纵即逝的机会,这一把的问题就是好多决策的时候犹豫了
  2. 武器上一波小龙团交了闪,然后又看到他出现在兵线上,ad和辅助也ping大龙,此时应该果断开大龙,本来是一个小劣的局,如果这波决策果断一些,很可能就打破平衡

总结

  1. 针对强势打野要规划好开野路线,避免被入侵
  2. 开野前想好哪路有线权，上有线权就从下开野，下有线权就从上开野
  3. 男枪发育为主,gank和控资源属于情况允许顺路
  4. 进对面野区之前一定要看自线上是否掌握线权,是否可以支援
  5. 帮队友击杀对面时看情况推线,这样既能保护残血队友推线被杀,线推进塔之后下一波兵线也会回推
