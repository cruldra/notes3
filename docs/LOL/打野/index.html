野区基本知识​

野怪刷新时间​

总结

除红蓝buff外,其他野怪刷新周期都是2分15秒

一级站位​

如果对面有强势打野且一级不打一级团的情况下,就去图中指定位置站位以观察对面打野动向

然后在52秒时按B键回城,在读到最后一秒时放置假眼,回家换扫描

tip

这么做的原因是因为野怪在1:30时刷新,在0:52秒回城,回家时是刚好1分钟,然后从泉水赶到野区大约需要20秒,这样在不耽误打野的情况下最大化视野有效期
