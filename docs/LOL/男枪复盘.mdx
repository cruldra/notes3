1. 针对强势打野要规划好开野路线,避免被入侵
2. 开野前想好哪路有线权，上有线权就从下开野，下有线权就从上开野
3. 男枪发育为主,gank和控资源属于情况允许顺路
4. 进对面野区之前一定要看自线上是否掌握线权,是否可以支援
