## 一级站位

如果对面有强势打野且一级不打一级团的情况下,就去图中指定位置站位以观察对面打野动向

![](https://github.com/cruldra/picx-images-hosting/raw/master/image.45hpme46xo.webp)

然后在`52`秒时按`B`键回城,在读到最后一秒时放置假眼,回家换扫描

:::tip
这么做的原因是因为野怪在`1:30`时刷新,在`0:52`秒回城,回家时是刚好1分钟,然后从泉水赶到野区大约需要`20`秒,这样在不耽误打野的情况下最大化视野有效期
:::


## 刷野路线

**哪路压线去哪路**

这是风险最低,收益最高的选择.能做的事情很多,比如：

1. 有硬辅可以考虑屯线越塔
2. 进对面野区反野
3. 同时相当于反蹲,打起来队友支援的概率也高一些
4. 控资源
