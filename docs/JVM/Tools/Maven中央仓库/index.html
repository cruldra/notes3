Maven中央仓库

发布自己的jar包​

  1. 使用谷歌账号登录Maven Central

  2. 创建一个命名空间,命名空间是域名反写,如com.github.zhangkaitao

  3. 验证命名空间,它会要求新建一个txt dns记录以验证域名所有权

    创建命名空间后会随机生成一个Verification Key,为了证明这个命名空间的所有权,需要添加一个DNS TXT记录,其值要与该值相同.
