第一课：代码是什么样的 (What Code is Like)

学习编程可能会让人望而生畏。

然而，既然你想制作电子游戏，那么学习编程是必经之路。每一个电子游戏本质上都是一个计算机程序。

告诉计算机做什么 (Telling the computer what to do)​

编程是编写精确指令的过程，这些指令告诉计算机如何执行一项任务。

游戏的指令例如：移动角色、绘制生命条或播放声音。

要做到这些，你需要学习一种编程语言：一种专门用于告诉计算机做什么的语言。

(在此处显示：CodeProgrammingLanguage.tscn)

编程语言与英语或西班牙语等自然语言不同。计算机不会思考。与我们不同，它无法领会你告诉它的内容。

你不能告诉它像“画一个圆”这样模糊的指令。

哪一个圆？在哪里？什么颜色？多大？

计算机需要精确的指令 (The computer needs exact instructions)​

要绘制一个实心圆，计算机需要知道精确的绘制坐标、半径、厚度和颜色。

实现这一点的代码可能看起来像这样。

点击按钮运行代码示例并查看结果。

(在此处显示：RunDrawCircle.tscn)

在接下来的课程中，你将学习这段代码是如何工作的。

现在，我们想让你感受一下计算机代码是什么样子的。在这个例子中，一切都很重要：每个括号、大写字母、句点和逗号。

计算机总是精确地执行你告诉它要做的事情。不多也不少。它盲目地遵循每一条指令。

当你编程时，你就是掌控者，你可以自由地做任何你想做的事情。

测验：你如何向计算机发出指令？​

  • 使用编程语言和精确的指令
  • 使用简单的英语散文

解释：计算机不懂像英语这样的自然语言。要让它们做任何事情，你需要使用编程语言给出它们能理解的精确指令。

你将学习使用 GDScript 编程 (You'll learn to code with GDScript)​

在本课程中，你将学习 GDScript 编程语言（名称代表“Godot 脚本”）。

这是由游戏开发者为游戏开发者设计的语言。你可以在 Godot 游戏引擎中使用它来创建游戏和应用程序。

它是像 Dome Keeper 和 Brotato 这样的游戏所使用的语言。

特斯拉的工程师也用它来开发汽车仪表盘。

(在此处显示：ImageBrotato.tscn)

GDScript 是开始学习编程的绝佳语言，因为它是专门化的。与其他一些语言不同，它没有压倒性数量的功能需要你去学习。

大多数编程语言都是相似的 (Most programming languages are similar)​

不要害怕被锁定。你在第一门编程语言中学到的概念将适用于所有其他语言。

大多数语言的相似之处多于不同之处。一旦你学会了一种，学习下一种语言并变得高效所需的时间就会少得多。

这是一个在三种语言中实现相同代码的例子：GDScript、JavaScript 和 Python。

试着找出相似之处和不同之处。

(在此处显示：ThreeCodeExamples.tscn)

看起来并没有那么不同，是吗？

测验：编程语言完全不同吗？​

  • 不，它们有很多相似之处
  • 是的，它们完全不同

解释：大多数编程语言都建立在相同的编程思想之上。因此，它们大多是相似的。这并不是说所有语言都是一样的。与 GDScript 相比，有些语言提供了非常独特的语法，需要完全不同的思维方式。然而，像 GDScript、Python、JavaScript、C++、C# 和许多其他语言都建立在相似的编程哲学之上。

这是一门面向初学者的课程 (This is a course for beginners)​

如果你想学习制作游戏或编程，但不知道从哪里开始，这门课程应该非常适合你。

(在此处显示：WelcomeAnimation.tscn)

我们要为绝对初学者设计了这门课程，但如果你已经了解另一种语言，这也可能是开始使用 Godot 的有趣方式。

我们将为你提供使用 Godot 开始制作游戏和应用程序所需的编程基础。

请耐心等待。在你能够独自制作第一个完整的游戏之前，这需要时间。

学习制作游戏需要练习 (Learning to make games takes practice)​

制作游戏比以往任何时候都更容易，但它仍然需要大量的工作和练习。

不要指望任何单一的课程或书籍能把你变成专业人士。编程是通过练习学到的。

如果有些东西不能立即理解，不要太有压力！继续学习，以后再回过头来看。

享受这个过程，庆祝每一个小小的成功。作为一名游戏开发者，你永远不会停止学习。

你将学到什么以及如何学习 (What and how you'll learn)​

在这个免费课程中，你将学习开始编写如下内容所需的基础知识：

(在此处显示：ExampleDisplayInventory.tscn) (在此处显示：ExamplePathfinding.tscn)

在此过程中，我们将教你：

  • 作为开发者需要的一些思维方式。太多的编程课程跳过了这一重要部分。
  • 如何编写 GDScript 代码。
  • 让你入门的基本编程基础。

随着课程的进行，你会有很多问题。我们会尽力回答。

但因为要涵盖的内容太多，我们不得不走一些捷径。我们不想用信息淹没你。我们也想尊重我们大脑记忆事物的速度。

我们将课程分解为简短的课程和练习。如果我们在每个部分放入太多内容，你学起来会更慢。

如果在任何时候你有迫切的问题，非常欢迎加入我们的 Discord 社区。

这适用于 Godot 4 吗？ (Is this for Godot 4?)​

你在这个免费课程中学到的内容与 Godot 4 完全兼容。虽然我们最初是为 Godot 3 设计的这个系列，但 GDScript 的基础在 Godot 4 中仍然是一样的。

编程是一项技能 (Programming is a skill)​

编程是一项技能，所以要学好它，你必须练习。这就是我们要构建这个应用程序的原因。

每节课后都有一个互动练习，以使用你学到的东西。

说到这，是时候看看练习屏幕了！

要继续，请点击下面的练习按钮。它将简要介绍练习是如何工作的。

练习：尝试你的第一段代码 (Try Your First Code)​

目标：我们为你准备了一个代码示例。对于这个练习，你不需要更改任何东西。要测试代码，请点击代码编辑器下方的 运行 按钮。

描述：运行你的第一段代码并查看结果。
