安装插件

Godot 拥有一个编辑器插件系统，社区开发了许多插件。插件可以用新节点、额外的停靠面板、便利功能等扩展编辑器的功能。

寻找插件​

寻找 Godot 插件的首选方式是使用 资产库 (Asset Library)。虽然可以在线浏览，但直接在编辑器中使用更加方便。要这样做，请点击编辑器顶部的 AssetLib 选项卡：

你也可以在 GitHub 等代码托管网站上找到资产。

注意

有些仓库自称为"插件"，但可能实际上不是编辑器插件。对于旨在在运行中的项目里使用的脚本来说尤其如此。你不需要启用此类插件即可使用它们。只需下载它们并将文件解压到你的项目文件夹中即可。

区分编辑器插件和非编辑器插件的一种方法是在托管插件的仓库中查找 plugin.cfg 文件。如果仓库在 addons/ 文件夹内的某个文件夹中包含 plugin.cfg 文件，那么它就是一个编辑器插件。

安装插件​

要安装插件，请将其下载为 ZIP 归档文件。在资产库上，这可以通过 Download 按钮完成，无论是从编辑器内部还是使用 Web 界面。

在 GitHub 上，如果插件声明了 tags（版本），请转到 Releases 选项卡下载稳定版本。这确保你下载的是作者声明为稳定的版本。

在 GitHub 上，如果插件没有任何声明的标签，请使用 Download ZIP 按钮下载最新修订版的 ZIP：

解压 ZIP 归档文件，将其包含的 addons/ 文件夹移动到你的项目文件夹中。如果你的项目已经包含一个 addons/ 文件夹，请将插件的 addons/ 文件夹移动到你的项目文件夹中，以将新文件夹内容与现有内容合并。你的文件管理器可能会询问你是否写入文件夹；回答 Yes（是）。在此过程中不会覆盖任何文件。

启用插件​

要启用刚安装的插件，请打开编辑器顶部的 项目 (Project) > 项目设置 (Project Settings)，然后转到 插件 (Plugins) 选项卡。如果插件打包正确，你应该会在插件列表中看到它。点击 启用 (Enable) 复选框以启用插件。

你可以在启用插件后立即使用它；无需重启编辑器。同样，禁用插件也无需重启编辑器即可完成。
