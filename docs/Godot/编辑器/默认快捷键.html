默认快捷键

Godot 编辑器的许多功能都可以通过键盘快捷键执行。本页列出了默认关联了快捷键的功能，但你也可以在编辑器设置中自定义许多其他功能。要更改与这些及其他操作关联的按键，请导航至 编辑器 > 编辑器设置 > 快捷键 (Editor > Editor Settings > Shortcuts)。

虽然有些操作是通用的，但很多快捷键是特定于个别工具的。因此，某些键组合可能会被分配给多个功能。编辑器会根据当前的上下文（Context）执行正确的操作。

注意

虽然 Windows 和 Linux 版编辑器的默认设置大部分相同，但 macOS 版的某些快捷键可能有所不同。这是为了更好地将编辑器集成到 macOS 生态系统中。熟悉该操作系统标准快捷键的用户应该会发现 Godot 编辑器的默认键位映射很直观。
