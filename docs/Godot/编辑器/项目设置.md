你可以更改数十种设置来控制项目的执行，包括物理、渲染和窗口设置。这些设置可以通过 **项目设置** 窗口、代码或手动编辑 `project.godot` 文件来更改。你可以在 [ProjectSettings](https://docs.godotengine.org/en/stable/classes/class_projectsettings.html) 类文档中查看完整的设置列表。

在内部，Godot 将项目的设置存储在 `project.godot` 文件中，这是一个 INI 格式的纯文本文件。虽然它对人类可读且对版本控制友好，但手动编辑并不总是最方便的。为此，**项目设置** 窗口提供了可视化编辑这些设置的功能。要打开项目设置，请从主菜单选择 **项目 > 项目设置 (Project > Project Settings)**。

![项目设置窗口概览](https://docs.godotengine.org/en/stable/_images/project_settings_basic.webp)

**项目设置** 窗口主要用于更改 **常规 (General)** 选项卡中的设置。此外，还有用于 [输入映射 (Input Map)](https://docs.godotengine.org/en/stable/tutorials/inputs/input_examples.html)、[本地化 (Localization)](https://docs.godotengine.org/en/stable/tutorials/i18n/internationalizing_games.html)、[全局变量 (Globals)](https://docs.godotengine.org/en/stable/tutorials/scripting/singletons_autoload.html)（自动加载）、[插件 (Plugins)](https://docs.godotengine.org/en/stable/tutorials/plugins/editor/installing_plugins.html) 和 **导入默认值 (Import Defaults)** 的选项卡。这些选项卡的使用方法在其他地方有详细记录。

## 更改项目设置 (Changing project settings)

项目设置窗口的 **常规** 选项卡工作方式与检查器非常相似。它显示了一个你可以更改的项目设置列表，就像检查器属性一样。左侧有一个类别列表，你可以通过它选择相关的设置组。你还可以使用 **搜索设置 (Filter Settings)** 字段搜索特定设置。

每个设置都有一个默认值。点击每个属性旁边的圆形箭头 **重置 (Reset)** 按钮，可以将设置重置为其默认值。

### 通过代码更改项目设置 (Changing project settings from code)

你可以使用 `set_setting()` 通过代码更改设置的值：

```gdscript
ProjectSettings.set_setting("application/run/max_fps", 60)
ProjectSettings.set_setting("display/window/size/mode", DisplayServer.WINDOW_MODE_WINDOWED)
```

```csharp
ProjectSettings.SetSetting("application/run/max_fps", 60);
ProjectSettings.SetSetting("display/window/size/mode", (int)DisplayServer.WindowMode.Windowed);
```

然而，许多项目设置仅在游戏启动时读取一次。之后使用 `set_setting()` 更改设置将不会产生任何效果。相反，大多数设置在运行时类（如 [Engine](https://docs.godotengine.org/en/stable/classes/class_engine.html) 或 [DisplayServer](https://docs.godotengine.org/en/stable/classes/class_displayserver.html)）中有对应的属性或方法：

```gdscript
Engine.max_fps = 60
DisplayServer.window_set_mode(DisplayServer.WINDOW_MODE_WINDOWED)
```

```csharp
Engine.MaxFps = 60;
DisplayServer.WindowSetMode(DisplayServer.WindowMode.Windowed);
```

通常，项目设置在运行时会同步到 [Engine](https://docs.godotengine.org/en/stable/classes/class_engine.html)、[PhysicsServer2D](https://docs.godotengine.org/en/stable/classes/class_physicsserver2d.html)、[PhysicsServer3D](https://docs.godotengine.org/en/stable/classes/class_physicsserver3d.html)、[RenderingServer](https://docs.godotengine.org/en/stable/classes/class_renderingserver.html)、[Viewport](https://docs.godotengine.org/en/stable/classes/class_viewport.html) 或 [Window](https://docs.godotengine.org/en/stable/classes/class_window.html) 类中。在 [ProjectSettings](https://docs.godotengine.org/en/stable/classes/class_projectsettings.html) 类参考中，设置通常会链接到它们等效的运行时属性或方法。

## 读取项目设置 (Reading project settings)

你可以通过 `get_setting()` 或 `get_setting_with_override()` 读取项目设置：

```gdscript
var max_fps = ProjectSettings.get_setting("application/run/max_fps")
var window_mode = ProjectSettings.get_setting("display/window/size/mode")
```

```csharp
int maxFps = (int)ProjectSettings.GetSetting("application/run/max_fps");
var windowMode = (DisplayServer.WindowMode)(int)ProjectSettings.GetSetting("display/window/size/mode");
```

由于许多项目设置仅在启动时读取一次，项目设置中的值可能不再准确。在这些情况下，最好从运行时等效的属性或方法中读取值：

```gdscript
var max_fps = Engine.max_fps
var window_mode = DisplayServer.window_get_mode()
```

```csharp
int maxFps = Engine.MaxFps;
DisplayServer.WindowMode windowMode = DisplayServer.WindowGetMode();
```

## 手动编辑 project.godot (Manually editing project.godot)

你可以使用文本编辑器打开 `project.godot` 文件并手动更改项目设置。请注意，如果 `project.godot` 文件中没有存储特定设置的值，则它隐式使用该设置的默认值。这意味着如果你手动编辑文件，你可能必须同时写入设置名称 *和* 值。

通常建议使用项目设置窗口，而不是手动编辑 `project.godot`。

## 高级项目设置 (Advanced project settings)

![高级项目设置开关](https://docs.godotengine.org/en/stable/_images/project_settings_advanced.webp)

默认情况下，仅显示部分项目设置。要查看所有项目设置，请启用 **高级设置 (Advanced Settings)** 开关。
