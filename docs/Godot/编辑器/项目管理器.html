项目管理器

当你启动 Godot 时，看到的第一个窗口就是项目管理器。通过它，你可以创建、移除、导入或运行游戏项目：

要更改编辑器的语言，请点击右上角的 设置 (Settings) 按钮：

在项目管理器设置中，你可以通过语言下拉菜单更改界面的 语言（默认使用系统语言）。

你还可以更改编辑器的 主题 (Theme) 和 配色预设 (Color Preset)、针对不同界面元素大小的 显示缩放 (Display Scale)，以及通过 网络模式 (Network Mode) 控制在线功能的可用性。如果网络模式设为“在线 (Online)”，Godot 还会检查并通知你有关 Godot 的新版本。

目录命名规范 (Directory Naming Convention) 也可以进行更改，以便在自动创建文件夹时，按照所选格式替换空格。

创建和导入项目 (Creating and importing projects)​

要创建一个新项目：

  1. 点击窗口左上角的 新建 (Create) 按钮。
  2. 给项目起一个名字，然后使用 浏览 (Browse) 按钮打开文件浏览器，在你的电脑上选择一个空文件夹来保存文件。或者，你可以启用 创建文件夹 (Create Folder) 选项，根据设置中的目录命名规范自动创建一个以项目名称命名的子文件夹。空文件夹右侧会显示一个绿色对勾。
  3. 选择一个渲染器（稍后也可以更改）。
  4. 点击“创建并编辑 (Create & Edit)”按钮来创建项目文件夹并在编辑器中打开它。

注意

你可以选择是否使用版本控制系统。目前仅支持 git，且需要安装 Godot Git 插件（可以手动安装，也可以使用 资产库）。要了解有关 Godot Git 插件的更多信息，请参阅其 维基 (wiki)。

使用文件浏览器 (Using the file browser)​

在“新建项目”窗口中，点击 浏览 (Browse) 按钮打开 Godot 的文件浏览器。选择驱动器后，你可以选取一个位置，或者在 路径 (Path) 字段中手动输入文件夹路径。

顶部路径字段左侧的箭头用于在上次访问的位置之间后退或前进。上箭头用于导航到父文件夹。路径字段右侧的按钮分别用于：刷新当前文件夹内容、收藏/取消收藏当前文件夹、以及显示/隐藏隐藏文件夹。

接下来的按钮用于在网格视图和列表视图之间切换文件夹和文件的显示类型。

右侧最后一个按钮用于新建文件夹。

已收藏的文件夹会显示在左侧的 收藏 (Favorites) 区域下。你可以使用该区域中的上下箭头对收藏夹进行排序。最后选取的文件夹会列在 最近 (Recent) 列表中。

打开和导入项目 (Opening and importing projects)​

下次打开项目管理器时，你会看到你的新项目出现在列表中。双击它即可在编辑器中打开。

你可以类似地使用 导入 (Import) 按钮导入现有项目。定位到包含项目的文件夹或 project.godot 文件以将其导入并编辑。

或者，也可以选择一个 zip 文件，由 Godot 自动解压。

当文件夹路径正确时，你会看到一个绿色的勾。

下载演示和模板 (Downloading demos and templates)​

在 资产库 (Asset Library) 选项卡中，你可以从 资产库 下载开源项目模板和演示，以帮助你更快上手。

第一次打开此选项卡时，你会发现它要求你上线。出于隐私原因，项目管理器和 Godot 编辑器默认无法访问互联网。要启用互联网访问，请点击 上线 (Go Online) 按钮。这也将允许项目管理器向你通知更新。如果你以后想关闭此功能，请进入项目管理器设置并将 网络模式 (Network Mode) 更改为“离线 (Offline)”。

现在 Godot 已连接到互联网，你可以下载演示或模板了，步骤如下：

  1. 点击其标题。
  2. 在打开的页面中，点击下载按钮。
  3. 下载完成后，点击安装并选择你想保存项目的位置。

使用标签管理项目 (Managing projects with tags)​

对于在一台 PC 上拥有大量项目的用户来说，跟踪所有项目可能会很麻烦。为了解决这个问题，Godot 允许你创建项目标签。要给项目添加标签，请在项目管理器中点击该项目，然后点击 管理标签 (Manage Tags) 按钮。

这将打开管理项目标签窗口。点击加号按钮添加标签。

输入标签名称并点击 确定 (OK)。你的项目现在就有了一个标签。这些标签也可以用于项目管理器中的任何其他项目。

要仅显示带有特定标签的项目，你可以点击标签，或者在过滤器栏中输入 tag: 后面跟着你想搜索的标签。要使用多个标签限制结果，你可以点击另一个标签，或者在空格后添加 tag: 并输入另一个标签。

此外，标签会随项目一起保存。因此，如果你给项目贴上标签并发送到另一台机器，然后在该机器的项目管理器中导入，你仍然会看到你创建的标签。

要从项目管理器中彻底删除一个标签，必须将其从所有使用该标签的项目中移除。完成后，关闭项目管理器再重新打开，该标签就会消失。

恢复模式 (Recovery Mode)​

如果一个项目在启动时立即崩溃，或在编辑过程中频繁崩溃，可以使用恢复模式打开，尝试在寻找并修复崩溃源头时使其更稳定。

通常，在崩溃后重新打开项目时，它应该会自动以恢复模式打开。如果没有，你可以通过在项目管理器中选择项目来手动开启恢复模式：从项目列表中选择项目，点击编辑（Edit）节点旁边的下拉按钮，然后选择 在恢复模式下编辑 (Edit in recovery mode)。

在恢复模式下，以下功能将被禁用：

  • 工具脚本 (Tool scripts)
  • 编辑器插件 (Editor plugins)
  • GDExtension 插件
  • 自动场景恢复
  • 运行项目

建议在恢复模式下编辑项目之前先备份你的项目。
