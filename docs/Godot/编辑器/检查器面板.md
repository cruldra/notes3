检查器（Inspector）面板列出了对象、资源或节点的所有属性。当你从场景树面板中选择不同的节点，或者从文件系统的上下文菜单中使用 **打开 (Open)** 命令时，它会实时更新属性列表。

![检查器面板概览](https://docs.godotengine.org/en/stable/_images/inspector_overview.webp)

本页将深入讲解检查器面板的工作原理。你将学习如何编辑属性、展开和折叠区域、使用搜索栏等。

## 使用方法 (Usage)

如果检查器面板可见，点击场景树中的节点将自动显示其属性。如果不可见，你可以通过 **编辑器 > 编辑器面板 > 检查器 (Editor > Editor Docks > Inspector)** 来显示它。

面板顶部是文件操作和导航按钮。

![检查器顶部按钮](https://docs.godotengine.org/en/stable/_images/inspector_top_buttons.webp)

从左到右依次为：

*   **创建资源**：打开一个新窗口，用于在内存中选择并创建资源，然后进行编辑。
*   **打开资源**：从文件系统中打开一个资源进行编辑。
*   **保存资源**：将当前编辑的资源保存到磁盘。
*   **额外选项**：提供以下选项：
    *   **从剪贴板编辑资源**：粘贴已复制的资源。
    *   **复制资源**：将资源复制到剪贴板。
    *   **在文件系统中显示**：如果资源已保存，在文件系统中定位它。
    *   **使资源内置 (Make Resource Built-In)**：将其转换为内置资源，而非磁盘上的独立文件。
*   **前进/后退箭头**：在编辑的对象历史记录中导航。
*   **历史记录列表**：点击旁边的小按钮打开历史列表以快速跳转。如果你在内存中创建了多个资源，也会在这里看到它们。

在这些按钮下方，你可以看到所选节点的图标、名称，以及右侧用于打开其文档的快速按钮。点击节点名称本身会列出该节点的子资源（如果有的话）。

接下来是 **搜索栏**。在其中输入任何内容即可过滤显示的属性。删除文本可清除搜索。该搜索不区分大小写，并且会随着你输入字符实时检索。例如，输入 `vsb`，你会看到 “Visibility” 属性出现在结果中，因为该单词包含这些字母。

在讨论过滤器栏旁边的工具按钮之前，值得先说明一下下方显示的内容及其结构。

![检查器面板结构说明](https://docs.godotengine.org/en/stable/_images/inspector_dock_overlay.webp)

属性按其所属的 **类 (classes)** 分组为不同的 **部分 (sections)**。你可以展开每个部分来查看相关的属性。

你还可以通过右键点击某个类并选择 **打开文档 (Open Documentation)** 来打开该类的文档。类似地，你可以右键点击某个属性来：复制或粘贴其值、复制属性路径、将其设为“收藏”以显示在检查器顶部，或者打开其文档页面。

如果你将鼠标悬停在某个属性上，会看到该属性的功能描述以及如何在脚本中调用它。

你可以直接通过点击、输入或从菜单中选择来更改值。如果属性是数字或滑块，你可以按住鼠标左键并拖动来快速更改值。

![检查器中的子资源编辑](https://docs.godotengine.org/en/stable/_images/inspector_dock_subresource.webp)

如果节点的属性是一个 **子资源 (sub-resource)**，你可以点击下拉箭头选取资源类型，或者使用 **快速加载 (Quick Load)** 或 **加载 (Load)** 选项。或者，也可以直接从文件系统拖入支持的资源。当你开始拖动时，兼容的属性会高亮显示。只需将其释放到相应属性的值上即可。

加载子资源后，你可以点击它来查看或调整其属性。

值被修改后（与原始值不同），旁边会出现一个 **重置图标** (![undo](https://docs.godotengine.org/en/stable/_images/inspector_dock_revert.webp))。点击此图标可将值恢复到原始状态。如果多个值相互关联，它们会显示一个链条图标，更改其中一个会同步更改其他值。你可以点击该图标取消关联。

如果你频繁更改某个属性，可以右键点击并选择 **收藏属性 (Favorite Property)**。这会将其置顶于该类所有对象的检查器顶部。

了解了这些术语后，我们来看看 **工具菜单 (tool menu)**。点击过滤器栏旁边的工具菜单图标，下拉菜单会提供各种查看和编辑选项：

![检查器工具菜单](https://docs.godotengine.org/en/stable/_images/inspector_tools_menu.webp)

*   **全部展开 (Expand All)**：展开所有部分，显示所有可用属性。
*   **全部折叠 (Collapse All)**：折叠所有属性，仅显示类名和部分名称。
*   **展开非默认值 (Expand Non-Default)**：仅展开当前值与原始值不同的部分（即带有重置图标的部分）。
*   **属性名称样式 (Property Name Style)**：决定属性文本的显示方式。`Raw` 使用属性的原生命名（通常是下划线分隔）；`Capitalized` 使用首字母大写样式，移除下划线；`Localized` 则根据你编辑器的语言显示翻译后的名称。
*   **复制属性 (Copy Properties)**：复制当前节点的所有属性及其当前值。
*   **粘贴属性 (Paste Properties)**：从剪贴板粘贴已复制的属性。常用于将一个节点的通用属性应用到另一个节点。
*   **使子资源唯一 (Make Sub-Resources Unique)**：默认情况下，复制的节点会与原节点共享子资源。更改其中一个节点的子资源参数会影响另一个。启用此选项可使该节点使用的每个子资源都变得唯一，与其他节点分离。

> **提示**
>
> 如果节点在其附加脚本中定义了导出变量（exported variables），你也会在检查器中看到它们。在本节的第一张图片中，Player 节点就有这样一个变量：Action Suffix。
