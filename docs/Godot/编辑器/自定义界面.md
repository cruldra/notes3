默认情况下，Godot 的界面位于单个窗口中。自 Godot 4.0 起，你可以将多个元素拆分为独立窗口，以更好地利用多显示器设置。

## 移动和调整面板大小 (Moving and resizing docks)

点击并拖动任何面板（dock）或窗口边缘，可以水平或垂直调整其大小：

![在编辑器中调整面板大小](https://docs.godotengine.org/en/stable/_images/editor_ui_resize_dock.webp)

点击任何面板顶部的“3 个垂直点”图标可以更改其位置，或者在弹出的子菜单中选择 **设为浮动 (Make Floating)** 将其拆分为独立窗口：

![在编辑器中移动面板](https://docs.godotengine.org/en/stable/_images/editor_ui_move_dock.webp)

要将浮动面板移回编辑器窗口，只需点击窗口右上角的 **×** 按钮（macOS 为左上角）关闭该面板窗口即可。此外，在拆分窗口获得焦点时按 Alt + F4 也可以实现。

## 将脚本或着色器编辑器拆分为独立窗口 (Splitting the script or shader editor to its own window)

> **注意**
>
> 此功能仅在支持生成多窗口的平台（Windows、macOS 和 Linux）上可用。
>
> 如果在编辑器设置中启用了 **单窗口模式 (Single Window Mode)**，则该功能不可用。

自 Godot 4.1 起，你可以将脚本编辑器或着色器（shader）编辑器拆分为独立窗口。

要将脚本编辑器拆分为独立窗口，请点击脚本编辑器右上角的相应按钮：

![将脚本编辑器拆分为独立窗口](https://docs.godotengine.org/en/stable/_images/editor_ui_split_script_editor.webp)

要将着色器编辑器拆分为独立窗口，请点击着色器编辑器右上角的相应按钮：

![将着色器编辑器拆分为独立窗口](https://docs.godotengine.org/en/stable/_images/editor_ui_split_shader_editor.webp)

要返回之前的状态（即将编辑器嵌入主窗口），请关闭拆分出来的窗口。

## 自定义编辑器布局 (Customizing editor layouts)

你可能希望根据正在进行的任务类型来保存和加载不同的面板配置。例如，在制作角色动画时，与在设计关卡时相比，采用不同的面板布局可能会更方便。

为此，Godot 提供了一种保存和恢复编辑器布局的方法。在保存布局之前，请先按照你的喜好调整面板。以下更改会被保留在保存的布局中：

*   移动面板。
*   调整面板大小。
*   使面板浮动。
*   更改浮动面板的位置或大小。
*   文件系统（FileSystem）面板的属性：拆分模式、显示模式、排序顺序、文件列表显示模式、选定路径和展开路径。

> **注意**
>
> 将脚本或着色器编辑器拆分为独立窗口的状态 **不会** 保留在布局中。

调整完毕后，打开编辑器顶部的 **编辑器 (Editor)** 菜单，选择 **编辑器布局 > 保存 (Editor Layouts > Save)**。输入布局名称，然后点击 **保存**。如果你之前已经保存过布局，可以从列表中选择并覆盖。

在 **编辑器 > 编辑器布局** 菜单中，你可以看到已保存的布局列表，以及 **Default (默认)** 布局（这是一个硬编码的布局，无法删除）。默认布局对应于刚安装 Godot 时的初始状态。

你可以通过下拉菜单中的 **删除 (Delete)** 选项来移除某个布局。

> **提示**
>
> 如果你将保存的布局命名为 `Default`（区分大小写），则会覆盖默认的编辑器布局。

编辑器布局保存在 [编辑器数据路径](https://docs.godotengine.org/en/stable/tutorials/io/data_paths.html) 下一个名为 `editor_layouts.cfg` 的文件中。

## 自定义编辑器设置 (Customizing editor settings)

在编辑器顶部的 **编辑器 (Editor)** 菜单中，你可以找到 **编辑器设置 (Editor Settings)** 选项。这会打开一个类似于项目设置的窗口，但其中包含的是由编辑器使用的设置。这些设置在所有项目中共享，不会保存到项目文件中。

![编辑器设置窗口](https://docs.godotengine.org/en/stable/_images/editor_settings.webp)

一些常用的设置包括：

*   **Interface > Editor > Editor Language**: 控制编辑器的显示语言。为了方便跟随英文教程，你可能想将其设为英语。
*   **Interface > Editor > Display Scale**: 控制 UI 元素的大小。**Auto** 设置会根据 DPI 自动寻找合适值。
*   **Interface > Editor > Single Window Mode**: 启用后，强制编辑器仅使用单个窗口。
*   **Interface > Theme > Preset**: 编辑器主题预设。例如 **Light**（明亮）或 **Black (OLED)**（黑色）。
*   **FileSystem > Directories > Autoscan Project Path**: 设置一个文件夹路径，项目管理器每次启动时都会自动扫描该路径下的项目。
*   **Editors > 3D > Emulate Numpad**: 允许使用键盘上排数字键模拟小键盘数字键（推荐给没有小键盘的用户）。
*   **Editors > 3D > Emulate 3 Button Mouse**: 允许在不按鼠标键的情况下使用视图平移、缩放和旋转修饰键（推荐给触控板用户）。

更多详细描述请参阅 [EditorSettings](https://docs.godotengine.org/en/stable/classes/class_editorsettings.html) 类参考文档。
