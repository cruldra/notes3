学习新特性

Godot 是一个功能丰富的游戏引擎，有很多东西需要学习。本页将介绍如何使用在线手册、内置代码参考，并加入在线社区来学习新特性和技术。

充分利用本手册 (Making the most of this manual)​

你现在阅读的正是用户手册。它记录了引擎的每个概念和可用特性。在学习新主题时，你可以从浏览本网站的相应部分开始。左侧菜单允许你探索广泛的主题，而搜索栏将帮助你找到更具体的页面。如果某个主题存在对应页面，它通常会链接到更多相关内容。

手册还有一个配套的类参考，解释了编程时每个 Godot 类可用的函数和属性。手册涵盖了通用特性、概念以及如何使用编辑器，而参考则是关于使用 Godot 的脚本 API（应用程序编程接口）。你可以通过在线和离线两种方式访问它。我们建议从 Godot 编辑器内部离线浏览参考。为此，请转到 帮助 -> 搜索帮助 或按 F1。

要在线浏览，请前往手册的 类参考 部分。

类参考页面会告诉你：

  1. 该类在继承层次结构中的位置。你可以点击顶部的链接跳转到父类，查看一个类型继承的属性和方法。
  2. 该类的作用和使用场景摘要。
  3. 该类的属性、方法、信号、枚举和常量的解释。
  4. 指向进一步详细说明该类的手册页面的链接。

注意

如果手册或类参考缺失信息或信息不足，请在官方 godot-docs GitHub 仓库中开启一个 Issue 来报告。

你可以按住 Ctrl（macOS 为 Cmd），然后将鼠标悬停在类名、属性、方法、信号或常量等文本上以显示下划线，接着使用 Ctrl + 点击（macOS 为 Cmd + 点击）跳转到对应内容。

学习像程序员一样思考 (Learning to think like a programmer)​

教授编程基础和如何像游戏开发者一样思考超出了 Godot 文档的范畴。如果你是编程新手，我们推荐两个优秀的免费资源供你开始：

  1. GDQuest 的 Learn GDScript From Zero 是一个免费开源的互动教程，适合绝对初学者学习使用 Godot 的 GDScript 语言进行编程。它可以作为 桌面应用程序 或 在浏览器中 使用。
  2. 如果你更喜欢书本，请查看 Al Sweigart 编写的免费电子书 Python 编程快速上手：让繁琐工作自动化 (Automate The Boring Stuff With Python)。

与社区共同学习 (Learning with the community)​

Godot 拥有一个不断壮大的用户社区。如果你在某个问题上卡住了，或者需要帮助以更好地理解如何实现某些目标，你可以在众多 活跃社区 之一寻求其他用户的帮助。

提问和寻找已有答案的最佳场所是官方 Godot 论坛。这些回复会出现在搜索引擎结果中并被保存下来，让其他用户也能从平台上的讨论中受益。在论坛提问后，你可以在其他社交平台上分享其链接。在提问之前，请务必在本网站或使用你喜欢的搜索引擎查找可能解决你问题的现有答案。

出色地提出问题并提供细节将帮助他人更快、更好地回答你。提问时，我们建议包含以下信息：

  1. 描述你的目标。你想解释你在设计上试图实现什么。如果你在弄清楚如何让某个解决方案奏效时遇到困难，可能存在另一个更简单的解决方案能实现相同的目标。
  2. 如果涉及错误，分享确切的错误信息。你可以点击编辑器底部 “调试器” 面板中的 “复制错误” 图标来复制确切的错误信息。了解报错内容可以帮助社区成员更好地识别你是如何触发该错误的。
  3. 如果涉及代码，分享代码样本。如果不看你的代码，其他用户将无法帮助你解决问题。直接以文本形式分享代码。你可以将简短的代码片段复制粘贴到聊天框中，或者使用类似 Pastebin 的网站来分享长文件。
  4. 分享一份你 场景 (Scene) 面板的截图 以及你编写的代码。你编写的大部分代码都会影响场景中的节点。因此，你应该将这些场景视为源代码的一部分。 另外，请不要用手机拍照，低质量和屏幕反光会让人难以看清图像。你的操作系统应该内置了使用 PrtSc (Print Screen) 键进行截图的工具（macOS：使用 Cmd + Shift + 3 进行全屏截图，更多信息请点击这里）。 或者，你可以在 Windows 上使用 ShareX，或者在 Windows/macOS/Linux 上使用 Flameshot。
  5. 分享一段游戏运行时的视频对于 排查游戏故障 也非常有用。你可以使用 OBS Studio 和 Screen to GIF 等程序来录制屏幕。 然后你可以使用类似 streamable 的服务或云盘来免费上传并分享你的视频。
  6. 如果你使用的不是 Godot 的稳定版本，请提及你使用的版本。随着可用特性和界面的快速演进，答案可能会有所不同。

遵循这些指南将最大限度地提高你获得所需答案的机会。这不仅能为你节省时间，也能为帮助你的人节省时间。

社区教程 (Community tutorials)​

本手册旨在提供 Godot 特性的全面参考。除 2D 和 3D 入门系列外，它不包含实现特定游戏类型的教程。如果你正在寻找关于创建角色扮演游戏、平台游戏或其他类型的教程，请参阅 教程与资源，其中列出了由 Godot 社区制作的内容。
