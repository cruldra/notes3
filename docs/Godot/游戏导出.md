## 1. 核心概念

在 Godot 中，我们不叫“编译 (Compile)”，而是叫 **“导出 (Export)”**。

* **原理**：Godot 引擎核心（Runner）+ 你的资源包（.pck） = 可运行的游戏。
* **目标**：将开发环境的项目转换为 Windows (.exe)、Android (.apk) 或 Web (HTML5) 等格式。

---

## 2. 准备工作：导出模板 (必做)

Godot 编辑器本身不包含导出功能，必须先安装对应的**导出模板 (Export Templates)**。

### A. 如何安装

1. **菜单路径**：`编辑器 (Editor)` -> `管理导出模板 (Manage Export Templates)`。
2. **版本匹配**：模板版本必须与编辑器版本**严格一致**（如 4.2.1 必须配 4.2.1 的模板）。

### B. 下载太慢/失败怎么办？(国内网络常见问题)

不要在编辑器里死等，使用 **“手动安装法”**：

1. **去官网库下载**：访问 [Godot 下载仓库](https://www.google.com/search?q=https://downloads.tuxfamily.org/godotengine/)。
2. **找文件**：进入对应版本文件夹，下载后缀为 `.tpz` 的文件（例如 `Godot_v4.x.x_export_templates.tpz`）。
3. **导入**：回到编辑器导出模板页面，点击 **从文件安装 (Install from File)**，选中该 `.tpz` 文件即可秒装。

---

## 3. 标准导出流程 (Windows 为例)

1. **打开面板**：点击顶部菜单 `项目 (Project)` -> `导出 (Export)`。
2. **添加预设**：点击顶部 `添加... (Add...)`，选择 `Windows Desktop`。
3. **关键配置 (由上至下)**：
* **Name**: 预设名称（随意）。
* **Options (选项页)**:
* `Embed PCK` (Godot 4 推荐): **勾选**。这样会生成单文件的 `.exe`，方便发送。如果不勾选，会生成 `.exe` + `.pck` 两个文件，必须放在一起才能跑。
* `Texture Format`: 默认 S3TC 即可。




4. **执行导出**：
* 点击底部的 **导出项目 (Export Project)**。
* **调试模式 (Debug)**：保存时的弹窗里有一个 `Export With Debug` 复选框。
* ✅ **勾选**：测试版。游戏运行时会附带一个黑色的 CMD 控制台窗口，用于查看 `print()` 报错信息。
* ❌ **不勾选**：正式版。发布给玩家时请取消勾选，这样就没有黑框了。

## 4. 常见平台特殊注意事项

### 🖥️ Windows 平台

* **修改图标 (Icon)**：
* 直接在导出选项里换 Icon 图片是不够的，导出的 `.exe` 可能还是 Godot 的默认图标。
* **解决方法**：你需要下载一个小工具叫 `rcedit`。
* **配置**：去 `编辑器设置` -> `Export` -> `Windows`，填入 `rcedit.exe` 的路径。再次导出即可生效。



### 📱 Android 平台

* **环境地狱**：这是最难配置的平台，需要 **Java JDK** 和 **Android SDK**。
* **设置路径**：在 `编辑器设置` -> `Export` -> `Android` 中填入 SDK 路径。如果路径留空，导出按钮会是灰色的。
* **签名密钥**：必须生成一个 `.keystore` 文件（调试用 debug.keystore 即可）并在导出预设中填入，否则无法生成 APK。

### 🌐 Web (HTML5) 平台

* **用途**：发布到 Itch.io 或个人网站。
* **注意**：Godot 4 的 Web 导出依赖 `SharedArrayBuffer`，这要求服务器配置特定的响应头（COOP/COEP）。如果你上传到某些旧的服务器，可能会报错无法加载。

## 5. 常见问题 (FAQ) & 避坑

| 问题现象 | 原因分析 | 解决方案 |
| --- | --- | --- |
| **打开游戏有个黑框** | 开启了 Debug 模式 | 导出保存时，**取消勾选** "Export With Debug"。 |
| **画面模糊/马赛克** | 窗口拉伸模式没设对 | `项目设置` -> `Window` -> `Stretch` -> `Mode` 改为 `canvas_items` (高清) 或 `viewport` (像素风)。 |
| **导出后找不到资源** | 动态加载路径错误 | 检查代码里的 `load()` 或 `preload()`。导出后文件系统区分大小写！确保 `res://Icon.png` 和文件名大小写完全一致。 |
| **没有 .exe 图标** | 缺少 rcedit | 下载 rcedit 并配置到编辑器设置中。 |
| **文件体积太大** | 包含了未用资源 | 检查导出选项里的 `Resources` 标签，选择 `Export selected scenes` (仅导出选中场景) 可以剔除无关文件，但新手慎用，容易漏文件。 |

---

## 6. 一句话总结

**下载 tpz 模板 -> 安装 -> 项目导出 -> 勾选 Embed PCK -> 取消 Debug -> 快乐分享！**
