简介

Godot 的 2D 游戏开发工具包括专门的 2D 渲染引擎、物理系统以及专为创建 2D 体验而定制的功能。你可以使用 TileMap 系统高效地设计关卡，使用 2D 精灵或剪纸动画制作角色动画，并利用 2D 灯光实现动态场景照明。内置的 2D 粒子系统允许你创建复杂的视觉效果，Godot 还支持自定义着色器（Shaders）以增强图形效果。这些功能结合 Godot 的易用性和灵活性，为创建引人入胜的 2D 游戏奠定了坚实的基础。

本页将向你展示 2D 工作区并帮助你熟悉它。

2D 工作区 (2D workspace)​

你将使用 2D 工作区来处理 2D 场景、设计关卡或创建用户界面。要切换到 2D 工作区，你可以从场景树中选择一个 2D 节点，或者使用编辑器顶边的工作区选择器：

你可以通过文件系统面板拖放兼容的资源到视口中以添加节点。拖放时，节点默认添加为当前选中节点的兄弟节点（如果是根节点则添加为子节点）。按住 Shift 拖放会添加为选中节点的子节点；按住 Alt 拖放会添加为根节点的子节点。

主工具栏 (Main toolbar)​

主工具栏包含了一系列用于操作 2D 对象的工具。将鼠标悬停在按钮上一秒钟即可查看简短说明和快捷键。

  • 选择模式 (Select Mode) (Q)：在视口中选择节点。可以点击单个节点，或点击并拖动框选多个。
  • 移动模式 (Move Mode) (W)：移动选中的节点。
  • 旋转模式 (Rotate Mode) (E)：旋转选中的节点。
  • 缩放模式 (Scale Mode) (S)：缩放选中的节点。
  • 旋转中心点 (Rotation pivot)：设置节点的旋转中心。默认情况下，节点的中心点位于其本地坐标的 (0, 0)。
  • 平移模式 (Pan Mode) (G)：平移视图而不选中节点。也可以按住 空格键 拖动。
  • 标尺模式 (Ruler Mode) (R)：点击并拖动以测量像素距离。开启吸附时，它还会显示网格数。
  • 智能吸附 (Smart Snap) 和 网格吸附 (Grid Snap)：控制节点是否吸附到网格或其他节点。

可以通过三点菜单进一步自定义吸附选项，包括 像素吸附 (Pixel Snap)、旋转吸附 和 缩放吸附。

坐标系 (Coordinate system)​

在 2D 编辑器中只有两个轴：x 和 y。

  • 红色 线代表 X 轴。
  • 绿色 线代表 Y 轴。 在 Godot 中，向右和向下是正方向。两条线的交点是原点 (0, 0)。

2D 视口 (2D Viewport)​

视口是你进行视觉设计的主要区域：

  • 导航：中键拖动或使用滚动条平移。
  • 缩放：左上角的 - 和 + 按钮，或使用滚轮。
  • 辅助线 (Guides)：从水平或垂直标尺（边缘的黑色长条）向视口拖动可以创建紫色的辅助线。

视图 (View) 菜单： 提供了一系列选项来更改视口的外观，例如开关网格、辅助器、原点线、视口矩形（指示默认游戏窗口大小）和辅助线。

Node2D 和 Control 节点​

  • CanvasItem 是所有 2D 节点的基类。
  • Node2D 是所有 2D 游戏对象的基类。
  • Control 是所有 GUI（用户界面）元素的基类。

2D 中的 3D (3D in 2D)​

可以在 2D 屏幕中显示 3D 场景，这通常通过 SubViewport 节点实现。
