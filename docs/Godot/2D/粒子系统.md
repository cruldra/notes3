## 简介 (Intro)

粒子系统用于模拟复杂的物理效果，如火花、火焰、魔法粒子、烟雾、薄雾等。

其基本理念是：以固定的时间间隔发射具有固定寿命的“粒子”。在寿命期间，每个粒子都遵循相同的基本行为。使每个粒子看起来不同且更具生动感的是与每个参数相关联的“随机性”。本质上，创建粒子系统意味着设置基础物理参数，然后为其添加随机性。

### 粒子节点 (Particle nodes)

Godot 为 2D 粒子提供了两种不同的节点：[GPUParticles2D](https://docs.godotengine.org/en/stable/classes/class_gpuparticles2d.html) 和 [CPUParticles2D](https://docs.godotengine.org/en/stable/classes/class_cpuparticles2d.html)。

*   **GPUParticles2D**：更高级，使用 GPU 处理粒子效果。
*   **CPUParticles2D**：由 CPU 驱动，功能与 GPUParticles2D 几乎一致，但在粒子数量较多时性能较低。不过，在低端系统或 GPU 瓶颈的情况下，其表现可能更好。

`GPUParticles2D` 是通过 [ParticleProcessMaterial](https://docs.godotengine.org/en/stable/classes/class_particleprocessmaterial.html)（以及可选的自定义着色器）进行配置的，而 `CPUParticles2D` 的对应选项则是直接作为节点属性提供的。

目前没有计划为 `CPUParticles2D` 添加新功能，因此除非有明确理由，否则我们建议优先使用 `GPUParticles2D`。

你可以在工具栏中通过 **CPUParticles2D > Convert to GPUParticles2D** 将 CPU 粒子转换为 GPU 粒子：

![转换粒子节点](https://docs.godotengine.org/en/stable/_images/particles_convert.webp)

本教程接下来的内容将以 `GPUParticles2D` 节点为例。首先向场景添加一个该节点，你会发现它最初只是一个白点，且旁边有警告图标。这是因为该节点需要一个 `ParticleProcessMaterial` 才能运行。

### 粒子处理材质 (ParticleProcessMaterial)

要添加处理材质，在检查器面板中找到 `Process Material`。点击 `Material` 旁边的框，并从下拉菜单中选择 `新建 ParticleProcessMaterial`。

![添加材质](https://docs.godotengine.org/en/stable/_images/particles_material.webp)

现在你的节点应该开始向下发射白色的点了：

![基础粒子发射效果](https://docs.godotengine.org/en/stable/_images/particles1.png)

### 纹理 (Texture)

粒子系统可以使用单张纹理，也可以使用 **动画翻页书 (Animation Flipbook)**。翻页书是一张包含多帧动画的纹理，可以在发射过程中播放或随机选择帧，相当于粒子的精灵表（Spritesheet）。

通过 **Texture** 属性设置纹理：

![设置纹理属性](https://docs.godotengine.org/en/stable/_images/particles2.webp)

#### 使用动画翻页书 (Using an animation flipbook)
翻页书适用于烟雾、火焰、爆炸等复杂效果。

![使用翻页书纹理的粒子效果示例](https://docs.godotengine.org/en/stable/_images/particles_flipbook_result.webp)

使用动画翻页书需要比单张纹理更多的配置。为了演示，我们将使用这张具有 5 列 7 行的纹理：

![翻页书纹理示例](https://docs.godotengine.org/en/stable/_images/particles_flipbook_example.webp)

要使用动画翻页书，你必须在 `GPUParticles2D`（或 `CPUParticles2D`）节点的 **Material** 部分创建一个新的 `CanvasItemMaterial`：

![创建 CanvasItemMaterial](https://docs.godotengine.org/en/stable/_images/particles_flipbook_create_canvasitemmaterial.webp)

在此 `CanvasItemMaterial` 中，启用 **Particles Animation** 并将 **H Frames** 和 **V Frames** 设置为你的翻页书纹理的行列数：

![配置 CanvasItemMaterial](https://docs.godotengine.org/en/stable/_images/particles_flipbook_configure_canvasitemmaterial.webp)

设置完成后，`ParticleProcessMaterial`（或 CPUParticles2D 检查器）中的 [Animation 部分](https://docs.godotengine.org/en/stable/tutorials/2d/particle_process_material_2d.html#doc-particle-systems-2d-animation) 设置即可生效。

> **提示**：如果你的翻页书纹理背景是黑色的而非透明的，需要将材质的混合模式（Blend Mode）设为 **Add**。

## 时间参数 (Time parameters)

### 寿命 (Lifetime)
每个粒子存活的时间（秒）。当寿命结束，系统会创建一个新粒子来替换它。

**Lifetime: 0.5**
![寿命 0.5 示例](https://docs.godotengine.org/en/stable/_images/paranim14.gif)

**Lifetime: 4.0**
![寿命 4.0 示例](https://docs.godotengine.org/en/stable/_images/paranim15.gif)

### 单次发射 (One Shot)
启用后，节点将发射完所有粒子一次后停止，不再循环。

### 预处理 (Preprocess)
粒子系统最初从零粒子开始发射。预处理允许系统在第一次实际绘制之前先运行若干秒，这对于场景加载时就需要“已经燃起来”的火把或弥漫的雾气非常有用。

### 速度缩放 (Speed Scale)
调整整个粒子系统的运行快慢。增加此值会让粒子运行得更快。

### 爆炸性 (Explosiveness)
*   **0**：按常规间隔发射粒子（默认）。
*   **1**：所有粒子同时喷发。
用于创建爆炸或瞬间喷溅效果：
![爆炸性示例](https://docs.godotengine.org/en/stable/_images/paranim18.gif)

### 随机性 (Randomness)
所有的物理参数都可以随机化，范围为 `0` 到 `1`。

### 固定 FPS (Fixed FPS)
让粒子系统以固定的帧率进行渲染。例如设为 `2` 会让粒子以每秒 2 帧的速率更新。注意这不会减慢物理模拟本身。

### 碎分数增量 (Fract Delta)
启用此项可以获得更精确电计算和更平滑的粒子显示效果，特别是在具有高随机性或快速移动粒子的系统中。这有助于保持视觉一致性，但会有性能代价。

## 绘制参数 (Drawing parameters)

### 可见性矩形 (Visibility Rect)
控制粒子在屏幕上的可见区域。如果此矩形在视口外，引擎将停止渲染以优化性能。

你可以通过 2D 视图上方的工具栏让 Godot 自动生成：选择节点并点击 `Particles > Generate Visibility Rect`。

### 本地坐标 (Local Coords)
*   **开启**（默认）：粒子随发射节点一起移动。
![本地坐标开启示例](https://docs.godotengine.org/en/stable/_images/paranim20.gif)
*   **禁用**：粒子发射到全局空间。如果节点移动，已经发射出去的粒子不会跟着动。
![本地坐标关闭示例](https://docs.godotengine.org/en/stable/_images/paranim21.gif)

### 绘制顺序 (Draw Order)
控制粒子的重叠顺序：`Index`（按发射顺序）或 `Lifetime`（按剩余寿命）。
