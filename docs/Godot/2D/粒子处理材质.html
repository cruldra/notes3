粒子处理材质

处理材质属性 (Process material properties)​

最小值、最大值与曲线属性 (Min, max, and curve properties)​

此材质中的属性控制粒子在其寿命期间的行为和变化。其中许多属性都有 Min (最小值)、Max (最大值) 和 Curve (曲线) 值，允许你精细调整它们的行为。

这些值之间的关系如下：当粒子生成时，该属性会被设置为 Min 和 Max 之间的一个随机值。如果两者相同，则每个粒子的该值都将一致。如果还设置了 Curve，则该属性的值将乘以曲线在粒子当前寿命点上的对应值。使用曲线可以随时间改变粒子的属性。通过这种方式可以表达非常复杂的行为。

注意：本页专门介绍如何在 2D 场景中使用 ParticleProcessMaterial。有关 3D 的信息，请参阅对应页面。

寿命随机性 (Lifetime Randomness)​

控制应用于每个粒子寿命的随机程度。值为 0 表示没有随机性，所有粒子的寿命都等于 Lifetime 属性。值为 1 表示粒子的寿命在 [0.0, Lifetime] 范围内完全随机。

生成 (Spawn)​

角度 (Angle)​

确定粒子的初始旋转角度（度）。该参数通常用于随机化。

速度 (Velocity)​

方向 (Direction)​

粒子发射的基础方向。默认为 Vector3(1, 0, 0)，即向右发射。但在默认重力设置下，粒子会直接向下掉落。

扩散 (Spread)​

在基础方向的任一侧随机增加的角度（度）。扩散为 180 将向全方位发射。

初始速度 (Initial Velocity)​

粒子发射的速度（像素/秒）。

动画速度 (Animated Velocity)​

角速度 (Angular Velocity)​

粒子绕其中心旋转的速度（度/秒）。

轨道速度 (Orbit Velocity)​

使粒子绕中心转圈。

加速度 (Accelerations)​

重力 (Gravity)​

应用于每个粒子的重力。

径向加速度 (Radial Acceleration)​

如果为正，粒子向远离中心的方向加速；如果为负，则向中心聚集。

切向加速度 (Tangential Acceleration)​

使用相对于中心的切向量进行加速。

阻尼 (Damping)​

为粒子应用摩擦力，迫使它们停止。非常适合火花或爆炸效果。

显示 (Display)​

缩放 (Scale)​

确定粒子的初始缩放比例。

颜色曲线 (Color Curves)​

用于随时间改变发射粒子的颜色。

色调变化 (Hue Variation)​

设置应用于每个粒子的初始色调变化。

动画 (Animation)​

注意：粒子翻页书（Flipbook）动画仅在 GPUParticles2D 或 CPUParticles2D 节点上使用的 CanvasItemMaterial 已经过 相应配置 后才有效。

发射形状 (Emission Shapes)​

你可以设置发射掩码（Emission Mask），它决定了粒子发射的区域和方向。

  1. 选中 GPUParticles2D 节点。
  2. 在工具栏点击 Particles 菜单，选择 Load Emission Mask (加载发射掩码)。
  3. 选择一张纹理作为掩码。

发射掩码类型​

  • Solid Pixels (实心像素)：粒子从纹理的任何非透明区域生成。
  • Border Pixels (边缘像素)：粒子从纹理的外边缘生成。
  • Directed Border Pixels (有向边缘像素)：类似于边缘像素，但粒子会沿边缘向外发射。

自定义处理材质 (Customizing the process material)​

如果你需要更改或实现新的着色器逻辑，可以将当前的 ParticleProcessMaterial 转换为 ShaderMaterial。只需在文件系统面板右键点击该材质并选择 Convert to ShaderMaterial (转换为 ShaderMaterial) 即可。
