灯光与阴影

简介 (Introduction)​

默认情况下，Godot 中的 2D 场景是不带光照（unshaded）的，看不到灯光和阴影。虽然这样渲染速度很快，但场景看起来可能会比较平淡。Godot 提供了实时 2D 灯光和阴影功能，可以极大地增强项目的空间感和深度感。

相关节点 (Nodes)​

一个完整的 2D 灯光设置通常涉及以下节点：

  • CanvasModulate (用于调暗整个场景)
  • PointLight2D (用于全向灯或聚光灯)
  • DirectionalLight2D (用于模拟太阳光或月光)
  • LightOccluder2D (用于遮挡光线并投射阴影)
  • 其他接收光照的 2D 节点，如 Sprite2D 或 TileMapLayer。

CanvasModulate 用于通过指定一个颜色作为基础“环境”色来调暗场景。这是光线照射不到区域的最终显示颜色。如果没有它，场景会显得太亮，因为 2D 灯光只会在已满亮（Unshaded）的基础上进行叠加。

PointLight2D 用于照亮场景。它通常通过在场景上方叠加一张纹理来模拟光效。

LightOccluder2D 告诉着色器哪些部分应该投射阴影。这些遮挡器可以是独立节点，也可以是 TileMapLayer 的一部分。

注意

项目的背景颜色（背景清除色）不会 接收光照。如果你想让光照投射到背景上，需要添加一个视觉背景，例如使用 Sprite2D 铺满背景。

点光源 (Point lights)​

点光源（也称位置光源）是 2D 灯光中最常用的。可用于表现火把、火焰、弹丸等发出的光。

常用属性：

  • Texture: 光源使用的纹理。纹理的大小决定了光照的范围。
  • Texture Scale: 光照大小的乘数。越大的灯光对性能消耗越高，因为影响的像素更多。
  • Height: 与法线贴图（Normal mapping）相关的虚拟高度。增加此值可以使法线贴图的效果更明显。

方向光 (Directional light)​

方向光用于表现太阳或月亮。光线彼此平行，仿佛光源在无限远处。

常用属性：

  • Max Distance: 对象距离相机中心的最大距离（像素），超过此距离的阴影将被剔除以优化性能。

注意

由于 2D 渲染方法的限制，方向光的阴影总是表现为无限长，不受 Height 属性影响。

共有灯光属性 (Common light properties)​

  • Enabled: 开关灯光。
  • Energy: 能量值（强度乘数）。
  • Blend Mode: 混合模式。默认 Add (相加) 适用于大多数情况。Mix (混合) 则基于 Alpha 通道进行线性插值。
  • Item Cull Mask: 控制哪些节点接收此灯光。

设置阴影 (Setting up shadows)​

在启用灯光的 Shadow > Enabled 属性后，你最初可能看不到任何变化。这是因为场景中还没有 遮挡器 (occluders)。

要显示阴影，必须添加 LightOccluder2D 节点，并为其分配一个匹配物体轮廓的遮挡多边形（Occluder Polygon）。

自动生成遮挡器​

选中 Sprite2D 节点，点击 2D 编辑器顶部的 Sprite2D 菜单，选择 Create LightOccluder2D Sibling。系统会自动根据精灵边缘生成轮廓。

阴影属性​

  • Filter: 阴影过滤模式。None 渲染最快，适合像素风；PCF5 或 PCF13 可以获得平滑的软阴影。
  • Filter Smooth: 控制阴影的柔化程度。

注意：像素风游戏中的阴影分辨率

引擎是以 视口像素分辨率 计算阴影的。如果你想要那种和像素画风格完全契合的“锯齿感”阴影，单纯设置纹理过滤为 Nearest 是不够的。你需要使用自定义着色器将 LIGHT_VERTEX 和 SHADOW_VERTEX 坐标对齐到像素网格。

法线贴图与高光贴图 (Normal and specular maps)​

法线贴图可以让 2D 光照看起来不再扁平，通过在像素级别改变光照强度来模拟凹凸感。高光贴图则可以让部分光线反射回观察者，提升质感。

要为 2D 节点设置法线/高光贴图：

  1. 在绘制纹理的属性上创建一个新的 CanvasTexture 资源。
  2. 在 CanvasTexture 中，将原始图像放入 Diffuse 槽。
  3. 将生成的法线图放入 Normal Map 槽，高光图放入 Specular 槽。

提示

你可以使用开源工具 Laigter 自动为你的精灵图生成法线和高光贴图。

性能优化建议 (Optimization)​

如果 2D 灯光导致性能下降，对于一些短时动态效果（如子弹或爆炸），可以考虑使用 叠加模式的精灵 (Additive Sprites)。 只需将 Sprite2D 的 CanvasItem > Material 设置为 CanvasItemMaterial，并将 Blend Mode 设为 Add 即可。这种方式渲染极快，但缺点是不能产生阴影，且无法与法线贴图交互。
