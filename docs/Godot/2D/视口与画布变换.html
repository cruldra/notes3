视口与画布变换

简介 (Introduction)​

本页概述了 2D 变换的过程：从节点在本地绘制内容，到内容最终显示在屏幕上的全过程。这涉及引擎的一些底层细节。

本教程的目标是教你如何向 Input 发送带有正确坐标系位置的自定义输入事件。

有关所有坐标系和 2D 变换的更详细描述，请参阅 2D 坐标系和 2D 变换。

画布变换 (Canvas transform)​

正如上一篇教程 画布图层 中提到的，每个 CanvasItem 节点（记住 Node2D 和 Control 节点都以 CanvasItem 为共同根）都位于一个 画布层 (Canvas Layer) 中。每个画布层都有一个变换（平移、旋转、缩放等），可以作为 Transform2D 进行访问。

节点默认在图层 0（内置画布）中绘制。要将节点放在不同的层中，可以使用 CanvasLayer 节点。

全局画布变换 (Global canvas transform)​

视口（Viewports）还拥有一个 全局画布变换 (Global Canvas transform)（也是一个 Transform2D）。这是主变换，会影响所有单独的画布层变换。通常，这主要由 Godot 的 CanvasItem 编辑器使用。

拉伸变换 (Stretch transform)​

最后，视口拥有一个 拉伸变换 (Stretch Transform)，用于缩放或拉伸屏幕。该变换由引擎内部使用（如 多分辨率支持 中所述），但也可以在每个视口上手动设置。

输入事件会乘以该变换，但不包含上述其他变换。为了方便将 InputEvent 坐标转换为本地 CanvasItem 坐标，引擎添加了 CanvasItem.make_input_local() 函数。

窗口变换 (Window transform)​

根视口是一个 Window (窗口)。为了按照 多分辨率支持 中的描述对窗口内容进行缩放和定位，每个窗口都包含一个 窗口变换 (window transform)。例如，它负责在窗口两侧产生黑边，以使视口以固定的纵横比显示。

变换函数 (Transform functions)​

上图展示了一些可用的变换函数。所有变换都是从右向左进行的，这意味着将变换与坐标相乘会得到链中左侧的坐标系；而乘以变换的 仿射逆矩阵 (affine inverse) 则会得到链中右侧的坐标系。

# 从 CanvasItem 调用。
canvas_pos = get_global_transform() * local_pos
local_pos = get_global_transform().affine_inverse() * canvas_pos
// 从 CanvasItem 调用。
canvasPos = GetGlobalTransform() * localPos;
localPos = GetGlobalTransform().AffineInverse() * canvasPos;

最后，要将 CanvasItem 的本地坐标转换为屏幕坐标，请按以下顺序相乘：

var screen_coord = get_viewport().get_screen_transform() * get_global_transform_with_canvas() * local_pos
var screenCoord = GetViewport().GetScreenTransform() * GetGlobalTransformWithCanvas() * localPos;

然而请记住，通常不建议直接操作屏幕坐标。推荐的做法是直接在画布坐标中工作（CanvasItem.get_global_transform()），以便自动屏幕分辨率缩放能正常工作。

发送自定义输入事件 (Feeding custom input events)​

通常需要向游戏发送自定义输入事件。利用上述知识，要在获得焦点的窗口中正确执行此操作，必须遵循以下方式：

var local_pos = Vector2(10, 20) # 相对于 Control/Node2D 的本地位置。
var ie = InputEventMouseButton.new()
ie.button_index = MOUSE_BUTTON_LEFT
ie.position = get_viewport().get_screen_transform() * get_global_transform_with_canvas() * local_pos
Input.parse_input_event(ie)
var localPos = new Vector2(10,20); // 相对于 Control/Node2D 的本地位置。
var ie = new InputEventMouseButton()
{
    ButtonIndex = MouseButton.Left,
    Position = GetViewport().GetScreenTransform() * GetGlobalTransformWithCanvas() * localPos,
};
Input.ParseInputEvent(ie);
