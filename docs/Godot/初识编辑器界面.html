初识编辑器界面

本页将带你简要回顾 Godot 的界面。我们将介绍不同的主要屏幕和面板，帮助你熟悉环境。

另见

有关编辑器界面及其使用方法的全面介绍，请参阅 编辑器手册。

项目管理器 (The Project Manager)​

启动 Godot 时，你看到的第一个窗口是项目管理器。在默认的 项目 (Projects) 选项卡中，你可以管理现有项目、导入或创建新项目等。

在窗口顶部，还有另一个名为 资产库 (Asset Library) 的选项卡。第一次访问此选项卡时，你会看到一个 “上线 (Go Online)” 按钮。出于隐私原因，Godot 项目管理器默认不会访问互联网。要更改此设置，请点击 “上线” 按钮。你稍后可以在设置中更改此选项。

将网络模式设置为 “在线” 后，你可以在开源资产库中搜索演示项目，其中包含许多由社区开发的项目：

可以使用 设置 (Settings) 菜单打开项目管理器的设置：

在这里，你可以更改编辑器的语言（默认为系统语言）、界面主题、显示缩放、网络模式以及目录命名规范。

要了解项目管理器的详细信息，请阅读 使用项目管理器。

五个主要屏幕 (The five main screens)​

编辑器顶部中心有五个主要屏幕按钮：2D、3D、Script、Game 和 Asset Library。

你将在 2D 屏幕 中开发各种类型的游戏。除了 2D 游戏外，2D 屏幕也是你构建界面的地方。

在 3D 屏幕 中，你可以处理网格体、光照，并为 3D 游戏设计关卡。

注意

阅读 3D 简介 以了解有关 3D 主屏幕 的更多详细信息。

Game 屏幕 是你从编辑器运行项目时，项目显示的地方。你可以浏览项目进行测试，并实时暂停和调整。请注意，这仅用于测试调整的效果，游戏停止运行后，此处所做的任何更改都不会被保存。

脚本 (Script) 屏幕 是一个完整的代码编辑器，带有调试器、丰富的自动补全和内置的代码参考。

最后，资产库 (Asset Library) 是一个免费开源插件、脚本和资产库，可供你在项目中使用。

另见

你可以在 关于资产库 中了解更多关于资产库的信息。

内置类参考 (Integrated class reference)​

Godot 附带了内置的类参考。

你可以通过以下任一方法搜索有关类、方法、属性、常量或信号的信息：

  • 在编辑器中的任何位置按下 F1（macOS 上为 Opt + Space，或者带 Fn 键的笔记本电脑为 Fn + F1）。
  • 点击脚本主屏幕右上角的 “搜索帮助 (Search Help)” 按钮。
  • 点击 “帮助 (Help)” 菜单并选择 “搜索帮助”。
  • 在脚本编辑器中，按住 Ctrl 点击（macOS 为 Cmd + 点击）类名、函数名或内置变量。

执行上述任何操作时，都会弹出一个窗口。输入内容以搜索任何条目。你也可以使用它来浏览可用的对象和方法。

双击某一条目即可在脚本主屏幕中打开相应的页面。

或者：

  • 在脚本编辑器中，按住 Ctrl (macOS 为 Cmd) 的同时点击类名、函数名或内置变量。
  • 右键点击节点并选择 打开文档 (Open Documentation)，或者在脚本编辑器中为元素选择 查找符号 (Lookup Symbol)，将直接打开其文档。
