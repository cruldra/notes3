在这个简短的第一部分中，我们将设置并组织项目。

启动 Godot 并创建一个新项目。

![New Project Button](https://docs.godotengine.org/en/stable/_images/new-project-button.webp)

创建新项目时，你只需选择一个有效的 **项目路径 (Project Path)**。你可以保持其他默认设置不变。

下载 [dodge_the_creeps_2d_assets.zip](https://github.com/godotengine/godot-docs-project-starters/releases/download/latest-4.x/dodge_the_creeps_2d_assets.zip)。该压缩包包含了制作游戏所需的图像和声音。解压该压缩包，并将 `art/` 和 `fonts/` 目录移动到你的项目目录中。

如果你打算使用 C#，请确保你安装了所需的依赖项。你需要最新的稳定版 .NET SDK 以及一个编辑器（如 VS Code）。详见 [前提条件](https://docs.godotengine.org/en/stable/tutorials/scripting/c_sharp/c_sharp_basics.html)。

> **注意**
>
> 本教程的 C++ 部分尚未针对新的 GDExtension 系统进行重写。

你的项目文件夹应该如下所示：

![Folder Content](https://docs.godotengine.org/en/stable/_images/folder-content.webp)

这款游戏是为竖屏模式设计的，所以我们需要调整游戏窗口的大小。点击 **项目 -> 项目设置 (Project -> Project Settings)** 打开项目设置窗口，在左侧列中打开 **显示 -> 窗口 (Display -> Window)** 选项卡。在那里，将 “视口宽度 (Viewport Width)” 设置为 `480`，“视口高度 (Viewport Height)” 设置为 `720`。你可以在左上角看到 “项目 (Project)” 菜单。

![Setting Project Width and Height](https://docs.godotengine.org/en/stable/_images/setting-project-width-and-height.webp)

此外，在 **拉伸 (Stretch)** 选项下，将 **模式 (Mode)** 设置为 `canvas_items`，将 **比例 (Aspect)** 设置为 `keep`。这可以确保游戏在不同尺寸的屏幕上能保持比例缩放。

![Setting Stretch Mode](https://docs.godotengine.org/en/stable/_images/setting-stretch-mode.webp)

## 组织项目 (Organizing the project)

在本项目中，我们将制作 3 个独立的场景：`Player` (玩家)、`Mob` (小怪) 和 `HUD` (图形界面)，最后我们会将它们组合到游戏的 `Main` (主) 场景中。

在较大的项目中，创建文件夹来存放各种场景及其脚本可能会很有用，但对于这个相对较小的游戏，你可以将场景和脚本保存在项目的根文件夹中，即 `res://`。你可以在左下角的 “文件系统 (FileSystem)” 面板中看到你的项目文件夹：

![FileSystem Dock](https://docs.godotengine.org/en/stable/_images/filesystem_dock.webp)

项目准备就绪后，我们就可以在下一课中设计玩家场景了。
