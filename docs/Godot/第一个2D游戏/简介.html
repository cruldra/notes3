简介

在这个循序渐进的教程系列中，你将使用 Godot 创建你的第一个完整的 2D 游戏。到本系列结束时，你将拥有一个属于自己的简单但完整的游戏，如下图所示。

你将学习 Godot 编辑器的工作原理、如何组织项目以及构建 2D 游戏。

注意

本项目是 Godot 引擎的入门介绍。它假设你已经具备一定的编程经验。如果你完全是编程新手，你应该从这里开始：脚本语言。

这款游戏名为 “Dodge the Creeps!”（躲避小怪！）。你的角色必须移动并尽可能长时间地躲避敌人。

你将学习：

  • 使用 Godot 编辑器创建一个完整的 2D 游戏。
  • 组织一个简单的游戏项目。
  • 移动玩家角色并更改其精灵。
  • 随机生成敌人。
  • 计分。

以及更多内容。

你还会发现另一个系列，在那里你将创建一个类似但为 3D 的游戏。不过，我们建议你从这个系列开始。

为什么要从 2D 开始？

如果你是游戏开发新手或不熟悉 Godot，我们建议从 2D 游戏开始。这将让你在应对往往更复杂的 3D 游戏之前，先熟悉这两者。

你可以在以下位置找到本项目的完整版本：

  • Dodge the Creeps 源代码 (GDScript)
  • Dodge the Creeps 源代码 (C#)

前提条件 (Prerequisites)​

本循序渐进教程面向遵循了完整 入门指南 (Step by step) 的初学者。

如果你是一名经验丰富的程序员，可以在这里找到完整演示的源代码：Dodge the Creeps 源代码。

我们准备了一些你需要的游戏资产供下载，这样我们可以直接进入代码部分。
